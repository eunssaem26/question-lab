# 기존 독서 문해력 진단 도구 벤치마크 분석

## 1. KOLRA (한국어 읽기검사)

| 항목 | 내용 |
|------|------|
| **개발** | 배소영 외 (학지사) |
| **대상** | 초등 1~6학년 (읽기 장애/지연 선별 목적) |
| **구성** | 핵심검사: 해독, 읽기이해, 문단글 읽기 유창성, 듣기이해 |
| | 상세검사: 음운처리, 쓰기 |
| | 선별검사: 낱말 읽기 유창성, 읽기 설문지 |
| **방식** | 개별 대면 검사 (검사자-학생 1:1) |
| **소요시간** | 약 30~40분 |
| **비용** | 검사 도구 구입 비용 (~10만원대) + 전문가 실시 |
| **강점** | 교육학적으로 검증됨, 하위영역별 강약점 파악, 표준화 규준 |
| **한계** | 초등만 대상(중학 미포함), 1:1 대면만 가능, 대규모 시행 불가, 비용 높음 |

## 2. 기초학력 진단-보정 시스템 (시도교육청)

| 항목 | 내용 |
|------|------|
| **개발** | 한국교육과정평가원 + 충남대 응용교육측정평가연구소 |
| **대상** | 초등~중등 전 학년 |
| **구성** | 국어, 수학, 영어, 사회, 과학 (교과 단위) |
| | 국어 내 읽기는 독립 영역이 아닌 교과 일부로 포함 |
| **방식** | 온라인(학생 로그인) 또는 PDF 출력 후 지필 |
| **시기** | 3월 진단, 5/9/12월 보정 |
| **비용** | 무료 (교육청 운영) |
| **강점** | 전국 규모, 무료, 교육과정 연계, 보정 학습 자료 제공 |
| **한계** | 읽기 전문 진단 아님(국어 교과 전체), 학년 단위 진단(수준별 아님), 학교 통해서만 접근 가능, 개인 접속 불가 |

## 4. 민간 EdTech 진단 (밀크T, 엘리하이, 아이스크림홈런)

| 항목 | 내용 |
|------|------|
| **대상** | 초등~중등 (유료 가입 회원) |

| **비용** | 월 10~15만원 유료 구독 |
| **강점** | 진단→학습 루프 완비, 적응형 출제, 대규모 데이터 기반 |
| **한계** | 높은 비용, 읽기 특화가 아닌 범용, 캐릭터/세계관 미약 |

## 질문연구소 별쌤 진단의 차별점

| 기존 도구의 한계 | 별쌤 진단의 해결 |
|----------------|----------------|
| 읽기 전용 진단이 없음 (교과 일부로만 포함) | **독서 문해력 5개 영역 전문 진단** |
| 학년 단위 결과 (수준별 진단 아님) | **학년 라벨 없는 1~7단계 수준 진단** |
| 학교/기관 통해서만 접근 가능 | **웹에서 누구나 무료 접속** |
| 1:1 대면 또는 학교 시행만 가능 | **CBT로 언제든 자가 진단** |
| 진단만 하고 끝 (보정 시스템은 별도) | **진단→책쌤 학습 콘텐츠 직접 연계** |
| 캐릭터/동기 부여 요소 없음 | **별쌤 캐릭터 세계관으로 따뜻한 경험** |
| 높은 비용 (민간) 또는 접근 제한 (공교육) | **완전 무료, 오픈 접근** |

## 벤치마크에서 배울 점

1. **KOLRA의 영역 구분**: 해독-이해-유창성-듣기의 분류 체계 참고. 우리의 5영역(어휘/사실적/추론적/비판적/구조)이 더 읽기 이해에 집중된 설계임을 명시.
2. **진단-보정 시스템의 보정 연계**: "진단만 하고 끝"이 아닌 보정 학습 연계가 핵심. 우리도 책쌤 콘텐츠 연계로 이 패턴을 따름.
3. **민간 EdTech의 적응형 출제**: v1은 순차출제로 시작하되, 데이터 축적 후 v2에서 적응형 도입 계획을 유지.
4. **표준화 검사의 신뢰성**: 은쌤 검수 + 파일럿 테스트로 신뢰성 확보. 장기적으로 문항 양호도 데이터 공개를 고려.
