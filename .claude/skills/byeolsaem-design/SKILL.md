# 별쌤-설계 에이전트

## 개요

2022 개정 교육과정 국어과 읽기 영역 성취기준 원문을 분석하여, 질문연구소의 7단계 독서 문해력 평가 준거틀을 생성한다.

**핵심 원칙**:
- 학년이 아닌 **1~7단계**로 표현 (학년 라벨 절대 사용 금지)
- 모든 분석은 `reference/achievement-standards.json` 원문에 근거 (기억 의존 금지)
- 5개 평가 영역별 독립 수준 판정이 가능하도록 설계
- 스키마는 `reference/schema.json`의 framework 구조를 따름

---

## Step 1: 교육과정 성취기준 분석

1. `reference/achievement-standards.json`을 읽는다
2. 평가 대상 학년군(초3~4, 초5~6, 중1~3)의 성취기준을 정리:
   - 각 성취기준 코드, 내용, 소속 역량 영역
   - 태도 영역 성취기준은 CBT 평가 불가이므로 제외 표시
3. `progression_matrix`를 참조하여 7단계 수준 진행 흐름 확인

**출력**: 성취기준 분석표 (코드 | 내용 | 평가영역 | CBT 가능 여부)

---

## Step 2: 평가 영역 확정

5개 평가 영역을 확정한다:

| 코드 | 영역 | 정의 | CBT 문항 유형 |
|------|------|------|-------------|
| VOC | 어휘력 | 어휘의 의미를 정확히 파악하고 문맥에 맞게 이해 | 4지선다 (동의어, 문맥 의미) |
| LIT | 사실적 이해 | 명시적 정보 확인, 중심 내용 파악, 요약 | 4지선다 |
| INF | 추론적 이해 | 생략/함축 내용 추론, 의도/관점 파악 | 4지선다 |
| CRT | 비판적 이해 | 사실/의견 구분, 타당성/신뢰성 평가 | 4지선다, O/X |
| STR | 구조 파악 | 글의 짜임, 문단 관계, 전개 방식 분석 | 4지선다, 순서 배열 |

각 영역의 가중치 초기값:
- VOC: 0.15, LIT: 0.25, INF: 0.25, CRT: 0.20, STR: 0.15

---

## Step 3: 7단계 수준 기술서 작성

`achievement-standards.json`의 `level_differentiation`과 `progression_matrix`를 기반으로 각 단계의 수준을 기술한다.

각 단계에 대해 다음을 작성:

### 종합 수준 기술
> "이 단계의 학생은 ___할 수 있다."

### 영역별 수준 기술
각 영역(VOC/LIT/INF/CRT/STR)에 대해:
> "이 단계에서 [영역명]: ___할 수 있다."

### 성취기준 매핑
이 단계에 해당하는 교육과정 성취기준 코드 나열.
- 같은 학년군의 두 단계(예: 1단계와 2단계)는 같은 성취기준을 공유하되, 수준 기술에서 차이를 둠

**규칙**:
- 단계 간 변별이 명확해야 함 (인접 단계끼리 읽었을 때 차이가 분명)
- 하위 단계의 수준 기술이 상위 단계에 포함되어야 함 (위계성)
- 학년 번호(초3, 중1 등)를 수준 기술에 사용하지 않음

---

## Step 4: 평가 준거 매트릭스 작성

7단계 × 5영역 = 35셀 매트릭스를 작성한다.

각 셀에 포함할 내용:
1. **평가 내용**: 이 단계, 이 영역에서 무엇을 평가하는가
2. **통과 기준**: 3문항 중 몇 개를 맞아야 해당 단계 인정인가
3. **예시 문항 유형**: 어떤 형태의 문항이 적합한가

---

## Step 5: 단계 판정 기준 설계

### 종합 단계 판정
1. 각 영역 정답률 산출 (0~1)
2. 가중 평균 = Σ(영역 정답률 × 가중치)
3. 가중 평균을 7단계 중 하나에 매핑

### 영역별 독립 수준 판정
- 각 영역 내에서 문항 난이도 기반으로 독립 단계 판정
- "어휘 5단계, 추론 3단계, 종합 4단계" 형태의 프로파일 제공

### 경계선 처리
- 두 단계의 경계에 걸치는 경우: "3~4단계" 형태로 범위 표시
- 영역별 편차가 큰 경우(최대-최소 ≥ 3단계): 특별 안내 메시지

## Step 6: 산출물 생성

### 파일 1: `별쌤-평가준거틀.md`
사람이 읽는 문서. 구조:

```markdown
# 질문연구소 독서 문해력 평가 준거틀 v1.0

## 1장: 평가 철학 및 설계 원칙
## 2장: 평가 영역 정의
## 3장: 7단계 수준 기술서
## 4장: 단계별 성취기준 매핑표
## 5장: 평가 준거 매트릭스 (7단계 × 5영역)
## 6장: 단계 판정 기준
## 부록: 2022 개정 교육과정 읽기 성취기준 원문 대조표
```

### 파일 2: `framework.json`
`reference/schema.json`의 framework 스키마를 따르는 기계 판독용 버전.
모든 필드를 빠짐없이 채운다.

## 검증 체크리스트

산출물 생성 후, 다음을 확인:

- [ ] achievement-standards.json의 모든 평가 대상 성취기준이 매핑됨
- [ ] 7단계 수준 기술이 위계적이고 단계 간 변별 가능
- [ ] 5개 영역 각각에 대해 7단계 수준 기술이 존재
- [ ] 학년 번호(초3, 중1 등)가 수준 기술에 등장하지 않음
- [ ] framework.json이 schema.json 구조와 일치
- [ ] 단계 판정 기준의 가중치 합이 1.0
- [ ] 35셀 매트릭스에 빈 셀이 없음
