# 질문연구소 독서 문해력 평가 준거틀 v1.1

> **작성**: 별쌤-설계 에이전트  
> **근거**: 2022 개정 교육과정 국어과 읽기 영역 (교육부고시 제2022-33호)  
> **작성일**: 2026-04-12  
> **버전**: 1.1.0

## 1장: 평가 철학 및 설계 원칙

### 1.1 평가 철학

독서 문해력 평가는 학생이 현재 "무엇을 할 수 있는가"를 정확히 파악하여, 다음 단계로 나아갈 수 있도록 돕는 **성장 지향적 진단 도구**이다.

2022 개정 교육과정은 읽기를 "독자가 배경지식과 경험을 활용하여 글의 의미를 능동적으로 구성하는 행위"로 정의한다. 이 준거틀은 읽기의 능동적·구성적 본질을 존중하면서, CBT(컴퓨터 기반 검사)로 측정 가능한 역량을 체계적으로 진단하는 것을 목표로 한다.

### 1.2 설계 원칙

| 번호 | 원칙 | 설명 |
|------|------|------|
| 1 | **단계 기반 표현** | 학년이 아닌 1~7단계로만 수준을 표현한다. 학년 라벨(초3, 중1 등)은 수준 기술에 사용하지 않는다. |
| 2 | **교육과정 근거** | 모든 평가 내용은 2022 개정 교육과정 국어과 읽기 성취기준에 직접 근거한다. |
| 3 | **영역별 독립 진단** | 5개 평가 영역 각각에서 독립적으로 수준을 판정하여, 학생의 강점과 약점을 프로파일로 제공한다. |
| 4 | **위계적 수준 구조** | 하위 단계의 역량은 상위 단계에 포함된다. 각 단계는 이전 단계를 전제로 한다. |
| 5 | **인접 단계 변별** | 인접 단계 간 차이가 명확하여, 진단 결과가 학습 방향 설정에 실질적으로 도움이 된다. |
| 6 | **CBT 측정 가능성** | 태도/정의적 영역은 결과 해석의 참고 정보로만 다루고, 단계 판정은 CBT로 측정 가능한 인지적 역량만으로 산출한다. |
| 7 | **성장 지향** | 부족함을 지적하는 것이 아니라, 현재 수준에서 다음 단계로의 성장 경로를 안내한다. |

## 2장: 평가 영역 정의

### 2.1 5개 평가 영역 개요

본 평가 준거틀은 독서 문해력을 5개 영역으로 나누어 진단한다. 각 영역은 독립적으로 수준 판정이 가능하며, 종합 판정 시에는 가중치를 적용하여 통합한다.

### 2.2 영역별 상세 정의

#### VOC — 어휘력

| 항목 | 내용 |
|------|------|
| **정의** | 글에 사용된 어휘의 의미를 정확히 파악하고, 문맥에 맞게 이해하는 능력 |
| **핵심 능력** | 어휘 의미 파악, 문맥 추론, 다의어/동음이의어 구분, 관용/비유 표현 이해, 학술 어휘 이해 |
| **CBT 문항 유형** | 4지선다 (동의어 선택, 문맥 의미 선택, 빈칸 완성) |
| **가중치** | 0.15 |

#### LIT — 사실적 이해

| 항목 | 내용 |
|------|------|
| **정의** | 글에 명시적으로 드러난 정보를 확인하고, 중심 내용을 파악하며, 내용을 요약하는 능력 |
| **핵심 능력** | 중심 내용 파악, 세부 정보 확인, 내용 간추리기, 구조 활용 요약, 목적별 요약 |
| **CBT 문항 유형** | 4지선다 (중심 내용 선택, 세부 정보 확인, 요약문 선택) |
| **가중치** | 0.25 |

#### INF — 추론적 이해

| 항목 | 내용 |
|------|------|
| **정의** | 글에 직접 드러나지 않은 내용을 예측하거나, 생략된 정보를 추론하며, 필자의 의도나 관점을 파악하는 능력 |
| **핵심 능력** | 내용 예측, 질문 생성, 생략 내용 추론, 함축 표현 해석, 의도/관점 추론, 메타인지 |
| **CBT 문항 유형** | 4지선다 (추론 내용 선택, 의도 파악, 함축 의미 선택) |
| **가중치** | 0.25 |

#### CRT — 비판적 이해

| 항목 | 내용 |
|------|------|
| **정의** | 사실과 의견을 구분하고, 글의 타당성과 신뢰성을 평가하며, 다양한 관점에서 글을 비판적으로 읽는 능력 |
| **핵심 능력** | 사실/의견 구분, 출처 신뢰성 판단, 타당성 평가, 표현 적절성 평가, 다관점 비교, 통합적 읽기 |
| **CBT 문항 유형** | 4지선다, 분류형 2지선다 (사실-의견 구분, 타당성 판단, 신뢰성 평가) |
| **가중치** | 0.20 |

#### STR — 구조 파악

| 항목 | 내용 |
|------|------|
| **정의** | 글의 짜임과 구조를 이해하고, 문단 간 관계를 파악하며, 글의 전개 방식을 분석하는 능력 |
| **핵심 능력** | 기본 구조 인식, 문단 역할 구분, 전개 방식 파악, 설명/논증 방법 분석 |
| **CBT 문항 유형** | 4지선다, 순서 배열 (구조 파악, 전개 방식 선택, 문단 순서 배열) |
| **가중치** | 0.15 |

### 2.3 가중치 근거

| 영역 | 가중치 | 근거 |
|------|--------|------|
| VOC | 0.15 | 어휘는 모든 읽기의 기반이나, 단독 문항으로는 맥락 읽기 능력을 완전히 반영하기 어려움 |
| LIT | 0.25 | 사실적 이해는 모든 읽기 활동의 핵심이며, 교육과정에서 가장 많은 성취기준이 배정됨 |
| INF | 0.25 | 추론적 이해는 교육과정이 강조하는 "능동적 의미 구성"의 핵심 역량 |
| CRT | 0.20 | 비판적 이해는 상위 수준에서 비중이 높아지며, 교육과정 전반에 걸쳐 강조됨 |
| STR | 0.15 | 구조 파악은 요약과 분석의 기반이나, 독립 평가 비중은 상대적으로 작음 |
| **합계** | **1.00** | |

## 3장: 7단계 수준 기술서

### 3.1 종합 수준 기술

#### 1단계

> 이 단계의 학생은 짧은 글을 또박또박 읽으며 의미를 파악할 수 있다. 한 문단에서 중심 문장을 찾고, 제목이나 그림을 단서로 글의 내용을 예측할 수 있다. 글에서 사실(일어난 일)과 의견(생각)을 구분할 수 있으며, 글의 처음-가운데-끝 구조를 인식할 수 있다.

#### 2단계

> 이 단계의 학생은 여러 문단으로 된 글을 자연스럽게 읽으며 중심 생각을 파악하고 내용을 간추릴 수 있다. 질문을 단서로 예측하며 읽고, 예측이 맞는지 점검할 수 있다. 필자의 의견과 자신의 의견을 비교하고, 출처의 신뢰성을 근거를 들어 판단할 수 있다. 문단의 역할(서론, 본론, 결론)을 구분할 수 있다.

#### 3단계

> 이 단계의 학생은 글의 구조(서론-본론-결론, 원인-결과 등)를 파악하고 주제를 찾을 수 있다. 글에 직접 드러나지 않은 내용을 앞뒤 문맥을 통해 짐작할 수 있으며, 글의 내용이 사실에 부합하는지, 주장에 근거가 있는지 판단할 수 있다. 같은 주제에 대해 서로 다른 의견이 있음을 이해할 수 있다.

#### 4단계

> 이 단계의 학생은 글의 구조를 활용하여 내용을 체계적으로 요약할 수 있다. 비유적 표현이나 관용 표현의 함축된 의미를 문맥에서 추론할 수 있으며, 글의 내용 타당성뿐 아니라 표현 방식의 적절성까지 평가할 수 있다. 다양한 관점의 글을 비교하며 읽고 자신의 판단에 활용할 수 있다. 글의 전개 방식을 분석하고 문단 간 관계를 설명할 수 있다.

#### 5단계

> 이 단계의 학생은 글의 종류에 따라 중심 내용과 세부 정보를 구분하여 요약할 수 있다. 글에 드러난 정보를 바탕으로 필자의 의도를 추론하고, 자신의 읽기 수준을 대략적으로 파악하여 적절한 글을 고를 수 있다. 다양한 종류의 글과 자료에서 주장의 타당성을 근거의 충분성과 적절성을 기준으로 판단할 수 있다. 기본 설명 방법(정의, 예시, 비교)을 파악할 수 있다.

#### 6단계

> 이 단계의 학생은 읽기 목적(학습, 감상, 정보 획득 등)에 따라 요약 방식을 달리할 수 있다. 배경지식을 활용하여 글에 드러나지 않은 관점이나 전제를 추론하며, 읽기 과정에서 이해가 안 되는 부분을 인식하고 전략을 조정할 수 있다. 그래프, 표, 이미지 등이 포함된 복합 자료의 신뢰성을 평가하고, 논증의 구조(주장-근거-뒷받침)를 분석할 수 있다.

#### 7단계

> 이 단계의 학생은 복잡한 글의 구조를 분석하고 목적에 맞게 효과적으로 요약할 수 있다. 필자의 의도, 관점, 숨겨진 전제를 종합적으로 추론하며 비판적으로 읽을 수 있다. 자신의 독서 상황과 수준을 정확히 판단하고 읽기 과정을 능동적으로 점검/조정할 수 있다. 복합양식 텍스트의 타당성, 신뢰성, 표현 적절성을 종합적으로 평가하며, 다양한 설명/논증 방법의 타당성을 평가할 수 있다. 동일 화제의 여러 글을 통합적으로 읽고 자신의 관점을 형성할 수 있다.

| 3 | 단일 글에서 주장에 근거가 있는지, 내용이 사실에 부합하는지 있다/없다 수준으로 판단할 수 있다. |

| 3 | 원인-결과, 비교-대조 등 글의 짜임을 파악할 수 있다. |
| 4 | 글의 전개 방식을 분석하고 문단 간 관계를 설명할 수 있다. |
| 5 | 기본 설명 방법(정의, 예시, 비교)을 파악할 수 있다. |
| 6 | 논증 구조(주장-근거-뒷받침)를 분석할 수 있다. |

## 4장: 단계별 성취기준 매핑표

### 4.1 매핑 원칙

- 같은 학년군의 두 단계(예: 1단계와 2단계)는 동일 성취기준을 공유하되, `level_differentiation`으로 수준 차이를 둔다.
- 태도/정의적 성취기준([4국02-06], [6국02-05], [9국02-07])은 CBT 단계 판정에서 제외하고, 리포트의 참고 코멘트로만 활용한다.
- 하나의 문항은 하나의 `주 측정 영역(primary_domain)`을 가진다. 성취기준이 둘 이상의 영역과 연결되는 경우에도 문항 통계와 수준 판정은 `primary_domain` 기준으로 집계한다.

### 6.2 종합 단계 판정

#### 판정 방법: 2단계 적응형 준거 판정

1. **1차 공통 라우팅**: 각 영역에서 앵커 문항 3개씩 총 15문항을 제시하여 대략적인 수행 대역을 추정한다.
2. **영역별 후보 단계 설정**: 1차 결과를 바탕으로 각 영역의 후보 단계를 `1~2`, `3~4`, `5~7` 중 하나로 좁힌다.
3. **2차 단계 확인**: 후보 단계 대역 안에서 해당 영역 문항 3개씩 총 15문항을 추가 제시하여, 각 영역의 최종 단계를 확정한다.
4. **영역 단계 확정 후 종합 단계 산출**: 확정된 영역 단계를 정수값(1~7)으로 두고, 가중 평균을 계산한 뒤 반올림이 아니라 경계 규칙을 적용해 종합 단계를 제시한다.

#### 1차 공통 라우팅 규칙

- 각 영역은 중간 난도 앵커 문항 3개로 구성한다.
- 3문항 중 0~1문항 정답: 하위 대역(`1~2` 또는 `3~4`) 우선 배정
- 3문항 중 2문항 정답: 현재 대역 유지, 경계선 표시 후보
- 3문항 중 3문항 정답: 상위 대역(`3~4` 또는 `5~7`) 우선 배정
- 라우팅 앵커의 실제 단계 배치는 파일럿 후 고정하되, 초기 버전에서는 `2단계`, `4단계`, `6단계` 앵커를 사용한다.

#### 2차 단계 확인 규칙

- 후보 단계 대역 안에서 인접 단계 문항을 제시한다.
- 각 단계는 해당 영역에서 3문항 중 2문항 이상 정답 시 `통과`로 본다.
- 최종 수준은 `안정적으로 통과한 최상위 단계`로 판정한다.
- 경계선 학생은 인접 두 단계 중 낮은 단계는 통과, 높은 단계는 1문항 정답인 경우로 정의한다.

#### 종합 단계 산출

1. **영역 단계값 부여**: VOC, LIT, INF, CRT, STR 각각의 최종 단계를 `1~7` 정수로 변환
2. **가중 평균 산출**: 가중 평균 = Σ(영역 단계값 × 영역 가중치)
3. **종합 단계 변환**: 가중 평균 소수점 첫째 자리에서 반올림하지 않고 아래 기준표를 적용

| 가중 평균 단계값 | 종합 단계 |
|----------------|----------|
| 1.0 ~ 1.4 | 1단계 |
| 1.5 ~ 2.4 | 2단계 |
| 2.5 ~ 3.4 | 3단계 |
| 3.5 ~ 4.4 | 4단계 |
| 4.5 ~ 5.4 | 5단계 |
| 5.5 ~ 6.4 | 6단계 |
| 6.5 ~ 7.0 | 7단계 |

### 6.3 영역별 독립 수준 판정

각 영역은 독립적으로 1~7단계를 판정한다. 문항은 공통 라우팅 후 후보 단계 대역 안에서 제시되며, 영역 내 단계 통과 여부에 따라 판정한다.

**판정 방식**: 해당 영역에서 출제된 문항의 단계와 정답 여부를 기반으로, 안정적으로 통과하는 최상위 단계를 해당 영역의 수준으로 판정한다.

- 3문항 중 2문항 이상 정답: 해당 단계 **통과**
- 3문항 중 1문항 이하 정답: 해당 단계 **미통과**
- 인접 단계 간 판정이 엇갈리는 경우: 낮은 단계 우선 판정 후 경계선 표시 검토

**결과 프로파일 예시**:
> 종합 4단계 / 어휘(VOC) 5단계 / 사실적 이해(LIT) 4단계 / 추론적 이해(INF) 3단계 / 비판적 이해(CRT) 4단계 / 구조 파악(STR) 4단계

### 6.4 경계선 처리

#### 경계선 학생 판정

가중 평균 단계값이 두 단계의 경계(±0.2 범위)에 해당하는 경우:
- **범위 표시**: "3~4단계"와 같이 두 단계를 병기한다.
- **추가 안내**: "현재 3단계와 4단계 사이에 있습니다. 4단계로 올라가려면 [구체적 영역]을 보완하세요."

#### 영역 편차 알림

영역별 수준의 최대-최소 차이가 3단계 이상인 경우:
- **특별 안내 메시지** 생성: "영역 간 편차가 큽니다. [최고 영역]은(는) [N]단계로 강점이고, [최저 영역]은(는) [M]단계로 집중 보완이 필요합니다."
- **강점/약점 영역**을 결과에 명시한다.

#### 경계선 점수 예시

| 가중 평균 단계값 | 판정 | 안내 |
|----------------|------|------|
| 2.4 | 2단계 | 정상 판정 |
| 2.5 | 2~3단계 | "2단계와 3단계 사이입니다." |
| 2.6 | 3단계 | 정상 판정 |
| 3.4 | 3~4단계 | "3단계와 4단계 사이입니다." |
