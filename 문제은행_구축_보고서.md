## 🎯 난이도 티어 적용

모든 문항에 3단계 난이도를 적용:
- **Easy**: 단계 기초 확인 문항 (정답률 85% 이상 목표)
- **Medium**: 단계 중심 문항 (정답률 60-70% 목표)
- **Hard**: 단계 상위/다음 단계 예측 문항 (정답률 40-50% 목표)

### 단계별 난이도 분포 (3단계 예시)
```
쉬움(Easy): 3문항   - 기초 확인
보통(Medium): 6문항 - 중심 문항
어려움(Hard): 5문항 - 심화/다음 단계
```

## 📚 콘텐츠 특징

### 읽기 영역 (5가지 도메인)
1. **어휘력 (VOC)**: 맥락에서의 어휘 이해
   - 1-2단계: 기본 어휘 파악
   - 3-4단계: 뉘앙스 차이, 비유적 의미
   - 5-7단계: 사회문화적/역사적 의미

2. **사실적 이해 (LIT)**: 글의 주제와 내용
   - 1-2단계: 간단한 사실 파악
   - 3-4단계: 글의 구조와 흐름
   - 5-7단계: 복합적 구조 분석

3. **추론적 이해 (INF)**: 암시된 의미 파악
   - 1-2단계: 기초 추론
   - 3-4단계: 글쓴이 의도
   - 5-7단계: 함축적/맥락적 의미

4. **비판적 이해 (CRI)**: 주장의 타당성 평가
   - 3-4단계: 주장 평가 시작
   - 5-7단계: 강점/약점 분석

5. **창의적 이해 (CRE)**: 의미 확장과 해석
   - 3-4단계: 다른 영역 적용
   - 5-7단계: 의미 확장

### 글쓰기 영역 (5가지 영역)
1. **내용 생성 (IDE)**: 주장과 근거 개발
2. **글 조직 (ORG)**: 논리적 구조 구성
3. **표현 (EXP)**: 어휘와 표현의 효과성
4. **어법과 규범 (GRA)**: 문법과 문법적 정확성
5. **고쳐쓰기 (REV)**: 글의 개선과 수정
