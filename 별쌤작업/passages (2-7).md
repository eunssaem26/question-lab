| Level | 2 |
| Genre | 설명문 |
| Chars | 215 |
| Source | AI생성 |
| Status | draft |

> 해가 떠서 땅이 뜨거워지면 땅 위의 공기도 따뜻해집니다. 따뜻한 공기는 가벼워져서 위로 올라갑니다. 그러면 그 빈자리를 채우기 위해 주변의 차가운 공기가 밀려옵니다.
>

| Level | 2 |
| Genre | 설명문 |
| Chars | 252 |
| Source | AI생성 |
| Status | draft |

> 우리 몸에는 약 206개의 뼈가 있습니다. 뼈는 몸을 지탱하고, 심장이나 폐 같은 중요한 장기를 보호하는 역할을 합니다. 예를 들어 갈비뼈는 심장과 폐를 감싸서 지켜 줍니다.

>
> 그래서 우유나 멸치 같은 음식을 잘 먹고, 밖에서 운동하며 햇빛을 쬐는 것이 뼈 건강에 좋습니다.

| Source | AI생성 |
| Status | draft |

> 우리 학교에 텃밭을 만들어야 한다고 생각합니다. 지난달 한 뉴스에서 "텃밭 가꾸기를 한 학생들이 채소를 더 잘 먹게 되었다"는 연구 결과가 나왔습니다. 이 연구는 한 대학교 연구팀이 500명의 학생을 대상으로 1년간 조사한 것입니다.

> 텃밭 가꾸기는 학생들에게 좋은 경험이 됩니다. 하지만 어떤 정보를 믿을 수 있는지 잘 살펴보는 것도 중요합니다.

## P-2-STR-001 | 낮과 밤이 생기는 까닭

| 항목 | 내용 |
|---|---|
| Level | 2 |
| Genre | 설명문 |
| Chars | 232 |
| Source | AI생성 |
| Status | draft |

> 여러분은 왜 낮과 밤이 생기는지 궁금한 적이 있나요? 이것은 지구가 하루에 한 바퀴씩 스스로 도는 '자전' 때문입니다.
>
> 지구가 자전하면서 해를 향한 쪽은 낮이 됩니다. 반대편은 해가 비치지 않아 밤이 됩니다. 지구가 쉬지 않고 돌기 때문에, 낮과 밤이 규칙적으로 바뀌는 것입니다.
>
> 따라서 낮과 밤은 지구의 자전 덕분에 생기는 자연 현상입니다. 만약 지구가 돌지 않는다면, 한쪽은 늘 낮이고 다른 쪽은 늘 밤일 것입니다.
