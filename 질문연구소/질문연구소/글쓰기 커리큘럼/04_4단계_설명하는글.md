# 4단계: 설명하는 글

## 기본 정보

| 항목 | 내용 |
|------|------|
| **대상** | 초등 5~6학년 (또는 3단계 이수자) |
| **총 차시** | 14차시 |
| **차시당 학습 시간** | 약 45분 |
| **핵심 질문** | "독자가 모르는 것을 어떻게 하면 정확하고 흥미롭게 전달할 수 있을까?" |
| **진급 조건** | 최종 과제 루브릭 합산 15점 이상 (20점 만점) |

---

## 단계 목표

이 단계를 마치면 학습자는:
- 설명문의 목적(정보 전달)과 특성을 이해한다
- 정의, 예시, 비교·대조, 원인·결과, 순서·절차 5가지 설명 방법을 구별하고 활용한다
- 조사한 정보를 자신의 말로 바꿔 쓸 수 있다 (출처 표기 포함)
- 전문 용어를 독자 수준에 맞게 쉽게 풀어쓸 수 있다
- 600~800자 분량의 설명문을 완성할 수 있다

## 핵심 개념

### 설명 방법 5가지

| 방법 | 신호어 | 예시 |
|------|--------|------|
| **정의** | "~란 ~이다", "~을 ~라고 한다" | "광합성이란 식물이 햇빛을 이용해 영양분을 만드는 과정이다." |
| **예시** | "예를 들어", "대표적으로" | "예를 들어 개미, 벌, 나비 등이 곤충에 해당한다." |
| **비교·대조** | "~와 달리", "반면에", "공통적으로" | "개구리는 물과 육지 모두에서 살지만, 도마뱀은 육지에서만 산다." |
| **원인·결과** | "왜냐하면", "그 결과", "~때문에" | "지구 온난화로 인해 해수면이 높아지고 있다." |
| **순서·절차** | "먼저", "다음으로", "마지막으로" | "먼저 재료를 준비한다. 다음으로 반죽을 만든다." |

### 설명문 vs 주장문

| 항목 | 설명문 | 주장문 |
|------|--------|--------|
| **목적** | 정보 전달, 이해 돕기 | 독자 설득 |
| **어조** | 객관적, 중립적 | 주관적, 설득적 |
| **내용** | 사실과 정보 | 주장과 근거 |
| **문체** | "~이다", "~한다" | "~해야 한다", "~이 중요하다" |

## 차시별 학습 내용

### 차시 1: 설명문이란 무엇인가

**① 개념 읽기**
- 핵심 내용: 설명문, 주장문, 이야기 글 — 무엇이 다른가

**② READ — 예시 분석**
- 같은 주제(환경 보호)로 쓴 설명문 vs 주장문 비교
- 분석 질문:
  - "두 글의 차이점을 3가지 찾아보세요."

**③ TRY — 연습 문제**
- 10개의 문장이 설명문체인지 주장문체인지 분류하기

**④ WRITE**
- 과제: 내가 잘 아는 것(게임, 요리, 운동 등) 하나를 설명하는 문장 5개 쓰기
  - 내 의견 없이 사실만 전달
- 자기 체크리스트:
  - □ "나는 생각한다", "해야 한다" 같은 표현이 없다
  - □ 모든 문장이 사실 정보를 담는다

**⑤ FEEDBACK** — AI 확인: 주관적 표현 감지

**⑥ REVISE** → 합격 시 차시 2 잠금 해제

### 차시 2: 설명 방법 — 정의와 예시

**① 개념 읽기**
- 핵심 내용: 정의와 예시로 설명하는 법

**② READ — 예시 분석**
- 정의 + 예시로 구성된 설명 단락 2개
- 분석 질문:
  - "정의 문장과 예시 문장을 각각 찾아 표시해보세요."

**③ TRY — 연습 문제**
- 과학 교과서 개념 1개를 쉬운 말로 정의 바꾸기 (2문항)
- 정의 문장에 이어지는 예시 문장 쓰기 (2문항)

**④ WRITE**
- 과제: "내 친구에게 설명하는 〇〇이란 무엇인가" (정의+예시 포함, 1단락)

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 3 잠금 해제

### 차시 3: 설명 방법 — 비교·대조

**① 개념 읽기**
- 핵심 내용: 비교와 대조 — 같은 점과 다른 점으로 설명하기

**② READ — 예시 분석**
- 벤다이어그램 → 비교 단락으로 변환된 예시
- 분석 질문: "비교 단락에서 공통점과 차이점이 균형 있게 다뤄졌나요?"

**③ TRY — 연습 문제**
- 벤다이어그램 채우기: 개 vs 고양이 (또는 여름 vs 겨울)

**④ WRITE**
- 과제: 비교·대조 단락 1개 작성 (자유 주제)
  - 반드시 공통점 1개 + 차이점 2개 이상 포함
- 자기 체크리스트:
  - □ "~와 달리", "반면에" 등 신호어를 사용했다
  - □ 두 대상이 균형 있게 다뤄진다

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 4 잠금 해제

### 차시 4: 설명 방법 — 원인·결과

**① 개념 읽기**
- 핵심 내용: 원인과 결과로 설명하기

**② READ — 예시 분석**
- 원인→결과 흐름의 설명 단락 2개
- 분석 질문: "원인과 결과가 논리적으로 연결되나요? 빠진 연결고리가 없나요?"

**③ TRY — 연습 문제**
- 원인만 주어졌을 때 결과 쓰기 / 결과만 주어졌을 때 원인 쓰기 (각 2문항)

**④ WRITE**
- 과제: 원인·결과 구조로 설명하는 단락 1개
  - 예: 미세먼지의 원인과 영향, 규칙적인 운동의 효과 등

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 5 잠금 해제

**① 개념 읽기**

## 그래픽 오거나이저

### 설명문 계획표
```
┌─────────────────────────────────────────────┐
│  주제:                                      │
│  독자 (누구에게 설명할 것인가?):            │
│  사용할 설명 방법:                          │
│  □정의 □예시 □비교·대조 □원인·결과 □순서│
├─────────────────────────────────────────────┤
│  서론 — 왜 이 내용을 알아야 하는가?         │
│                                             │
├─────────────────────────────────────────────┤
│  본론 단락 1          본론 단락 2           │
│  주제:                주제:               │
│  설명 방법:           설명 방법:          │
│  핵심 정보:           핵심 정보:          │
│  출처:                출처:              │
├─────────────────────────────────────────────┤
│  결론 — 핵심 정보 요약 + 독자에게 메시지    │
│                                             │
└─────────────────────────────────────────────┘
```

### 비교·대조 벤다이어그램
```
    A만의 특징       공통점       B만의 특징
   ┌─────────┐     ┌─────┐     ┌─────────┐
   │         │     │     │     │         │
   │         │     │     │     │         │
   └─────────┘     └─────┘     └─────────┘
```

## 평가 루브릭 (최종 과제)

| 평가 기준 | 4점 | 3점 | 2점 | 1점 |
|-----------|-----|-----|-----|-----|
| **객관성** | 완전히 사실 기반, 의견 배제 | 대부분 사실 | 의견과 사실 혼재 | 주로 의견 |
| **설명 방법** | 3가지 이상 적절히 혼합 | 2가지 방법 | 1가지 방법만 | 방법 없이 나열 |
| **정보 정확성** | 출처 있음, 정확 | 출처 있음 | 출처 일부 | 출처 없음 |
| **독자 고려** | 전문 용어 모두 풀이 | 대부분 풀이 | 일부 풀이 | 풀이 없음 |
| **분량/구조** | 700자 이상, 3부 구조 명확 | 600자 이상 | 500자 이상 | 500자 미만 |

**진급 기준**: 합산 15점 이상
