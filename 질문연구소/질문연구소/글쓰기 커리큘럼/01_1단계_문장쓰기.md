# 1단계: 문장 쓰기

## 기본 정보

| 항목 | 내용 |
|------|------|
| **대상** | 초등 3학년 (또는 진단 테스트에서 1단계 배정된 학습자) |
| **총 차시** | 12차시 |
| **차시당 학습 시간** | 약 40분 |
| **핵심 질문** | "온전한 문장이란 무엇인가? 내 생각을 문장 하나로 표현하려면?" |
| **진급 조건** | 최종 과제 루브릭 합산 14점 이상 (20점 만점) |

---

## 단계 목표

이 단계를 마치면 학습자는:
- 주어와 서술어가 갖춰진 완전한 문장을 쓸 수 있다
- 육하원칙을 활용해 문장을 풍부하게 확장할 수 있다
- 문장 부호(마침표, 물음표, 느낌표, 쉼표)를 올바르게 사용할 수 있다
- 3~5개의 문장을 연결해 짧은 일기나 관찰일지를 완성할 수 있다

---

## 핵심 개념

### 좋은 문장의 3가지 조건
1. **완전성** — 주어와 서술어가 있다
2. **명확성** — 읽는 사람이 단번에 이해한다
3. **구체성** — 언제, 어디서, 어떻게가 담겨 있다

### 문장 확장 공식
```
기본 문장:   [주어] + [서술어]
             "강아지가 달린다."

확장 문장:   [언제] + [어디서] + [주어] + [어떻게] + [서술어]
             "오늘 아침 공원에서 우리 강아지가 신나게 달렸다."
```

## 차시별 학습 내용

### 차시 1: 문장이란 무엇인가

**① 개념 읽기**
- 핵심 내용: 문장과 단어·어절의 차이, 완전한 문장의 조건

**② READ — 예시 분석**
- 예시 A (불완전): "빠르게 달리는." / "학교에서."
- 예시 B (완전): "아이가 빠르게 달린다." / "우리는 학교에서 공부한다."
- 분석 질문:
  - "예시 A가 문장이 아닌 이유는 무엇인가요?"
  - "예시 B에서 주어는 무엇인가요?"

**③ TRY — 연습 문제**
- 다음 중 완전한 문장에 ○표 하기 (5문항)
- 불완전한 문장에 주어 또는 서술어 추가해 완성하기 (3문항)

**④ WRITE**
- 과제: 오늘 학교에서 있었던 일 3가지를 각각 하나의 완전한 문장으로 쓰기
- 자기 체크리스트 (제출 전 필수):
  - □ 모든 문장에 주어가 있다
  - □ 모든 문장에 서술어가 있다
  - □ 마침표로 끝냈다

**⑤ FEEDBACK** — AI 확인 항목
- 주어-서술어 존재 여부
- 마침표 사용 여부
- 칭찬: 가장 자연스러운 문장 하이라이트

**⑥ REVISE**
- AI 피드백을 보고 어색한 문장 수정
- 수정 후 재제출 → 합격 시 차시 2 잠금 해제

---

### 차시 2: 문장을 구체적으로 만들기

**① 개념 읽기**
- 핵심 내용: 육하원칙 6가지(누가·언제·어디서·무엇을·어떻게·왜)와 문장 확장 방법

**② READ — 예시 분석**
- 전/후 비교: "강아지가 짖었다." → "어젯밤 10시에 현관 앞에서 우리 강아지가 낯선 사람을 보고 크게 짖었다."
- 분석 질문:
  - "확장된 문장에 추가된 정보 3가지를 찾아보세요."
  - "이 문장이 더 잘 전달되는 이유는 무엇인가요?"

**③ TRY — 연습 문제**
- 짧은 문장을 받아 육하원칙 요소 1~2개 추가하기 (3문항)
- AI가 즉시 채점: 추가된 정보가 문장과 자연스럽게 어울리는지 확인

**④ WRITE**
- 과제: 오늘 먹은 음식에 대해 육하원칙이 포함된 문장 3개 쓰기
- 자기 체크리스트:
  - □ 각 문장에 육하원칙 요소가 2개 이상 포함되어 있다
  - □ 문장이 자연스럽게 읽힌다

**⑤ FEEDBACK** — AI 확인 항목
- 육하원칙 요소 포함 개수
- 문장 자연스러움 (어색한 나열 여부)
- 칭찬: 가장 구체적인 문장 하이라이트

**⑥ REVISE** → 합격 시 차시 3 잠금 해제

---

### 차시 3: 문장 부호 바르게 쓰기

**① 개념 읽기**
- 핵심 내용: 마침표·물음표·느낌표·쉼표의 역할 구별과 사용법

**② READ — 예시 분석**
- 문장 부호가 없는 글과 있는 글 비교
- 분석 질문:
  - "쉼표가 없으면 이 문장이 어떻게 읽히나요?"

**③ TRY — 연습 문제**
- 문장 부호가 빠진 글에 알맞은 문장 부호 넣기 (8문항)

**④ WRITE**
- 과제: 오늘 가장 궁금했던 것 질문 문장 2개 + 오늘 가장 기뻤던 것 느낌 문장 2개 쓰기
- 자기 체크리스트:
  - □ 질문 문장 끝에 물음표를 썼다
  - □ 느낌 문장 끝에 느낌표 또는 마침표를 썼다
  - □ 긴 문장에 쉼표를 사용했다

**⑤ FEEDBACK** — AI 확인 항목
- 문장 부호 종류별 올바른 사용 여부

**⑥ REVISE** → 합격 시 차시 4 잠금 해제

---

### 차시 4: 맞춤법과 띄어쓰기 기초

**① 개념 읽기**
- 핵심 내용: 돼/되, 안/않, 며칠, 됐다, 기본 띄어쓰기

**② READ — 예시 분석**
- 맞춤법 오류가 있는 글 → 오류 찾기
- 분석 질문:
  - "이 글에서 틀린 곳 3개를 찾아보세요."

**③ TRY — 연습 문제**
- 맞춤법 오류 수정하기 (10문항)
- AI가 즉시 채점

**④ WRITE**
- 과제: 어제 한 일을 4문장으로 쓰기 (맞춤법에 집중)
- 자기 체크리스트:
  - □ 돼/되를 올바르게 사용했다
  - □ 안/않을 올바르게 사용했다
  - □ 단어 사이 띄어쓰기를 했다

**⑤ FEEDBACK** — AI 확인 항목
- 맞춤법·띄어쓰기 오류 목록 (오류별 설명 포함)

**⑥ REVISE** → 합격 시 차시 5 잠금 해제

---

### 차시 5: 관찰 일지 쓰기

**① 개념 읽기**
- 핵심 내용: 관찰 일지의 구성 요소(대상·날짜·관찰 내용), 5감(시각·청각·후각·촉각) 활용법

**② READ — 예시 분석**
- 식물 관찰 일지 예시 (5문장)
- 분석 질문:
  - "이 관찰 일지에서 가장 구체적인 문장은 어떤 것인가요? 왜 그 문장이 구체적인가요?"

**③ TRY — 연습 문제**
- 관찰 일지 그래픽 오거나이저 채우기
  - 오늘 관찰한 것: ( )
  - 어떻게 보였나: ( )
  - 어떤 느낌이었나: ( )

**④ WRITE**
- 과제: 관찰 일지 쓰기 (5문장 이상)
  - 관찰 대상: 식물, 날씨, 동물, 가족 중 자유 선택
- 자기 체크리스트:
  - □ 5문장 이상 썼다
  - □ 각 문장에 주어-서술어가 있다
  - □ 구체적인 정보(색, 크기, 소리 등)가 담겨 있다

**⑤ FEEDBACK** — AI 확인 항목
- 분량 (5문장 이상)
- 구체적 묘사 표현 확인

**⑥ REVISE** → 합격 시 차시 6 잠금 해제

---

### 차시 6~8: 일기 쓰기 (3차시)

#### 차시 6: 일기 구조 배우기

**① 개념 읽기**
- 핵심 내용: 일기의 3구성 요소(날짜·날씨, 있었던 일, 느낌·생각), 평범한 일기와 좋은 일기의 차이

**② READ — 예시 분석**
- 평범한 일기 예시 vs 잘 쓴 일기 예시 비교
- 분석 질문:
  - "잘 쓴 일기가 더 좋은 이유 2가지를 써보세요."

**③ TRY**
- 일기 계획표(그래픽 오거나이저) 채우기
  - 날짜/날씨: ( )
  - 오늘 있었던 일: ( )
  - 가장 기억에 남는 순간: ( )
  - 그때 어떤 느낌이었나: ( )
  - 왜 그런 느낌이었나: ( )

**④ WRITE**
- 과제: 계획표를 바탕으로 일기 초안 (6문장 이상)

**⑤ FEEDBACK** — AI 확인 항목
- 있었던 일 + 느낌/생각 모두 포함 여부
- 구체적 표현 확인

**⑥ REVISE** → 합격 시 차시 7 잠금 해제

---

#### 차시 7: 일기 — 느낌 표현하기

**① 개념 읽기**
- 핵심 내용: 감정 직접 표현(Tell) vs 간접 표현(Show), 감정별 몸의 반응·행동 표현 사례

**② READ — 예시 분석**
- 감정 표현 전/후 비교: "슬펐다" → "눈물이 날 것 같았다. 아무 말도 하기 싫었다."

**③ TRY**
- "행복했다", "무서웠다", "지루했다"를 구체적 표현으로 바꾸기 (3문항)

**④ WRITE**
- 과제: 오늘 일기에서 내 감정이 드러나는 문장을 3개 이상 포함해 일기 쓰기

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 8 잠금 해제

---

#### 차시 8: 일기 — 자유 일기

**① 개념 읽기**
- 핵심 내용: 차시 06~07 복습 — 일기 3구성 + 감정 간접 표현 총정리, 좋은 일기 완성 예시

**② READ — 예시 분석**
- 자유 일기 예시 읽기
- 분석 질문:
  - "이 일기에서 가장 생생한 문장은 무엇인가요?"

**③ TRY**
- 오늘 쓸 일기의 장면 1개와 넣고 싶은 표현 3개 적기

**④ WRITE**
- 과제: 자유 주제 일기 (8문장 이상)
  - 배운 것 모두 적용: 완전한 문장, 구체성, 문장 부호, 맞춤법, 느낌 표현

**⑤ FEEDBACK** — AI가 1단계 전체 기준으로 종합 평가

**⑥ REVISE** → 합격 시 차시 9 잠금 해제

---

### 차시 9~11: 관찰 일지 심화 (3차시)

#### 차시 9: 감각으로 관찰하기

**① 개념 읽기**
- 핵심 내용: 5감(시각·청각·후각·촉각·미각) 관찰 방법, 단일 감각 vs 다감각 표현 비교

**② READ — 예시 분석**
- 시각만 사용한 관찰 vs 5감 사용한 관찰 비교
- 분석 질문: "이 두 관찰 글 중 어느 것이 더 생생한가? 왜인가?"

**③ TRY**
- 사진 1장을 보고 5감 관찰 오거나이저 채우기
  - 보이는 것 / 들릴 것 같은 소리 / 냄새 / 질감 / 맛

**④ WRITE**
- 과제: 5감 중 3가지 이상을 사용한 관찰 일지 쓰기 (6문장 이상)

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 10 잠금 해제

---

#### 차시 10: 변화 관찰하기

**① 개념 읽기**
- 핵심 내용: 변화 관찰의 의미, 비교 표현 방법("어제는 ~였는데 오늘은 ~")

**② READ — 예시 분석**
- 같은 대상을 두 번 관찰한 예시 읽기
- 분석 질문:
  - "어느 문장이 변화를 가장 잘 보여주나요?"
  - "어제와 오늘을 비교하는 표현을 찾아보세요."

**③ TRY**
- 두 장면을 비교하는 문장 2개 완성하기
- 예: "어제는 ~였는데 오늘은 ~이다."

**④ WRITE**
- 과제: 같은 대상을 이틀 관찰한 내용 비교해서 쓰기 (6문장 이상)
  - "어제는 ~였는데, 오늘은 ~이다."

**⑤ FEEDBACK**
- 비교 표현 포함 여부
- 두 시점 모두 구체적으로 썼는지 확인

**⑥ REVISE** → 합격 시 차시 11 잠금 해제

---

### 차시 11~12: 단계 마무리

#### 차시 11: 최종 과제 준비

**① 개념 읽기**
- 핵심 내용: 1단계 전체 학습 내용 복습 표(차시 01~10 핵심 정리)

**② READ — 예시 분석**
- 최종 과제 예시 2편 비교: 기준 충족 글 / 기준 미달 글
- 분석 질문:
  - "합격하는 글에 들어간 요소 3가지를 찾아보세요."

**③ TRY**
- 1단계 핵심 개념 복습 퀴즈 (10문항)
- 80점 이상 통과 → 차시 12 잠금 해제

**④ WRITE**
- 최종 과제 계획표 작성
  - 언제, 어디서, 누구와 있었던 일인지
  - 느낌·생각 문장 2개 이상
  - 감각 표현 1개 이상

**⑤ FEEDBACK**
- 계획표에 빠진 요소 안내

**⑥ REVISE**
- 빠진 요소를 보완한 뒤 차시 12 잠금 해제

---

#### 차시 12: 최종 과제

**① 개념 읽기**
- 핵심 내용: 1단계 총복습 표, 좋은 일기 완성 예시(나쁜 예/좋은 예 대비)

**② READ — 예시 분석**
- 합격 예시 1편 읽고 좋은 점 표시하기

**③ TRY**
- 제출 전 자가 점검 5문항

**④ WRITE — 최종 과제**
- 과제: 자유 주제 일기 (8문장 이상)
  - 모든 1단계 기준을 스스로 적용

**자기 평가 (제출 전 필수)**
- □ 모든 문장에 주어-서술어가 있다
- □ 육하원칙 요소가 문장에 포함되어 있다
- □ 문장 부호를 올바르게 사용했다
- □ 맞춤법·띄어쓰기를 확인했다
- □ 느낌이나 생각이 구체적으로 표현되어 있다
- □ 8문장 이상 썼다

**⑤ FEEDBACK** — AI 종합 평가 (공통 루브릭 5개 항목)

**⑥ REVISE** — 14점 이상 → **2단계 잠금 해제**
  - 미달 시: AI가 부족한 항목 안내 → 재작성 → 재제출
  - 3회 이상 미달 시: 은쌤 상담 버튼 활성화

## 평가 루브릭 (최종 과제)

| 평가 기준 | 4점 | 3점 | 2점 | 1점 |
|-----------|-----|-----|-----|-----|
| **유창성** | 8문장 이상을 끊기지 않게 자연스럽게 이어 썼다 | 8문장 이상이나 일부 반복이 있다 | 6~7문장, 전개가 자주 끊긴다 | 5문장 이하이거나 글 흐름이 매우 약하다 |
| **구조** | 날짜·날씨, 있었던 일, 느낌·생각, 마무리가 모두 드러난다 | 요소 3가지는 분명하다 | 요소 2가지만 드러난다 | 사건 나열에 그친다 |
| **내용** | 육하원칙 요소 3개 이상, 상황이 구체적이다 | 육하원칙 요소 2개, 상황이 대체로 보인다 | 육하원칙 요소 1개, 상황이 막연하다 | 상황 설명이 매우 부족하다 |
| **어휘/표현** | 감정 간접 표현 2개 이상 + 감각 표현 1개 이상 | 감정 간접 표현 1개 또는 감각 표현 1개가 있다 | 느낌은 있으나 표현이 막연하다 | "좋았다/슬펐다" 같은 직접 표현에만 의존한다 |
| **관례** | 맞춤법·띄어쓰기·문장 부호 오류가 거의 없다 | 1~2개 오류 | 3~5개 오류 | 6개 이상 오류 |

**진급 기준**: 합산 14점 이상
