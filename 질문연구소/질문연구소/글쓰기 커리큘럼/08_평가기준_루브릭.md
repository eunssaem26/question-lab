# 평가 기준 및 루브릭 (전체 단계 통합)

## 평가의 원칙

1. **과정 평가 우선** — 최종 결과물보다 수정 과정과 성장을 중시
2. **기준 참조 평가** — 다른 학생과 비교하지 않고, 목표 기준에 도달했는지 확인
3. **즉각 피드백** — AI가 제출 즉시 루브릭 기반 피드백을 제공
4. **자기 평가 필수** — 모든 최종 과제 제출 전 자기 평가 체크리스트를 완성해야 제출 버튼이 활성화됨

---

## 공통 평가 기준 5가지

모든 단계에서 아래 5가지 기준으로 평가합니다. **단계마다 기대 수준이 다릅니다.**

| 기준 | 정의 |
|------|------|
| **유창성** | 생각을 글로 자연스럽게 옮기는 능력 |
| **구조** | 글 전체의 조직과 단락 구성 |
| **내용** | 아이디어의 구체성, 풍부함, 목적 부합성 |
| **어휘** | 어휘 선택의 다양성, 정확성, 맥락 적합성 |
| **관례** | 맞춤법, 띄어쓰기, 문장 부호의 정확성 |

## 단계별 기대 수준 요약

### 분량 기준

| 단계 | 최종 과제 목표 분량 |
|------|-------------------|
| 1단계 | 8문장 이상 |
| 2단계 | 2단락 이상 (단락당 5~6문장) |
| 3단계 | 500자 이상 |
| 4단계 | 700자 이상 |
| 5단계 | 900자 이상 |
| 6단계 | 1100자 이상 |
| 7단계 | 1400자 이상 |

### 구조 기준

| 단계 | 기대 구조 |
|------|----------|
| 1단계 | 날짜·날씨, 있었던 일, 느낌·생각, 마무리가 드러나는 짧은 글 |
| 2단계 | 중심 문장 + 뒷받침 3개(방법 2가지 이상) + 마무리 문장 |
| 3단계 | 서론(도입 방법)-본론(2단락)-결론 |
| 4단계 | 3부 구조 + 설명 방법 2가지 이상 |
| 5단계 | 발단-전개-절정-결말 4막 구조 |
| 6단계 | 도입-근거1(ARE)-근거2(ARE)-반박고려-결론 5단 구조 |
| 7단계 | 목적·독자·형식에 맞는 자율 구조 |

### 내용 기준

| 단계 | 핵심 내용 요건 |
|------|--------------|
| 1단계 | 있었던 일 + 느낌·생각 포함, 육하원칙 요소 2개 이상 |
| 2단계 | 뒷받침 방법 2가지 이상 혼합 |
| 3단계 | 서론 도입 방법 의도적 선택 |
| 4단계 | 설명 방법 2가지 이상, 출처 1개 이상 |
| 5단계 | Show 기법 3곳 이상, 대화 포함 |
| 6단계 | ARE 근거 2개, 반박 고려 포함 |
| 7단계 | 1~6단계 기법 3가지 이상 통합 적용 |

## 과제 유형별 평가 비중

| 과제 유형 | 비중 | 평가 방식 |
|-----------|------|----------|
| **연습 문제(TRY)** | 통과/미통과 | AI 자동 채점, 즉시 재시도 가능 |
| **차시별 쓰기(WRITE)** | 30% | AI 루브릭 채점 + 합격 기준 도달 시 다음 차시 해제 |
| **최종 과제** | 70% | AI 루브릭 채점 + 단계 진급 결정 |

## 자기 평가 체크리스트 (단계 공통)

최종 과제 제출 전 아래 항목을 스스로 확인합니다. **모든 항목에 체크해야 제출 가능**하지만, 문항은 단계 수준에 맞게 조금씩 달라질 수 있습니다.

- 1단계: 문장 완전성, 구체성, 느낌·생각 표현, 맞춤법·문장 부호 중심
- 2단계 이상: 단락 구조, 독자 고려, 글 전체 조직까지 포함

```
[ 내용 ]
□ 이 글에서 말하고 싶은 내용이 분명하다
□ 이 단계에서 필요한 내용이 충분히 들어 있다
□ 읽는 사람이 이해할 수 있게 구체적으로 썼다

[ 구조 ]
□ 이 단계에서 배운 구조를 적용했다
□ 글의 처음, 가운데, 끝이 자연스럽게 이어진다

[ 과정 ]
□ 계획표를 작성하고 글을 썼다
□ AI 피드백을 보고 의미 있는 수정을 했다
□ 처음 초안과 최종본이 다르다

[ 성장 ]
□ 이전 과제에 비해 나아진 점이 있다
□ 아직 더 노력이 필요한 부분이 있다
   → 어떤 부분인가? (직접 입력)
```

## AI 피드백 구성 방식

모든 AI 피드백은 아래 3-파트 구조를 따릅니다.

### 1. 칭찬 (Glow)
- 이 글에서 가장 효과적이었던 것
- 구체적으로 어떤 문장/부분인지 지목

### 2. 개선점 (Grow)
- 루브릭에서 기준 미달인 항목
- 구체적으로 어떤 부분을, 어떻게 수정하면 좋은지 안내

### 3. 다음 단계 힌트
- "이 글을 더 좋게 만들려면 딱 한 가지만 수정한다면?"
- 학습자가 스스로 우선순위를 정하도록 돕는 안내

## 은쌤 상담 활성화 기준

| 조건 | 상담 방식 |
|------|----------|
| 동일 과제 3회 이상 미달 | "은쌤에게 질문하기" 버튼 자동 활성화 |
| 7일 이상 미진행 | 격려 메시지 + 상담 버튼 제공 |
| 학생 직접 요청 | "이해가 안 돼요" 버튼으로 언제든 요청 가능 |
| AI 판단 불가 | 자동으로 은쌤에게 글 전달 + 피드백 대기 |

## 포트폴리오 구성 가이드

전 과정을 마친 학습자의 포트폴리오 구성:

| 항목 | 내용 |
|------|------|
| **단계별 대표작** | 각 단계에서 최고 점수를 받은 글 1편씩 (자동 선정) |
| **자기 코멘트** | 각 글에 대해 100자 이내 직접 작성 |
| **성장 비교** | 1단계 첫 번째 글 vs 7단계 최종 글 나란히 보기 |
| **성장 에세이** | "글쓰기를 통해 내가 달라진 것" (선택, 500자 이상) |
| **수료 배지** | 전 과정 이수 시 자동 발급 |
