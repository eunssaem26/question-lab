# 차시 06 | 완전한 단락 1개 완성

**소요 시간**: 약 40분  
**학습 목표**: 중심 문장 + 뒷받침 3개(방법 2가지 이상) + 마무리 문장으로 완전한 단락을 혼자 완성할 수 있다

---

## ① 개념 읽기

> 아래 내용을 천천히 읽어보세요.

### 완전한 단락의 구조 — 햄버거 모델

지금까지 배운 내용을 모두 합쳐보면 이렇습니다.

```
        ┌──────────────────────────────────┐
        │        중심 문장                 │  ← 이 단락에서 말하고 싶은 핵심
        ├──────────────────────────────────┤
        │       뒷받침 문장 1              │  ← 예시 / 이유 / 경험 / 비교
        │       뒷받침 문장 2              │
        │       뒷받침 문장 3              │
        ├──────────────────────────────────┤
        │        마무리 문장               │  ← 반복 아닌 의미 있는 마무리
        └──────────────────────────────────┘
```

### 완전한 단락 체크리스트

오늘은 처음부터 끝까지 혼자서 완전한 단락을 씁니다.  
쓰기 전에 아래 조건을 기억하세요.

| 조건 | 확인 방법 |
|------|---------|
| 중심 문장이 명확하다 | "무엇에 대해 쓰는 단락인지" 첫 문장을 보면 알 수 있어야 함 |
| 뒷받침 문장이 3개 있다 | 중심 문장과 마무리 문장 사이에 3개의 뒷받침 문장이 있어야 함 |
| 뒷받침 방법 2가지 이상 혼합 | 각 뒷받침 문장에 서로 다른 신호어(예를 들어/왜냐하면/한 번은/~와 달리)가 있어야 함 |
| 마무리 문장이 반복이 아니다 | 중심 문장을 그대로 다시 쓰지 않아야 함 |
| 맞춤법·문장 부호가 올바르다 | 모든 문장 끝에 마침표(.), 물음표(?) 등이 있어야 함 |

---

### 모범 단락 예시

> 주제: 우리 집 고양이

```
우리 집 고양이 나비는 내가 가장 좋아하는 가족이다.
왜냐하면 나비는 내가 기분이 나쁠 때 꼭 옆에 와서 앉아주기 때문이다.
예를 들어 시험을 못 봐서 속상한 날에도 나비는 내 무릎 위에서 조용히 가르랑거렸다.
한 번은 나비가 내 숙제 위에 올라앉아서 나를 웃게 만든 적도 있었다.
강아지와 달리, 고양이는 조용하고 독립적이어서 작은 집에서도 함께 살기 좋다.
이처럼 나비는 내게 단순한 반려동물이 아니라, 언제나 곁에 있어주는 친구 같은 존재다.
```

> **분석**
> - 중심 문장: "우리 집 고양이 나비는 내가 가장 좋아하는 가족이다."
> - 뒷받침 1 (이유): "왜냐하면~"
> - 뒷받침 2 (예시): "예를 들어~"
> - 뒷받침 3 (경험): "한 번은~"
> - 뒷받침 4 (비교): "강아지와 달리~"
> - 마무리 문장: "이처럼~" (의미 확장)

## ② 예시 분석

> 아래 단락을 읽고 분석 질문에 답해보세요.

---

**분석 대상 단락**

```
우리 동네 도서관은 정말 훌륭한 공간이다.
왜냐하면 조용한 환경에서 집중해서 공부하거나 책을 읽을 수 있기 때문이다.
예를 들어 방학 때마다 도서관에 가면 수백 권의 책 중에서 원하는 것을 골라 읽을 수 있다.
카페에 비해, 도서관은 무료로 이용할 수 있고 오랫동안 있어도 눈치 보지 않아도 된다.
그래서 도서관은 나에게 특별한 공간이다.
```

---

**분석 질문 1** (주관식)

> 이 단락의 구조를 아래 표에 채워보세요.

| 구성 요소 | 해당 문장 | 사용한 방법 (해당하면) |
|---------|---------|---------------------|
| 중심 문장 | | |
| 뒷받침 1 | | |
| 뒷받침 2 | | |
| 뒷받침 3 | | |
| 마무리 문장 | | |

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 3점 | 5개 구성 요소 모두 정확히 찾고, 뒷받침 방법 이름도 정확히 씀 |
| 2점 | 4개 이상 정확, 방법 이름 1~2개 틀림 |
| 1점 | 3개 이하 정확 |
| 0점 | 무응답 |

> **정답**:
> - 중심 문장: "우리 동네 도서관은 정말 훌륭한 공간이다."
> - 뒷받침 1: "왜냐하면~" (이유)
> - 뒷받침 2: "예를 들어~" (예시)
> - 뒷받침 3: "카페에 비해~" (비교)
> - 마무리 문장: "그래서 도서관은 나에게 특별한 공간이다."

---

**분석 질문 2** (주관식)

> 이 단락에서 아쉬운 점이 있다면 무엇인가요? (마무리 문장에 집중해서 생각해보세요)

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 2점 | "마무리 문장이 중심 문장과 비슷하다", "의미 확장이 없다", "경험 뒷받침이 없다" 등 구체적 지적 |
| 1점 | "마무리가 약하다" 등 모호한 지적 |
| 0점 | 무응답 |

> **모범 답안**: "마무리 문장('그래서 도서관은 나에게 특별한 공간이다')이 중심 문장과 너무 비슷하다. '이처럼 도서관은 공부와 휴식을 동시에 줄 수 있는 보물 같은 장소다' 처럼 의미를 확장하면 더 좋다."

## ③ TRY — 단락 완성하기

> 오늘의 TRY는 연습 문제 대신 단락 완성 훈련으로 진행합니다.

---

**문제 1**: 아래 단락에서 비어있는 부분을 채워 완전한 단락으로 만들어보세요.

```
[중심 문장]: 나는 아침 일찍 일어나는 것이 하루를 잘 시작하는 방법이라고 생각한다.

[뒷받침 1 — 이유]: ___________________________________________

[뒷받침 2 — 경험]: ___________________________________________

[뒷받침 3 — 비교]: ___________________________________________

[마무리 문장]: ___________________________________________
```

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 8점 | 이유·경험·비교 뒷받침 모두 신호어 포함 + 중심 문장과 연결 + 마무리 문장이 반복 아님 |
| 6점 | 3개 뒷받침 신호어 포함, 마무리 문장 약간 반복 |
| 4점 | 2개 뒷받침 완성, 마무리 있음 |
| 2점 | 1개 뒷받침만 완성 |
| 0점 | 무응답 |

> **모범 답안 예시**:
> - 이유: "왜냐하면 아침 시간이 가장 집중이 잘 되고 하루 계획을 세울 여유가 있기 때문이다."
> - 경험: "한 번은 7시에 일어나 독서와 스트레칭을 하고 학교에 갔더니 하루 종일 기분이 좋았다."
> - 비교: "늦게 일어나는 날에 비해, 일찍 일어난 날은 아침 식사도 여유 있게 할 수 있다."
> - 마무리: "이처럼 이른 아침은 하루를 활기차게 만들어주는 소중한 시간이다."

## ⑤ AI 피드백 기준

### AI가 단락 전체 구조를 종합 평가합니다

| 평가 항목 | 확인 내용 | 피드백 메시지 예시 |
|---------|---------|------------------|
| 중심 문장 명확성 | 첫 문장이 단락 주제를 명확히 제시하는지 | "중심 문장이 조금 넓어요. '나는 ___(구체적인 것)을/를 좋아한다' 형태로 좁혀보세요." |
| 뒷받침 개수 | 뒷받침 문장이 3개인지 | "뒷받침 문장이 2개밖에 없어요. 하나 더 추가해볼까요?" |
| 방법 혼합 | 2가지 이상 방법 사용 여부 | "예시 방법만 3번 사용됐어요. 이유나 경험 방법도 한번 써보세요." |
| 마무리 문장 | 반복이 아닌지, 단락과 연결되는지 | "마무리 문장이 중심 문장과 거의 똑같아요. '이처럼', '앞으로'로 시작해서 다르게 마무리해보세요." |
| 맞춤법·부호 | 오류 감지 | "2번째 문장 끝에 마침표가 없어요. 확인해보세요." |
| 종합 칭찬 | 모든 항목 충족 시 | "완벽한 단락이에요! 중심 문장부터 마무리 문장까지 깔끔하게 구성했고, 두 가지 뒷받침 방법이 자연스럽게 섞였어요." |

### 합격 기준
5가지 체크리스트 항목 모두 충족 → **차시 07 잠금 해제**
