# 2단계 Claude API 테스트 세트

## 목적

1단계 테스트 세트와 동일한 방식으로 2단계 AI 피드백 프롬프트의 일관성과 정확성을 검증합니다.

테스트 목표:
1. 중심 문장 / 뒷받침 / 마무리 문장 구분이 정확한가
2. 뒷받침 방법(예시/이유/경험/비교) 식별이 맞는가
3. 단락 간 접속어 흐름 판정이 맞는가
4. 1단계 평가 기준(주어, 맞춤법)을 이 단계에서 과잉 적용하지 않는가
5. 차시12 루브릭 점수가 일관적인가

---

## 케이스 1. 차시02 명확한 합격

### 테스트 목적
- 중심 문장 위치(첫 번째)와 뒷받침 연결성 체크가 정확한지 확인

### 입력 정보
- 차시: 02
- 과제: 내 친구(또는 가족)를 소개하는 단락 (중심 문장 1개 + 뒷받침 3개)

### 학생 글
```
나는 우리 형이 정말 멋지다고 생각한다.
왜냐하면 형은 무슨 일이든 포기하지 않기 때문이다.
예를 들어 형은 수영을 배울 때 3개월 동안 매일 연습했다.
또 형은 나에게 모르는 것을 물어보면 항상 친절하게 알려준다.
```

### 기대 판정
- 결과: 합격
- 핵심 이유:
  - 중심 문장이 첫 번째 문장
  - 이유 뒷받침 1개 + 예시 뒷받침 1개 + 추가 뒷받침 1개
  - 뒷받침 3개 모두 중심 문장과 연결

### 체크 포인트
- 마무리 문장 없다고 불합격시키면 안 됨 (차시02에서 평가 안 함)
- 뒷받침 방법이 2가지인지 여부로 불합격시키면 안 됨 (차시04에서 배움)

---

## 케이스 2. 차시02 명확한 불합격 — 뒷받침이 중심 문장과 무관

### 테스트 목적
- 뒷받침 연결성 체크가 되는지 확인

### 입력 정보
- 차시: 02
- 과제: 내 친구(또는 가족)를 소개하는 단락 (중심 문장 1개 + 뒷받침 3개)

### 학생 글
```
나는 내 친구 민준이가 재미있다고 생각한다.
왜냐하면 민준이는 항상 웃기 때문이다.
민준이는 키가 나보다 크다.
민준이는 어제 떡볶이를 먹었다.
```

### 기대 판정
- 결과: 아직 조금 더 다듬어봐요
- 핵심 이유:
  - 세 번째 문장(키가 크다)이 "재미있다"는 중심 문장과 연결 안 됨
  - 네 번째 문장(어제 떡볶이)이 완전히 무관

### 기대 수정 방향
- "민준이는 어제 떡볶이를 먹었다." → 삭제 또는 연결
- "민준이는 키가 나보다 크다." → 재미있다는 것과 어떻게 연결되는지 설명 추가

### 체크 포인트
- 뒷받침이 3개라고 합격시키면 안 됨 (내용 연결성 체크 필요)
- 주어 없다고 불합격시키면 안 됨 (주어 있음)

---

## 케이스 3. 차시06 아슬아슬한 합격 — 마무리 문장이 약한 경우

### 테스트 목적
- 마무리 문장이 중심 문장 반복이지만 완전히 같은 건 아닌 경계 케이스

### 입력 정보
- 차시: 06
- 과제: 완전한 단락 (중심 문장 + 뒷받침 3개(방법 2가지 이상) + 마무리 문장)

### 학생 글
```
나는 독서가 즐거운 활동이라고 생각한다.
왜냐하면 책을 읽으면 새로운 세계를 경험할 수 있기 때문이다.
예를 들어 내가 탐정 소설을 읽을 때 나도 탐정이 된 것 같은 느낌이 들었다.
독서는 수영이나 달리기와 달리 비가 와도 할 수 있다.
그래서 나는 독서가 좋은 활동이라고 생각한다.
```

### 기대 판정
- 결과: 합격 가능성이 높음 (경계 케이스)
- 핵심 이유:
  - 중심 문장 첫 번째 ✅
  - 이유 + 경험 + 비교 = 3가지 방법 ✅
  - 마무리 문장 있음 (내용이 "즐거운 활동"→"좋은 활동"으로 약간 변형됨)
  - 마무리가 단순 반복에 가깝지만 완전히 같지는 않음 → 경계

### 체크 포인트
- 마무리 문장이 너무 반복적이라고 불합격시키는 경우: 흔들림 신호
- 경험 뒷받침을 "내가 탐정이 된 것 같은 느낌" → 실제 경험 아니라고 지적하면 안 됨

## 케이스 4. 차시06 명확한 불합격 — 중심 문장이 마지막에 있음

### 테스트 목적
- 중심 문장 위치 오류를 정확히 잡는지 확인

### 입력 정보
- 차시: 06
- 과제: 완전한 단락 (중심 문장 + 뒷받침 3개(방법 2가지 이상) + 마무리 문장)

### 학생 글
```
나는 매일 아침 강아지와 산책을 한다.
한 번은 비가 오는 날 산책을 했는데, 강아지가 빗속에서 뛰어노는 것을 보고 나도 기분이 좋아졌다.
강아지는 고양이와 달리 산책을 좋아해서 같이 밖에 나갈 수 있다.
그래서 나는 강아지가 최고의 반려동물이라고 생각한다.
```

### 기대 판정
- 결과: 아직 조금 더 다듬어봐요
- 핵심 이유:
  - 중심 문장이 마지막 문장 (첫 번째여야 함)
  - 마무리 문장 없음

### 기대 수정 방향
- "강아지가 최고의 반려동물이라고 생각한다."를 첫 번째 문장으로 이동
- 마무리 문장 추가: 단순 반복이 아닌 내용 확장

### 체크 포인트
- 내용 자체가 좋다고 합격시키면 안 됨 (구조 기준 실패)
- 뒷받침 방법이 경험+비교 2가지 있다는 것은 C3 통과로 인정

## 케이스 5. 차시07 접속어 흐름 오류

### 테스트 목적
- 접속어가 있지만 흐름이 맞지 않는 경우를 잡는지 확인

### 입력 정보
- 차시: 07
- 과제: 단락 2개를 접속어로 연결

### 학생 글
```
나는 수학을 좋아한다.
왜냐하면 문제를 풀었을 때 정답이 나오면 기분이 좋기 때문이다.
예를 들어 어려운 문제를 오래 생각해서 풀었을 때 정말 뿌듯했다.

하지만 나는 미술도 좋아한다.
왜냐하면 내가 그린 그림이 완성됐을 때 뿌듯하기 때문이다.
예를 들어 작년 미술 시간에 그린 그림을 선생님이 칭찬해주셨다.
```

### 기대 판정
- 결과: 아직 조금 더 다듬어봐요
- 핵심 이유:
  - 두 단락이 모두 좋아하는 과목이므로 "하지만"이 맞지 않음 (반대가 아님)
  - "그리고" 또는 "또한"으로 바꿔야 함

### 체크 포인트
- 접속어가 있다는 이유만으로 합격시키면 안 됨
- 두 단락의 구조(중심문장+뒷받침)가 잘 됐다는 이유로 접속어 오류를 넘기면 안 됨

## 케이스 6. 차시12 명확한 불합격 (루브릭)

### 테스트 목적
- 뒷받침이 중심 문장과 연결되지 않을 때 R4 점수가 낮은지 확인
- 14점 미만일 때 합격시키지 않는지 확인

### 입력 정보
- 차시: 12
- 과제: 자유 주제 소개글 (2단락 이상, 각 단락 중심 문장 + 뒷받침 3개 + 마무리, 접속어 연결)

### 학생 글
```
나는 축구를 좋아한다.
축구는 재미있다.
나는 어제 학교에 갔다.
친구들이 운동장에서 뛰었다.
축구가 최고이다.

그리고 나는 피자도 좋아한다.
피자는 맛있다.
나는 피자를 먹은 적이 있다.
피자는 치즈가 있다.
피자가 제일 맛있다.
```

### 기대 판정
- 결과: 아직 조금 더 다듬어봐요
- 예상 점수:
  - R1(중심 문장): 2 — 있으나 불분명
  - R2(뒷받침): 1 — 방법 없이 단순 나열
  - R3(마무리): 2 — 있으나 단순 반복
  - R4(연결성): 1 — 대부분 무관한 내용
  - R5(접속어): 3 — 단락 간 "그리고" 사용
  - 예상 총점: 9점

### 체크 포인트
- 접속어(그리고)가 있다고 R5를 4점 주면 안 됨
- 문장 수가 많다고 합격시키면 안 됨

## 먼저 수정해야 하는 흔들림 신호

- 뒷받침 방법 이름(예시/이유/경험/비교)을 잘못 식별함
- 마무리 문장이 있는데 "단순 반복"과 "의미 있는 마무리" 구분을 못함
- 접속어가 있기만 하면 합격시킴 (흐름 적절성 미확인)
- 차시 범위 밖 기준(맞춤법, 주어 누락)을 주요 불합격 사유로 사용
- 중심 문장 위치(첫 번째 문장)를 확인하지 않음
