# 차시 04 | 뒷받침 방법 혼합하기

**소요 시간**: 약 40분  
**학습 목표**: 4가지 뒷받침 방법(예시·이유·경험·비교)을 구별하고, 2가지 이상을 혼합해 더 풍부한 단락을 쓸 수 있다

---

## ① 개념 읽기

> 아래 내용을 천천히 읽어보세요.

### 4가지 방법 복습

지금까지 4가지 뒷받침 방법을 모두 배웠어요.

| 방법 | 신호어 | 하는 일 |
|------|--------|--------|
| **예시** | 예를 들어, 예컨대 | 구체적인 경우를 보여줌 |
| **이유** | 왜냐하면, 그 이유는 | 근거를 설명함 |
| **경험** | 한 번은, 내가 직접 | 직접 겪은 일을 이야기함 |
| **비교** | ~와 달리, ~에 비해 | 다른 것과 대조해 명확히 함 |

---

### 왜 방법을 혼합해야 할까?

한 가지 방법만 계속 쓰면 단락이 단조로워집니다.

**한 가지 방법만 쓴 단락 (이유만)**
```
독서는 매우 중요하다.
왜냐하면 지식을 넓혀주기 때문이다.
왜냐하면 어휘력이 좋아지기 때문이다.
왜냐하면 집중력이 높아지기 때문이다.
그래서 독서는 중요하다.
```

**여러 방법을 혼합한 단락**
```
독서는 매우 중요하다.
왜냐하면 책을 읽으면 새로운 지식과 어휘를 배울 수 있기 때문이다. (이유)
예를 들어 과학책을 읽으면 우주와 생물에 대해 학교에서 배우지 못한 것들을 알 수 있다. (예시)
실제로 나는 독서를 시작한 뒤 독후감 점수가 크게 올랐다. (경험)
TV나 스마트폰과 달리, 책은 스스로 생각하는 힘을 길러준다. (비교)
그래서 꾸준히 책을 읽는 습관을 가지는 것이 중요하다.
```

> 두 번째 단락이 훨씬 풍부하고 다양한 시각에서 중심 문장을 뒷받침하고 있어요!

---

### 어떤 방법을 언제 섞을까?

정해진 규칙은 없어요. 하지만 아래 질문을 떠올려보면 도움이 됩니다.

1. 이유가 있어? → **이유** 방법 쓰기
2. 구체적인 예가 있어? → **예시** 방법 쓰기
3. 내가 직접 겪은 일이 있어? → **경험** 방법 쓰기
4. 다른 것과 비교하면 더 명확해져? → **비교** 방법 쓰기

---

## ② 예시 분석

> 아래 단락을 읽고 분석 질문에 답해보세요.

---

**예시 단락**
```
우리 학교 체육 시간은 매우 즐겁다.
왜냐하면 좋아하는 친구들과 함께 움직이며 스트레스를 풀 수 있기 때문이다.
예를 들어 지난주 체육 시간에는 피구를 했는데, 온 반이 응원하며 열정적으로 경기했다.
한 번은 내가 마지막까지 살아남아서 우리 팀을 이기게 한 적도 있었다.
반면에 교실 수업은 앉아서 듣는 시간이 많지만, 체육 시간은 직접 몸을 움직일 수 있다.
그래서 체육 시간이 있는 날은 학교에 오는 것이 더 즐겁다.
```

---

**분석 질문 1** (주관식)

> 이 단락에서 사용된 뒷받침 방법을 모두 찾고, 각 문장 옆에 방법 이름을 적어보세요.

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 3점 | 이유·예시·경험·비교 4가지 모두 정확하게 찾아 이름 표기 |
| 2점 | 3가지 정확히 찾음 |
| 1점 | 1~2가지만 찾음 |
| 0점 | 무응답 또는 모두 틀림 |

> **정답**:
> - "왜냐하면~" → **이유**
> - "예를 들어 지난주~" → **예시**
> - "한 번은 내가~" → **경험**
> - "반면에 교실 수업은~" → **비교**

---

**분석 질문 2** (주관식)

> 이 단락에서 가장 효과적이라고 생각하는 뒷받침 문장을 하나 고르고, 왜 그 문장이 효과적인지 설명해보세요.

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 2점 | 특정 뒷받침 문장을 고르고, 그 방법의 특성(생생함/설득력/명확성 등)과 연결해 이유 설명 |
| 1점 | 고른 문장은 있으나 이유가 막연함 ("재미있어서", "좋아서") |
| 0점 | 무응답 |

### [객관식] 문제 1~4: 방법 이름 붙이기

**다음 뒷받침 문장의 방법 이름을 고르세요.**

**1번**
> "예컨대 매년 가을에 열리는 독서 축제에서는 좋아하는 작가를 직접 만날 수도 있다."

- ① 이유
- ② 예시 ✅
- ③ 경험
- ④ 비교

> **정답**: ② | **해설**: "예컨대"는 "예를 들어"와 같은 예시 신호어입니다.

---

**2번**
> "독서실에 비해, 도서관은 다양한 책을 무료로 빌릴 수 있다는 장점이 있다."

- ① 이유
- ② 예시
- ③ 경험
- ④ 비교 ✅

> **정답**: ④ | **해설**: "~에 비해"는 비교 방법의 신호어입니다.

---

**3번**
> "실제로 3학년 때 발표를 두려워하던 나는, 매일 큰 소리로 읽기 연습을 한 뒤 학교 대표로 웅변 대회에 나갔다."

- ① 이유
- ② 예시
- ③ 경험 ✅
- ④ 비교

> **정답**: ③ | **해설**: "실제로"는 경험 방법의 신호어입니다.

---

**4번**
> "그 이유는 악기를 배우면 집중력과 인내심이 동시에 길러지기 때문이다."

- ① 이유 ✅
- ② 예시
- ③ 경험
- ④ 비교

> **정답**: ① | **해설**: "그 이유는~때문이다"는 이유 방법의 신호어입니다.

---

### [주관식] 문제 5: 빠진 뒷받침 방법 채우기

> 아래 단락에서 (   ) 부분에 들어갈 뒷받침 문장을 주어진 방법으로 써보세요.

**단락**
```
요리는 누구나 배울 수 있는 즐거운 취미이다.
왜냐하면 요리를 할 줄 알면 건강한 음식을 직접 만들어 먹을 수 있기 때문이다. (이유)
(    ← 여기에 예시 뒷받침 문장을 써보세요    )
(    ← 여기에 경험 뒷받침 문장을 써보세요    )
패스트푸드와 달리, 집에서 만든 음식은 재료를 선택할 수 있어 더 건강하다. (비교)
그래서 요리를 배우는 것은 평생 도움이 되는 일이다.
```

**채점 루브릭**

| 점수 | 기준 | 예시 |
|------|------|------|
| 4점 | 예시·경험 뒷받침 모두 신호어 포함, 요리와 연결 | "예를 들어 김밥, 볶음밥, 된장찌개처럼 간단한 음식부터 시작할 수 있다." / "한 번은 혼자 달걀볶음밥을 만들어봤는데 처음치고 꽤 맛있게 됐다." |
| 3점 | 둘 중 하나는 완벽, 다른 하나는 신호어 없음 | |
| 2점 | 두 문장 모두 신호어 없거나 방법이 서로 뒤바뀜 | |
| 1점 | 한 문장만 작성 | |
| 0점 | 무응답 또는 요리와 무관 | |

## ⑤ AI 피드백 기준

### 자동 채점 항목

| 항목 | 감지 방법 | 피드백 메시지 예시 |
|------|----------|------------------|
| 방법 1가지만 사용 | 같은 신호어만 반복 사용 | "뒷받침 문장에 '왜냐하면'만 3번 사용됐어요. 2가지 이상의 방법을 섞어보세요. 예시, 경험, 비교 방법도 한번 써볼까요?" |
| 신호어 누락 | 뒷받침 문장에 신호어 없음 | "2번째 뒷받침 문장에 신호어가 없어요. 어떤 방법을 쓰려 했는지 알 수 없어요. 신호어를 문장 앞에 넣어보세요." |
| 방법 혼합 성공 | 2가지 이상 다른 신호어 감지 | "이유 방법과 경험 방법을 잘 혼합했어요! 두 가지 다른 시각에서 중심 문장을 뒷받침하고 있어요." |
| 칭찬 | 3가지 이상 혼합 + 자연스러운 연결 | "무려 3가지 방법을 섞어서 단락을 정말 풍부하게 만들었어요. 읽는 사람이 여러 각도에서 이해할 수 있겠어요!" |

### 합격 기준
2가지 이상 방법 혼합 + 각 방법에 신호어 포함 + 중심 문장과 연결 → **차시 05 잠금 해제**

### 미합격 처리
"아직 다듬을 부분이 있어요. 위 피드백을 보고 고친 다음 다시 제출해보세요."  
3회 미합격 시: 은쌤 상담 버튼 활성화
