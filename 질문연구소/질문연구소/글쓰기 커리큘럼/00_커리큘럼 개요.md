# 글쓰기 온라인 교육 커리큘럼 개요

## 설계 원칙

### 대상
- 초등학교 3학년 ~ 중학교 3학년 (만 9~15세)
- 한국어 모국어 학습자

### 핵심 교육 철학
1. **자기주도 학습** — 학생이 스스로 콘텐츠를 보고, 쓰고, 수정한다. 교사 없이도 진행 가능한 구조.
2. **과정 중심 글쓰기** — 결과물보다 쓰는 과정(계획→초안→AI 피드백→수정→완성)을 중시한다.
3. **잠금 해제 방식** — 각 수업은 이전 수업의 조건을 충족해야 열린다. 서두르지 않아도 되고, 건너뛸 수도 없다.
4. **AI 피드백 중심** — 모든 글쓰기 과제에 AI가 즉시 피드백을 제공한다. 학생은 피드백을 보고 스스로 수정한다.
5. **은쌤 상담은 특별한 경우에만** — 아래 조건에 해당할 때만 상담이 열린다.

---

## 은쌤 상담 트리거 조건

다음 중 하나에 해당하면 학생이 "은쌤에게 질문하기" 버튼을 누를 수 있다.

| 조건 | 설명 |
|------|------|
| **반복 미달** | 동일 과제를 3회 이상 제출했으나 합격 기준 미충족 |
| **장기 미진행** | 7일 이상 접속했으나 과제를 완료하지 못한 경우 |
| **학생 요청** | 학생이 직접 "이해가 안 돼요" 버튼 클릭 |
| **AI 판단 불가** | AI가 피드백을 제공하기 어려운 특수한 글 (시, 방언 등) |

## 7단계 구조 한눈에 보기

| 단계 | 이름 | 대상 학년 | 핵심 역량 | 대표 글 유형 | 차시 수 |
|------|------|-----------|-----------|-------------|--------|
| 1단계 | 문장 쓰기 | 초3 | 완전한 문장 구성 | 일기, 관찰일지 | 12 |
| 2단계 | 단락 쓰기 | 초3~4 | 중심문장 + 뒷받침 | 소개글, 설명 단락 | 12 |
| 3단계 | 짧은 글 완성 | 초4~5 | 서론-본론-결론 구조 | 독후감, 편지 | 14 |
| 4단계 | 설명하는 글 | 초5~6 | 정보 전달, 논리 전개 | 설명문, 보고서 | 14 |
| 5단계 | 이야기 글 | 초6~중1 | 서사 구조, 묘사 | 단편소설, 경험 서사 | 16 |
| 6단계 | 주장하는 글 | 중1~2 | 논거 구성, 반박 고려 | 논설문, 주장문 | 16 |
| 7단계 | 통합 글쓰기 | 중2~3 | 목적-독자-형식 통합 | 에세이, 칼럼, 제안서 | 18 |

## 자기주도 수업 흐름 (모든 차시 공통)

한 차시는 아래 6단계로 구성된다. 학생은 순서대로 진행하며, 이전 단계를 완료해야 다음으로 넘어간다.

```
① 개념 읽기 — 핵심 원리 텍스트로 읽기
              개념 설명, 예시 공식, 비교표 등으로 구성

② 예시 분석 — 예시 읽고 분석하기
              모범 예시를 보며 "이 글이 좋은 이유"를 스스로 찾기
              분석 질문 2~3개에 답변 입력 (주관식)

③ TRY      — 연습 문제 풀기
              개념을 짧게 적용하는 연습 (3~5문항)
              AI가 즉시 채점 + 틀리면 해설 제공

④ WRITE    — 내 글 쓰기
              오늘 배운 개념을 적용해 직접 글 작성
              제출 전 자기 체크리스트 완성 필수

⑤ FEEDBACK — AI 피드백 받기
              제출한 글에 대해 AI가 구체적 피드백 제공
              잘한 점 + 개선할 점 + 수정 제안

⑥ REVISE  — 피드백 보고 수정하기
              수정본 제출 → AI가 재채점
              합격 기준 충족 시 다음 차시 잠금 해제
```

## AI 피드백 원칙

| 원칙 | 내용 |
|------|------|
| **즉시성** | 제출 즉시 피드백 제공 (기다림 없음) |
| **구체성** | "더 잘 써봐" 같은 모호한 피드백 금지. "3번째 문장에서 주어가 빠졌어요" 수준으로 구체적 |
| **격려 우선** | 칭찬 → 개선점 순서로 피드백 |
| **수정 가이드** | 개선점을 지적할 때 반드시 수정 방향을 함께 제시 |
| **단계 맞춤** | 각 단계에서 배운 개념에 집중, 아직 배우지 않은 기준으로 평가하지 않음 |

### AI 피드백 초점 (단계별)

| 단계 | AI가 주로 확인하는 것 |
|------|----------------------|
| 1단계 | 주어-서술어 호응, 맞춤법, 문장 부호 |
| 2단계 | 중심 문장 명확성, 뒷받침 연결성, 마무리 문장 |
| 3단계 | 3부 구조, 서론 도입, 결론 완성도 |
| 4단계 | 객관성, 설명 방법 다양성, 전문 용어 풀이 |
| 5단계 | 4막 구조, Show vs Tell, 감각 묘사 |
| 6단계 | 주장 일관성, 근거 타당성, 논리 오류, 반박 고려 |
| 7단계 | 목적·독자·형식 일치, 4단계 퇴고 반영 여부 |
