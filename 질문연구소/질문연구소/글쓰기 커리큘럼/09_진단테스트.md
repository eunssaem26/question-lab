# 진단 테스트 설계

## 목적

신규 학습자가 처음 접속했을 때 적합한 단계를 자동으로 배정하는 도구입니다.
완전히 자기주도로 진행되며 은쌤의 개입 없이 AI가 자동 채점합니다.

---

## 진단 흐름

```
처음 접속
    ↓
온보딩 안내 (5분)
"이 서비스는 어떻게 작동하는가" 안내
    ↓
객관식 진단 (10분, 20문항)
    ↓
글쓰기 실습 진단 (15~20분, 3과제)
    ↓
AI 자동 채점 (즉시)
    ↓
단계 배정 결과 + 설명
    ↓
1차시 잠금 해제
```

---

## 온보딩 안내 내용

진단 시작 전 학습자에게 안내하는 내용:

1. **수업 방식**: 영상 보기 → 예시 분석 → 연습 → 글쓰기 → AI 피드백 → 수정 순서
2. **잠금 해제**: 각 차시를 완료해야 다음 차시가 열린다
3. **AI 피드백**: 글을 제출하면 즉시 피드백을 받는다. 여러 번 수정할 수 있다
4. **은쌤 상담**: 3번 이상 안 되거나, 막히면 언제든 도움을 요청할 수 있다
5. **포트폴리오**: 쓴 글은 모두 저장되고, 나중에 다시 볼 수 있다

## 1단계: 객관식 진단 (20문항)

AI가 자동 채점. 문항당 1점, 총 20점.

### 문항 구성

| 영역 | 문항 수 | 주요 내용 |
|------|---------|----------|
| 문장 완성도 | 5문항 | 완전한 문장 vs 불완전한 문장 구별 |
| 단락 구조 | 5문항 | 중심 문장-뒷받침 문장 관계 이해 |
| 글 구조 파악 | 4문항 | 서론/본론/결론 구분 |
| 설명 방법 | 3문항 | 정의/예시/비교·대조/원인·결과 구별 |
| 논증 이해 | 3문항 | 주장-근거 관계, 타당한 근거 고르기 |

### 예시 문항

**[문장 완성도] 다음 중 완전한 문장은?**
- ① 빠르게 달리는 아이
- ② 학교에서
- ③ **아이가 빠르게 달린다.** ← 정답
- ④ 오늘 아침에

**[단락 구조] 다음 단락에서 중심 문장은?**
- ① 나는 봄을 좋아한다. ← 정답
- ② 봄에는 꽃이 핀다.
- ③ 날씨도 따뜻해진다.
- ④ 기분이 좋아진다.

**[설명 방법] "광합성이란 식물이 햇빛을 이용해 영양분을 만드는 과정이다"에서 사용된 설명 방법은?**
- ① 예시
- ② **정의** ← 정답
- ③ 비교·대조
- ④ 원인·결과

---

## 2단계: 글쓰기 실습 진단 (3과제)

AI가 자동 채점. 각 과제별 12~16점 만점.

---

### 과제 A: 자유 문장 쓰기 (5분)

**지시문 (화면 표시)**
> 오늘 하루 중 가장 기억에 남는 것 하나를 골라 5개의 문장으로 써보세요.
> 잘 쓰려고 너무 애쓰지 않아도 됩니다. 평소에 쓰는 대로 써주세요.

**AI 채점 기준 (총 12점)**

| 기준 | 4점 | 3점 | 2점 | 1점 |
|------|-----|-----|-----|-----|
| 문장 완성도 | 모든 문장 주어-서술어 완비 | 4개 | 3개 | 2개 이하 |
| 구체성 | 육하원칙 요소 3개 이상 | 2개 | 1개 | 없음 |
| 분량 | 5문장 이상 | 4문장 | 3문장 | 2문장 이하 |

---

### 과제 B: 단락 완성하기 (5분)

**지시문 (화면 표시)**
> 아래 첫 문장에 이어지는 문장 3개와 마무리 문장 1개를 써서 단락을 완성해보세요.
>
> **첫 문장**: "나는 봄이 1년 중 가장 좋은 계절이라고 생각한다."

**AI 채점 기준 (총 12점)**

| 기준 | 4점 | 3점 | 2점 | 1점 |
|------|-----|-----|-----|-----|
| 뒷받침 연결성 | 모든 문장이 첫 문장과 연결 | 3개 | 2개 | 1개 |
| 뒷받침 다양성 | 2가지 이상 방법 사용 | 1가지 방법 | 나열 수준 | 없음 |
| 마무리 문장 | 의미 있게 마무리 | 단순 반복 | 있으나 어색 | 없음 |

---

### 과제 C: 짧은 글 쓰기 (10분)

**지시문 (화면 표시)**
> 다음 주제 중 하나를 골라 짧은 글을 써보세요. (250~500자)
> 처음, 중간, 끝이 있는 글을 써보려고 노력해보세요.
>
> - 내가 좋아하는 음식
> - 내 학교의 좋은 점과 아쉬운 점
> - 내가 가장 힘들었던 순간과 그것을 이겨낸 방법

**AI 채점 기준 (총 16점)**

| 기준 | 4점 | 3점 | 2점 | 1점 |
|------|-----|-----|-----|-----|
| 3부 구조 | 서론-본론-결론 명확 | 2부분 구분 | 1부분만 | 구조 없음 |
| 서론 도입 | 흥미로운 도입 방법 사용 | 주제 제시 수준 | 매우 단순 | 없음 |
| 내용 풍부성 | 구체적 사례/이유 2개 이상 | 1개 | 간략한 언급 | 없음 |
| 결론 | 자연스럽게 마무리 | 있으나 약함 | 단순 반복 | 없음 |

## 단계 배정 알고리즘

### 점수 계산

| 항목 | 최대 점수 |
|------|---------|
| 객관식 (20문항 × 1점) | 20점 |
| 과제 A (3개 기준 × 4점) | 12점 |
| 과제 B (3개 기준 × 4점) | 12점 |
| 과제 C (4개 기준 × 4점) | 16점 |
| **합계** | **60점** |

### 배정 기준

| 점수 | 배정 단계 |
|------|----------|
| 0~15점 | **1단계** — 문장 완성이 아직 어색함 |
| 16~25점 | **2단계** — 문장은 쓰지만 단락 구조 연습 필요 |
| 26~35점 | **3단계** — 단락은 쓰지만 전체 구조 연습 필요 |
| 36~42점 | **4단계** — 구조는 있지만 정보 전달 훈련 필요 |
| 43~48점 | **5단계** — 설명은 되지만 서사 표현 훈련 필요 |
| 49~54점 | **6단계** — 서사는 되지만 논증 훈련 필요 |
| 55~60점 | **7단계** — 전반적 역량 갖춤, 통합 훈련 |

## 배정 결과 화면 구성

결과 화면에서 학습자에게 보여주는 내용:

```
┌─────────────────────────────────────────────┐
│  🎉 진단 완료!                              │
│                                             │
│  지금 시작하기 좋은 단계: 3단계             │
│  "짧은 글 완성"                             │
│                                             │
│  이 단계에서 배울 것:                       │
│  ✓ 서론-본론-결론 3부 구조                 │
│  ✓ 독자의 관심을 끄는 서론 도입            │
│  ✓ 독후감과 편지 쓰기                      │
│                                             │
│  [3단계로 시작하기]  [한 단계 낮게 시작]    │
└─────────────────────────────────────────────┘
```

**옵션 제공**:
- **그대로 시작**: 배정 단계에서 시작
- **한 단계 낮게**: 더 자신감 있게 시작하고 싶을 때
- **한 단계 높게**: (54점 이상인 경우에만) 도전하고 싶을 때

## 재진단 기준

| 상황 | 처리 |
|------|------|
| 같은 단계 과제 5개 연속 4점 만점 | "다음 단계 도전 제안" 팝업 |
| 같은 단계 과제 3개 연속 합격 미달 | "이전 단계 복습 제안" 팝업 |
| 학습자 직접 요청 | 설정 메뉴에서 언제든 재진단 가능 |
