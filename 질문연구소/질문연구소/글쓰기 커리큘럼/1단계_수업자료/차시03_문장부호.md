# 차시 03 | 문장 부호 바르게 쓰기

**소요 시간**: 약 40분  
**학습 목표**: 마침표·물음표·느낌표·쉼표의 역할을 이해하고 올바르게 사용할 수 있다

---

## ① 개념 읽기

### 문장 부호란?

문장 부호는 "어떤 목소리로 읽어야 하는지"를 알려주는 신호입니다.  
부호가 없으면 읽는 사람이 어디서 끊어야 할지, 어떤 감정인지 알기 어렵습니다.

---

### 4가지 문장 부호

| 부호 | 이름 | 쓰는 상황 | 읽는 방법 |
|------|------|---------|---------|
| **.** | 마침표 | 일반 문장이 끝날 때 | 목소리를 내리며 끝냄 |
| **?** | 물음표 | 질문하는 문장 | 목소리를 올리며 끝냄 |
| **!** | 느낌표 | 감탄, 강한 감정, 명령, 외침 | 강하고 힘차게 끝냄 |
| **,** | 쉼표 | 단어 나열 / 긴 문장에서 숨 쉬는 곳 | 짧게 쉬고 계속 읽음 |

---

### 헷갈리는 상황 구별하기

**마침표 vs 느낌표**
```
"오늘 날씨가 좋다."   → 사실을 전달할 때 → 마침표
"오늘 날씨가 정말 좋다!" → 감탄하거나 기쁠 때 → 느낌표
```

**쉼표를 쓰는 두 가지 경우**
```
① 단어를 나열할 때
   "나는 사과, 바나나, 포도를 좋아한다."

② 문장이 길어서 잠깐 쉬어야 할 때
   "밥을 다 먹었는데도, 배가 계속 고팠다."
```

## ② 예시 분석

**예시 A** (문장 부호 없음)
```
오늘 체육 시간에 우리 반은 운동장에서 피구를 했다 나는 처음에 공에 맞았다
너무 아팠다 우리 팀이 이겼다 정말 신났다 오늘 또 피구 할 수 있을까
```

**예시 B** (문장 부호 올바르게 사용)
```
오늘 체육 시간에 우리 반은 운동장에서 피구를 했다.
나는 처음에 공에 맞았다. 너무 아팠다!
우리 팀이 이겼다. 정말 신났다!
오늘 또 피구 할 수 있을까?
```

---

**분석 질문 1** (주관식)

> 예시 A를 읽을 때 어떤 불편함이 있나요?

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 2점 | 끊어읽기 어려움, 감정 전달 안 됨, 숨막히는 느낌 등 구체적 불편 언급 |
| 1점 | "읽기 불편하다", "이상하다" 등 현상만 언급 |
| 0점 | 무응답 또는 무관한 답 |

> **모범 답안**: "어디서 문장이 끝나는지 알 수 없어서 어디서 끊어 읽어야 할지 모르겠다. 감탄하는 부분과 질문하는 부분도 구별이 안 된다."

---

**분석 질문 2** (주관식)

> 예시 B에서 느낌표(!)를 사용한 문장을 모두 찾고, 왜 느낌표를 썼는지 설명해보세요.

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 4점 | 해당 문장 2개 모두 찾고, 각각 강한 감정(아픔·기쁨 등)이 이유임을 설명 |
| 3점 | 2개 다 찾았지만 이유 설명이 한 개만 됨 |
| 2점 | 1개만 찾고 이유 설명함 |
| 1점 | 문장만 찾고 이유 없음 |
| 0점 | 무응답 또는 둘 다 틀림 |

> **정답**: "너무 아팠다!" → 강한 아픔을 강조하기 위해 / "정말 신났다!" → 기쁘고 흥분된 감정을 강하게 표현하기 위해

## ③ TRY — 연습 문제

### [객관식] 문제 1~5: 알맞은 문장 부호 고르기

**(마침표 / 물음표 / 느낌표) 중에서 고르세요.**

---

**1번**: "오늘 점심에 나는 김치찌개를 먹었다( )"

> **정답**: 마침표(.) | **해설**: 사실을 전달하는 문장입니다.

---

**2번**: "와, 이 케이크 정말 맛있다( )"

> **정답**: 느낌표(!) | **해설**: 강한 감탄을 표현하는 문장입니다.

---

**3번**: "이 숙제 언제까지 내야 해( )"

> **정답**: 물음표(?) | **해설**: 모르는 것을 묻는 질문입니다.

---

**4번**: "내일 비가 올까( )"

> **정답**: 물음표(?) | **해설**: 미래에 대한 궁금증입니다.

---

**5번**: "야, 빨리 와( )"

> **정답**: 느낌표(!) | **해설**: 강한 명령이나 외침입니다.

---

### [주관식] 문제 6~8: 쉼표 위치 찾기

> 아래 문장에서 쉼표(,)가 필요한 곳을 찾아 쉼표를 넣어 다시 써보세요.

---

**6번**: "나는 국어 수학 영어 과학을 공부했다."

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 2점 | 나열된 단어 모두(국어, 수학, 영어) 사이에 쉼표를 정확히 넣음 |
| 1점 | 일부 위치만 정확히 넣음 (2곳만 등) |
| 0점 | 쉼표를 엉뚱한 곳에 넣거나 무응답 |

> **정답**: "나는 국어, 수학, 영어, 과학을 공부했다."  
> **자주 나오는 오답**: 마지막 단어 앞에도 쉼표 → "국어, 수학, 영어, 과학을" ✅ / "국어 수학 영어, 과학을" ❌

---

**7번**: "밥을 다 먹었는데도 배가 계속 고팠다."

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 2점 | "먹었는데도," 또는 "먹었는데도, 배가" 위치에 정확히 넣음 |
| 1점 | 문장 내에 쉼표를 넣었지만 위치가 어색함 |
| 0점 | 전혀 다른 곳에 넣거나 무응답 |

> **정답**: "밥을 다 먹었는데도, 배가 계속 고팠다."

---

**8번**: "공원에는 강아지 고양이 비둘기가 있었다."

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 2점 | "강아지, 고양이," 위치에 정확히 모두 넣음 |
| 1점 | 하나만 넣음 |
| 0점 | 무응답 또는 엉뚱한 위치 |

> **정답**: "공원에는 강아지, 고양이, 비둘기가 있었다."

---

### [주관식] 문제 9~10: 전체 문장에 부호 넣기

> 아래 글을 읽고 빈칸에 알맞은 문장 부호를 넣어 완성해보세요.

---

**9번**:
```
오늘 생일 파티가 있었다( )
친구들이 케이크( ) 선물( ) 편지를 가져왔다( )
생일 축하 노래를 불러줬을 때( ) 나는 너무 행복했다( )
이런 날이 매일 있으면 얼마나 좋을까( )
```

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 4점 | 7개 부호 모두 정확 (.  ,  ,  .  ,  !  ?) |
| 3점 | 5~6개 정확 |
| 2점 | 3~4개 정확 |
| 1점 | 1~2개 정확 |
| 0점 | 무응답 |

> **정답**:
> ```
> 오늘 생일 파티가 있었다.
> 친구들이 케이크, 선물, 편지를 가져왔다.
> 생일 축하 노래를 불러줬을 때, 나는 너무 행복했다!
> 이런 날이 매일 있으면 얼마나 좋을까?
> ```
> **자주 나오는 오답**: "행복했다." (느낌표 대신 마침표) → 감정 강도를 고려하지 못한 것 → 부분 점수

---

**10번**:
```
엄마가 소리쳤다( ) "빨리 일어나( )"
나는 눈을 떴다( ) 시계를 보니 8시 50분이었다( )
"학교 늦겠다( )" 나는 화들짝 놀랐다( )
```

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 4점 | 6개 부호 모두 정확 (.  !  .  .  !  .) |
| 3점 | 4~5개 정확 |
| 2점 | 2~3개 정확 |
| 1점 | 1개만 정확 |
| 0점 | 무응답 |

> **정답**:
> ```
> 엄마가 소리쳤다. "빨리 일어나!"
> 나는 눈을 떴다. 시계를 보니 8시 50분이었다.
> "학교 늦겠다!" 나는 화들짝 놀랐다.
> ```

## ④ WRITE — 오늘의 글쓰기

**과제**: 오늘 가장 궁금했던 것 **질문 문장 2개**, 오늘 가장 기쁘거나 놀랐던 것 **느낌 문장 2개** 쓰기 (총 4문장)

---

### 그래픽 오거나이저

**[포맷 A: 텍스트 표]**

| 종류 | 내용 | 쓸 부호 |
|------|------|--------|
| 궁금한 것 1 | | ? |
| 궁금한 것 2 | | ? |
| 기쁘거나 놀란 것 1 | | ! |
| 기쁘거나 놀란 것 2 | | ! |

---

**[포맷 B: 인터랙티브 카드]**

```

│  ❓ 질문 문장 1                                    │

│  문장 완성:        [                           ? ] │

┌────────────────────────────────────────────────────┐

│  오늘 궁금했던 것: [                             ] │
│  문장 완성:        [                           ? ] │
└────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────┐

```

---

**📌 제출 전 자기 체크리스트**

- □ 질문 문장 2개 끝에 물음표(?)를 썼다
- □ 느낌 문장 2개 끝에 느낌표(!)를 썼다
- □ 나열하는 부분이 있다면 쉼표(,)를 썼다
- □ 모든 문장에 주어와 서술어가 있다

## ⑤ AI 피드백 기준

### 자동 채점 항목

| 항목 | 감지 방법 | 피드백 메시지 예시 |
|------|----------|------------------|
| 물음표 누락 | 질문 의미의 문장이 마침표나 느낌표로 끝남 | "1번 문장이 질문인데 '?'가 없어요. 끝에 물음표를 붙여보세요." |
| 느낌표 누락 | 감탄·놀람 문장이 마침표로 끝남 | "3번 문장에서 강한 감정이 느껴지는데 '.'로 끝났어요. '!'로 바꿔볼까요?" |
| 쉼표 누락 | 나열 구조에서 쉼표 없는 경우 | "단어 나열에서 쉼표를 빠뜨렸어요." |
| 칭찬 | 부호 사용이 가장 정확한 문장 | "'왜 하늘은 낮에는 파랗고 저녁에는 빨개지는 걸까?'가 물음표를 정확히 썼고, 궁금증이 구체적으로 담겼어요!" |

### 합격 기준
물음표 2개·느낌표 2개 올바른 위치 사용 → **차시 04 잠금 해제**
