**단계**: 1단계 문장 쓰기  
**소요 시간**: 차시당 약 40분  
**학습 목표**: 5감을 모두 활용해 관찰 일지를 풍부하게 쓰고, 시간에 따른 변화를 기록할 수 있다

# 차시 09 | 5감으로 관찰하기

**이 차시 목표**: 시각 외에 청각·후각·촉각을 활용해 대상을 입체적으로 묘사할 수 있다

---

## ① 개념 읽기

### 왜 5감을 모두 써야 할까?

시각만 사용하면 글이 평면적입니다.  
다른 감각을 더하면 독자가 직접 그 자리에 있는 것처럼 느낍니다.

---

**5감 관찰 방법과 표현 예시**

| 감각 | 관찰 방법 | 표현 예시 |
|------|----------|---------|
| **👁 시각** | 색깔, 크기, 모양, 움직임 | "연두색, 손바닥만 한, 가장자리가 들쭉날쭉한 잎" |
| **👂 청각** | 소리의 종류, 크기, 리듬 | "바람이 불 때마다 바스락바스락 소리가 났다" |
| **👃 후각** | 향의 종류, 세기 | "코를 대니 달콤하면서도 흙 냄새가 섞인 향이 났다" |
| **✋ 촉각** | 부드러움, 거칠기, 온도, 무게 | "잎이 부드러울 것 같았지만 실제로 만지니 살짝 까슬까슬했다" |
| **👅 미각** | 맛의 종류, 세기 | 식물이라면 생략 가능 — 함부로 먹으면 안 됨 |

> ⚠️ 모르는 식물·물체는 절대 입에 넣지 마세요. 미각은 안전한 음식에서만!

---

**하나의 대상을 5감으로 표현하는 방법**

```
대상: 화분의 장미

👁 시각:  짙은 빨간색, 꽃잎이 여러 겹, 안쪽은 아직 오므려 있음
👂 청각:  바람이 불 때 꽃잎이 살랑거리는 소리
👃 후각:  가까이 대면 달콤하고 진한 향
✋ 촉각:  줄기의 날카로운 가시, 꽃잎은 비단처럼 얇고 부드러울 것 같음
```

---

## ② 예시 분석

**예시 A** (시각만 사용)

```
관찰 대상: 화분의 장미
날짜: 5월 3일

장미 꽃이 피었다.
꽃잎은 붉은색이고 여러 겹으로 겹쳐 있다.
줄기에는 날카로운 가시가 있다.
잎은 초록색이다.
```

**예시 B** (5감 활용)

```
관찰 대상: 화분의 장미
날짜: 5월 3일

화분 가장 높은 가지에 붉은 장미 한 송이가 완전히 피었다.
꽃잎은 짙은 빨간색에서 바깥쪽으로 갈수록 연해졌고, 안쪽은 아직 꽉 쥔 주먹처럼 오므려 있었다.
꽃 가까이 얼굴을 대자 달콤하고 진한 향이 코를 가득 채웠다.
줄기를 살짝 잡아보니 날카로운 가시가 손가락을 찔렀다. 예상보다 훨씬 날카로웠다.
바람이 불 때마다 꽃잎이 살랑살랑 흔들리며 향이 더 멀리 퍼지는 것 같았다.
```

**분석 질문 1** (주관식)

> 예시 B에서 사용한 감각을 모두 찾고, 해당하는 표현을 함께 써보세요.

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 6점 | 3가지 감각 정확히 찾고 각각 해당 표현까지 정확히 연결 |
| 4~5점 | 2가지 감각 + 표현 정확 |
| 2~3점 | 1가지 감각 + 표현 정확 |
| 1점 | 감각 이름만 씀, 표현 연결 못 함 |
| 0점 | 무응답 |

> **정답**:
> - 시각: "짙은 빨간색에서 바깥쪽으로 갈수록 연해졌고" / "살랑살랑 흔들리며"
> - 후각: "달콤하고 진한 향이 코를 가득 채웠다"
> - 촉각: "날카로운 가시가 손가락을 찔렀다. 예상보다 훨씬 날카로웠다"
>
> *청각의 직접 표현은 없지만 "살랑살랑"에 청각이 암시됨 — 이를 청각으로 쓴 경우도 인정*

**분석 질문 2** (주관식)

> 예시 B에서 가장 인상적인 표현을 하나 골라 그 이유를 설명해보세요.

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 3점 | 표현을 정확히 인용하고, "생생하다", "직접 경험한 것 같다", "예상과 달라서 놀랍다" 등 구체적 이유 |
| 2점 | 표현은 인용했지만 이유가 막연 ("좋다", "잘 썼다") |
| 1점 | 표현 인용했지만 이유 없음 |
| 0점 | 무응답 |

> **채점 메모**: 어떤 표현을 골랐는지는 채점에 영향 없음. 이유의 구체성만 평가.

# 차시 10 | 변화 관찰하기

**이 차시 목표**: 같은 대상을 두 번 관찰해 시간에 따른 변화를 비교 표현으로 기록할 수 있다

---

## ① 개념 읽기

### 관찰 일지의 핵심은 "변화"

관찰 일지는 한 번만 쓰는 것이 아닙니다.  
같은 대상을 여러 번 관찰하면서 무엇이 달라졌는지를 기록하는 것이 진짜 관찰입니다.

---

**변화를 기록하는 표현 방법**

| 표현 방법 | 예시 |
|---------|------|
| **수치 비교** | "어제보다 약 2센티미터 더 자랐다" |
| **색깔 변화** | "어제는 연두색이었는데 오늘은 짙은 초록으로 변했다" |
| **상태 변화** | "어제는 꽉 오므려 있었는데 오늘은 반쯤 피었다" |
| **변화 없음** | "어제와 큰 차이가 없었다. 다만 잎 표면에 작은 물방울이 맺혀 있었다" |

**비교 표현 연습**

```
기본 형태:  "어제는 ~ (이)었는데, 오늘은 ~ (이)다."
            "~ 에 비해 오늘은 ~"
            "어제보다 ~ 더 ~"
```

---

## ② 예시 분석

**변화 관찰 일지 예시**

```
관찰 대상: 화분에 심은 강낭콩

--- 1일차 (4월 20일) ---

흙 위에 아무것도 보이지 않는다.
물을 뿌렸더니 흙이 갈색으로 젖었다.
흙 표면을 손가락으로 눌러보니 촉촉했다.
언제 싹이 날지 기대가 됐다.

--- 2일차 (4월 21일) ---

오늘 드디어 작은 싹이 보였다!
어제는 아무것도 없던 자리에 연두색 고리 모양의 새싹이 흙을 뚫고 나왔다.
새싹 높이는 약 1센티미터였다.
가느다란 줄기가 조금 구부러진 채로 위를 향해 있었다.
어제와 달리 흙 위에 생명이 생긴 것 같아서 신기했다.
```

---

**분석 질문 1** (주관식)

> 이 관찰 일지에서 변화를 설명한 문장을 2개 찾아보세요.

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 4점 | 변화 표현 문장 2개를 정확히 찾고, 어떤 변화인지 설명 |
| 3점 | 2개를 찾았지만 설명 없음 |
| 2점 | 1개만 정확히 찾음 |
| 1점 | 찾은 문장이 변화 표현이 아닌 단순 묘사 |
| 0점 | 무응답 |

> **정답**:
> - "어제는 아무것도 없던 자리에 연두색 고리 모양의 새싹이 흙을 뚫고 나왔다." → 어제 → 오늘 변화 비교
> - "어제와 달리 흙 위에 생명이 생긴 것 같아서 신기했다." → 어제와 비교

---

**분석 질문 2** (주관식)

> 이 관찰 일지에서 1일차와 2일차에 사용된 감각을 각각 찾아보세요.

**채점 루브릭**

| 점수 | 기준 |
|------|------|
