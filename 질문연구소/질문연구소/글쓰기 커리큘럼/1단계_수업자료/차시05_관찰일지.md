# 차시 05 | 관찰 일지 쓰기

**소요 시간**: 약 40분  
**학습 목표**: 관찰 일지의 특징을 이해하고 5감을 활용해 대상을 구체적으로 묘사하는 문장을 쓸 수 있다

---

## ① 개념 읽기

### 관찰 일지란?

어떤 대상을 자세히 살펴보고, 본 것을 글로 기록하는 것입니다.

**좋은 관찰 일지의 3가지 조건**

| 조건 | 나쁜 예 | 좋은 예 |
|------|---------|---------|
| **구체적이다** | "꽃이 예쁘다" | "분홍색 꽃잎 5장이 별 모양으로 퍼져 있다" |
| **5감을 활용한다** | 눈으로만 봄 | 냄새, 소리, 촉감도 표현 |
| **기록이 분명하다** | 무엇을 봤는지 모름 | 대상, 날짜, 본 내용이 또렷하다 |

---

### 5감(다섯 가지 감각)

| 감각 | 방법 | 표현 예시 |
|------|------|---------|
| 👁 **시각** | 색깔, 모양, 크기, 움직임 관찰 | "연두색 잎이 5장 달렸다", "줄기가 구불구불하다" |
| 👂 **청각** | 소리 듣기 | "바람에 잎이 바스락거렸다", "물 주는 소리" |
| 👃 **후각** | 냄새 맡기 | "달콤하고 은은한 향이 났다", "풀 냄새가 났다" |
| ✋ **촉각** | 만져보기 | "잎이 부드럽고 서늘했다", "표면이 살짝 끈적했다" |
| 👅 **미각** | 맛보기 (안전한 것만) | "잎을 씹자 쓴맛이 났다" |

> ⚠️ 모르는 식물은 절대 입에 넣지 마세요. 미각은 안전한 음식에서만!

---

## ② 예시 분석

**예시 A** (시각만 사용)
```
관찰 대상: 화분의 장미  /  날짜: 5월 3일

장미 꽃이 피었다. 꽃잎은 붉은색이다. 줄기에는 가시가 있다. 잎은 초록색이다.
```

**예시 B** (5감 활용)
```
관찰 대상: 화분의 장미  /  날짜: 5월 3일

화분 가장 높은 가지에 붉은 장미 한 송이가 완전히 피었다.
꽃잎은 짙은 빨간색에서 바깥쪽으로 갈수록 연해졌고, 안쪽은 아직 꽉 쥔 주먹처럼 오므려 있었다.
꽃 가까이 얼굴을 대자 달콤하고 진한 향이 코를 가득 채웠다.
줄기를 살짝 잡아보니 날카로운 가시가 손가락을 찌를 것 같아 얼른 손을 뗐다.
바람이 불 때마다 꽃잎이 살랑살랑 흔들렸다.
```

---

**분석 질문 1** (주관식)

> 예시 B에서 사용한 감각을 모두 찾고, 해당하는 문장을 쓰세요.

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 6점 | 3가지 감각 모두 찾고 각각 정확한 문장 연결 |
| 4~5점 | 2가지 감각 정확히 찾음 |
| 2~3점 | 1가지 감각 정확히 찾음 |
| 1점 | 감각 이름만 씀, 문장 연결 못 함 |
| 0점 | 무응답 |

> **정답**:
> - 시각: "꽃잎은 짙은 빨간색에서 바깥쪽으로 갈수록 연해졌고~" / "꽃잎이 살랑살랑 흔들렸다"
> - 후각: "달콤하고 진한 향이 코를 가득 채웠다"
> - 촉각: "날카로운 가시가 손가락을 찌를 것 같아"

---

**분석 질문 2** (주관식)

> 예시 A와 B 중 더 생생하게 느껴지는 것은 어느 쪽인가요? 그 이유를 구체적으로 설명하세요.

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 4점 | B를 선택하고, "여러 감각 사용", "구체적 묘사", "장면이 상상됨" 등 구체적 이유 2가지 이상 |
| 3점 | B를 선택하고 구체적 이유 1가지 |
| 2점 | B를 선택했지만 이유가 "더 잘 썼다" 수준 |
| 1점 | B를 선택했지만 이유 없음 |
| 0점 | A를 선택하거나 무응답 |

> **모범 답안**: "예시 B가 더 생생하다. 시각뿐만 아니라 후각(달콤한 향)과 촉각(가시가 찌를 것 같음)도 표현해서 직접 그 자리에 있는 것 같은 느낌이 든다."

## ③ TRY — 연습 문제

### [객관식] 문제 1~5: 어떤 감각인가?

**아래 표현에서 주로 사용한 감각을 고르세요. (시각 / 청각 / 후각 / 촉각 / 미각)**

**1번**: "껍질을 벗기자 즙이 손에 묻었다"
> **정답**: **촉각**

**2번**: "멀리서 피아노 소리가 들렸다"
> **정답**: **청각**

**3번**: "차가운 바람이 뺨을 스쳤다"
> **정답**: **촉각**

**4번**: "달걀 후라이에서 고소한 냄새가 났다"
> **정답**: **후각**

**5번**: "사과가 빨갛고 표면이 반질반질했다"
> **정답**: **시각** (+ 촉각이 암시됨 — 반질반질)

---

### [주관식] 문제 6~8: 시각 표현에 다른 감각 추가하기

> 아래 시각 표현에 어울리는 다른 감각 표현을 1개 추가하세요.

---

**6번**: "노란 꽃이 피었다." → 후각 표현 추가

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 3점 | 자연스러운 후각 표현 추가 + 문장 형태 완성 | 
| 2점 | 후각 내용은 맞지만 문장이 다소 어색함 |
| 1점 | 후각이 아닌 다른 감각을 추가했지만 내용은 자연스러움 |
| 0점 | 시각 표현만 반복하거나 무응답 |

> **예시 정답**: "노란 꽃이 피었다. 코를 가까이 대니 달콤하고 옅은 향이 났다."
> **자주 나오는 오답**: "노란 꽃이 예쁘게 피었다" → 시각만 추가한 것 → 1점

---

**7번**: "강아지 털이 하얗다." → 촉각 표현 추가

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 3점 | 자연스러운 촉각 표현 추가 + 문장 완성 |
| 2점 | 촉각 내용이지만 어색함 |
| 1점 | 다른 감각이지만 자연스러움 |
| 0점 | 시각만 추가 또는 무응답 |

> **예시 정답**: "강아지 털이 하얗다. 손으로 쓸어보니 솜처럼 부드럽고 따뜻했다."

---

**8번**: "꽃잎이 바람에 흔들린다." → 청각 표현 추가

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 3점 | 자연스러운 청각 표현 추가 + 문장 완성 |
| 2점 | 청각 내용이지만 어색함 |
| 1점 | 다른 감각이지만 자연스러움 |
| 0점 | 시각만 추가 또는 무응답 |

> **예시 정답**: "꽃잎이 바람에 흔들린다. 잎들이 서로 스치며 바스락바스락 소리가 났다."

---

### [주관식] 문제 9: 5감 오거나이저 채우기

> 아래 주제 중 하나를 골라 5감 오거나이저를 채우세요.

주제: ① 교실 창문 밖 풍경  ② 책상 위 물건  ③ 방금 먹은 음식

**채점 루브릭**

| 항목 | 2점 | 1점 | 0점 |
|------|-----|-----|-----|
| 시각 | 색깔·모양·크기·움직임 중 2개 이상 | 1개 | 없음 |
| 청각 | 상황에 맞는 구체적 소리 | 막연한 "소리가 났다" | 없음 |
| 후각 | 상황에 맞는 냄새 표현 | 막연한 "냄새" | 없음 |
| 촉각 | 질감·온도·무게 등 | 막연한 "느낌" | 없음 |

> **모범 답안 예시**:
> - 시각: 책상 위 연필이 노랗고 길쭉하다
> - 청각: 창문 밖에서 자동차 소리가 작게 들린다
> - 후각: 방금 먹은 빵에서 달콤한 냄새가 난다
> - 촉각: 지우개가 말랑하고 조금 미끄럽다

## ⑤ AI 피드백 기준

### 자동 채점 항목

| 항목 | 감지 방법 | 피드백 메시지 예시 |
|------|----------|------------------|
| 분량 | 5문장 미만 감지 | "4문장으로 조금 짧아요. 감각 표현 한 문장만 더 추가해볼까요?" |
| 막연한 형용사 | "예쁘다", "좋다", "있다" 등만 반복 | "'예쁜 꽃'이라고 썼는데 어떻게 예쁜지 더 구체적으로 써볼까요? 색깔이나 모양은 어떤가요?" |
| 단일 감각 | 시각 표현만 5문장 이상 | "시각 표현을 잘 썼어요! 여기에 냄새나 소리 표현을 한 문장 더 추가하면 더 생생해져요." |
| 칭찬 | 가장 생생한 감각 표현 | "'잎을 살짝 만지니 솜털처럼 부드럽고 조금 서늘했다'는 표현이 정말 생생해요! 만져보지 않아도 느낌이 상상돼요." |

### 합격 기준
5문장 이상, 구체적 묘사 2개 이상, 시각 외 감각 1개 이상 → **차시 06 잠금 해제**
