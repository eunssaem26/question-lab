# 차시 04 | 맞춤법과 띄어쓰기 기초

**소요 시간**: 약 40분  
**학습 목표**: 자주 틀리는 맞춤법과 띄어쓰기를 이해하고 올바르게 쓸 수 있다

---

## ① 개념 읽기

### 자주 틀리는 맞춤법 10가지

> 하나씩 읽고, 왜 틀리는지 이해해보세요.

---

**1. 돼 vs 되**

| 상황 | 규칙 | 예시 |
|------|------|------|
| "되어"로 바꿀 수 있으면 | **돼** | "안 돼" = "안 되어" ✅ |
| "되어"로 바꿀 수 없으면 | **되** | "되다", "될" |

```
❌ 안되  →  ✅ 안 돼
❌ 됬다  →  ✅ 됐다  (되었다 → 됐다)
```

---

**2. 안 vs 않**

| 단어 | 뜻 | 뒤에 오는 것 |
|------|-----|-----------|
| **안** | "아니"의 줄임 | 동사/형용사 바로 앞 |
| **않** | "아니하"의 줄임 | "-지 않다" 형태 |

```
✅ 안 갔다      (아니 갔다)
✅ 가지 않았다  (가지 아니하였다)
❌ 않 갔다      (이런 표현은 없음)
```

---

**3. 며칠 (몇일 ❌)**

```
❌ 몇일  →  ✅ 며칠
"생일이 며칠이야?"  "며칠째 비가 온다."
```

---

**4. 왠지 vs 웬**

| 단어 | 뜻 | 예시 |
|------|-----|------|
| **왠지** | "왜인지"의 줄임 | "왠지 기분이 좋다" |
| **웬** | "어떤", "어찌 된" | "웬 일이야?", "웬 사람" |

---

**5. 오랜만에 (오랫만에 ❌)**

```
❌ 오랫만에  →  ✅ 오랜만에
```

---

**6. 됐다 (됬다 ❌)**

```
되었다 → 됐다  (줄여쓰면 됐다, 됬다가 아님)
❌ 됬어  →  ✅ 됐어
```

---

### 띄어쓰기 기본 규칙 3가지

**규칙 1**: 단어와 단어 사이는 띄어 씁니다.
```
❌ 나는학교에갔다.   ✅ 나는 학교에 갔다.
```

**규칙 2**: "안", "못"은 다음 단어와 **띄어** 씁니다.
```
❌ 안했다    ✅ 안 했다
❌ 못갔다    ✅ 못 갔다
```

**규칙 3**: 조사(은/는/이/가/을/를/에서 등)는 앞 단어에 **붙여** 씁니다.
```
❌ 나 는 학교 에 갔다.   ✅ 나는 학교에 갔다.
```

## ② 예시 분석

**예시 글** (오류 포함)

```
오늘은 왠지 학교에 가기 싫었다.
몇일 전부터 감기 기운이 있었는데, 오늘은 특히 몸이 안좋았다.
엄마한테 학교에 안가도 됬냐고 물어봤더니,
엄마는 어이없다는 표정으로 "당연히 가야지!" 라고 했다.
오랫만에 학교를 쉬고 싶었는데, 결국 가게 됬다.
```

---

**분석 질문 1** (주관식)

> 위 글에서 틀린 맞춤법이나 띄어쓰기를 **3개 이상** 찾아, 틀린 표현 → 바른 표현 형식으로 써보세요.

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 6점 | 6개 이상 정확히 찾고 바르게 고침 |
| 5점 | 5개 정확 |
| 4점 | 4개 정확 |
| 3점 | 3개 정확 |
| 2점 | 2개 정확 |
| 1점 | 1개 정확 |
| 0점 | 무응답 또는 전혀 못 찾음 |

> **전체 오류 목록**:
> - 몇일 → **며칠**
> - 안좋았다 → **안 좋았다** (띄어쓰기)
> - 안가도 → **안 가도** (띄어쓰기)
> - 됬냐고 → **됐냐고**
> - 오랫만에 → **오랜만에**
> - 됬다 → **됐다**
>
> *왠지는 맞는 표현이므로 오류가 아님. 이를 오류로 잡은 경우 채점에 반영 안 함.*

## ③ TRY — 연습 문제

### [객관식] 문제 1~5: 올바른 표현 고르기

**1번**: 시험이 내일인데 공부를 ( **안 했다** / 않했다 / 안했다 ).

> **정답**: **안 했다** | **해설**: "안"은 다음 단어와 띄어 씁니다.

---

**2번**: 생일이 ( **며칠** / 몇일 ) 남았어?

> **정답**: **며칠** | **해설**: "몇일"은 표준어가 아닙니다.

---

**3번**: 지금 나가면 ( 안돼 / **안 돼** ).

> **정답**: **안 돼** | **해설**: "안"은 띄어 쓰고, "되어"→"돼"가 맞습니다.

---

**4번**: 오늘 ( **왠지** / 웬지 ) 기분이 좋다.

> **정답**: **왠지** | **해설**: "왠지" = "왜인지"의 줄임말.

---

**5번**: 어제 숙제가 다 ( **됐어** / 됬어 )?

> **정답**: **됐어** | **해설**: "되었어" → 줄이면 "됐어". "됬"은 없는 글자입니다.

---

### [주관식] 문제 6~8: 띄어쓰기 고치기

> 아래 문장에서 띄어쓰기가 잘못된 부분을 모두 고쳐서 다시 써보세요.

---

**6번**: "나는학교에서친구를만났다."

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 2점 | "나는 학교에서 친구를 만났다." 정확히 씀 |
| 1점 | 1~2곳만 띄어쓰기 수정 |
| 0점 | 그대로 쓰거나 더 틀리게 씀 |

> **정답**: "나는 학교에서 친구를 만났다."

---

**7번**: "숙제를못했어서선생님께혼났다."

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 2점 | "숙제를 못 했어서 선생님께 혼났다." 정확히 씀 |
| 1점 | "못 했어서"는 고쳤으나 다른 곳 미수정 또는 반대 |
| 0점 | 그대로 또는 더 틀리게 |

> **정답**: "숙제를 못 했어서 선생님께 혼났다."

---

**8번**: "오늘은비가와서운동장에서놀수없었다."

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 2점 | "오늘은 비가 와서 운동장에서 놀 수 없었다." 정확히 씀 |
| 1점 | 대부분 수정했으나 1~2곳 누락 |
| 0점 | 그대로 또는 더 틀리게 |

> **정답**: "오늘은 비가 와서 운동장에서 놀 수 없었다."

---

### [주관식] 문제 9~10: 전체 문장 고치기

> 아래 문장에서 틀린 부분을 모두 고쳐 다시 써보세요.

---

**9번**: "며칠전부터 몸이안좋아서 학교를 못갔다."

**채점 루브릭**

| 점수 | 기준 | 오류 목록 |
|------|------|---------|
| 4점 | 3곳 모두 정확히 수정 | 며칠전→며칠 전, 안좋아서→안 좋아서, 못갔다→못 갔다 |
| 3점 | 2곳 수정 | |
| 2점 | 1곳 수정 | |
| 1점 | 수정 시도했으나 모두 틀림 | |
| 0점 | 무응답 또는 그대로 | |

> **정답**: "며칠 전부터 몸이 안 좋아서 학교를 못 갔다."

---

**10번**: "오랫만에 학교에 갔더니 왠지 어색했다."

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 2점 | "오랜만에" 수정 + "왠지" 유지 (왠지는 맞으므로) |
| 1점 | "오랫만에"만 수정 |
| 0점 | "왠지"까지 틀리게 수정하거나 무응답 |

> **정답**: "오랜만에 학교에 갔더니 왠지 어색했다."  
> **함정 포인트**: "왠지"는 맞는 표현. 이를 "웬지"로 바꾸면 오히려 감점.

## ④ WRITE — 오늘의 글쓰기

**과제**: 어제 한 일을 **4문장**으로 쓰세요. 맞춤법과 띄어쓰기에 특히 집중해보세요.

---

### 그래픽 오거나이저

**[포맷 A: 텍스트 표]**

| 순서 | 어제 있었던 일 | 주의할 맞춤법 |
|------|--------------|-------------|
| 1 | | |
| 2 | | |
| 3 | | |
| 4 | | |

---

**[포맷 B: 인터랙티브 카드]**

```
┌────────────────────────────────────────────────────┐
│  문장 1                                            │
│  있었던 일:  [                                   ] │
│  주의할 점:  [  안/못 띄어쓰기?  됐다/됬다?      ] │
│  완성 문장:  [                                   ] │
└────────────────────────────────────────────────────┘
```
*(2, 3, 4도 동일)*

---

**📌 제출 전 자기 체크리스트**

- □ "안", "못" 뒤에 띄어쓰기를 했다
- □ "됐다"를 올바르게 썼다 ("됬다" ❌)
- □ "며칠"을 올바르게 썼다 ("몇일" ❌)
- □ 단어와 단어 사이를 모두 띄어 썼다
- □ 모든 문장 끝에 마침표를 썼다

## ⑤ AI 피드백 기준

### 자동 채점 항목

| 항목 | 감지 방법 | 피드백 메시지 예시 |
|------|----------|------------------|
| 됬→됐 | "됬" 포함 감지 | "'됬어요'는 없는 말이에요. '됐어요'로 고쳐주세요." |
| 몇일→며칠 | "몇일" 포함 감지 | "'몇일'은 없는 말이에요. '며칠'이 바른 표현이에요." |
| 안/못 띄어쓰기 | "안했다", "못갔다" 등 붙여쓰기 감지 | "'안했다'는 '안 했다'로 띄어 써야 해요." |
| 오랫만에→오랜만에 | "오랫만에" 감지 | "'오랫만에'는 '오랜만에'가 바른 표현이에요." |
| 칭찬 | 4문장 중 오류 없는 문장 | "2번째 문장은 맞춤법과 띄어쓰기가 모두 완벽해요!" |

### 합격 기준
4문장 중 맞춤법·띄어쓰기 오류 2개 이하 → **차시 05 잠금 해제**
