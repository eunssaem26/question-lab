# 차시 02 | 문장을 풍부하게 만들기

**소요 시간**: 약 40분  
**학습 목표**: 육하원칙을 활용해 기본 문장에 정보를 추가하고 구체적인 문장을 쓸 수 있다

---

## ① 개념 읽기

### 기본 문장의 한계

아래 두 문장을 비교해보세요.

> "강아지가 달렸다."  
> "오늘 아침에 공원에서 우리 강아지가 낯선 고양이를 보고 신나게 달렸다."

두 번째 문장을 읽으면 장면이 머릿속에 그려지죠?  
이렇게 문장을 풍부하게 만드는 도구가 **육하원칙**입니다.

---

### 육하원칙 6가지

| 질문 | 추가되는 정보 | 예시 표현 |
|------|------------|---------|
| **누가** | 주어를 더 구체적으로 | "우리 강아지가", "반 친구 민호가" |
| **언제** | 시간 정보 | "오늘 아침에", "점심시간에", "어제 밤 10시에" |
| **어디서** | 장소 정보 | "공원에서", "교실 복도에서", "집 앞에서" |
| **무엇을** | 대상 정보 | "고양이를 보고", "숙제를", "밥을" |
| **어떻게** | 방식·태도 정보 | "신나게", "조용히", "천천히", "소리를 지르며" |
| **왜** | 이유 정보 | "배가 고파서", "심심해서", "화가 나서" |

---

### 문장 확장 방법

```
① 기본 문장을 먼저 씁니다.
   "강아지가 달렸다."

② 육하원칙 질문을 하나씩 던집니다.
   언제? → 오늘 아침에
   어디서? → 공원에서
   어떻게? → 신나게

③ 자연스러운 위치에 넣어 확장합니다.
   "오늘 아침에 공원에서 강아지가 신나게 달렸다."
```

> **주의**: 정보를 너무 많이 넣으면 오히려 어색해집니다.  
> 가장 중요하거나 흥미로운 정보 2~3개만 골라 넣는 게 좋아요.

---

## ② 예시 분석

---

**예시 A** (기본 문장만)
```
나는 밥을 먹었다.
친구를 만났다.
집에 갔다.
```

**예시 B** (육하원칙으로 확장)
```
어제 저녁에 나는 온 가족과 함께 부엌에서 따뜻한 된장찌개를 맛있게 먹었다.
학교 끝나고 교문 앞에서 단짝 친구 지수를 우연히 만났다.
비가 오기 시작해서 우산을 쓰고 빠른 걸음으로 집으로 돌아갔다.
```

---

**분석 질문 1** (주관식)

> 예시 B의 첫 번째 문장에서 추가된 육하원칙 요소를 찾아 쓰세요.  
> (찾은 것만큼만 써도 됩니다)

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 4점 | 4가지 이상 정확히 찾음 |
| 3점 | 3가지 정확히 찾음 |
| 2점 | 2가지 정확히 찾음 |
| 1점 | 1가지만 찾거나, 있는 것을 틀리게 표현 |
| 0점 | 무응답 또는 전혀 무관 |

> **정답**: 언제(어제 저녁에), 어디서(부엌에서), 누구와(온 가족과 함께), 무엇을(된장찌개를), 어떻게(맛있게)

---

**분석 질문 2** (주관식)

> 예시 A와 B 중 어느 쪽이 더 읽기 좋은가요? 이유를 한 문장으로 설명해보세요.

**채점 루브릭**

| 점수 | 기준 |
|------|------|
| 2점 | B가 낫다고 고르고, 구체적 정보·장면 상상 등의 이유를 제시 |
| 1점 | B가 낫다고 골랐지만 이유가 막연함 ("더 잘 쓴 것 같다") |
| 0점 | 이유 없이 선택만 하거나, A를 선택 |

> **모범 답안**: "예시 B가 더 읽기 좋다. 언제, 어디서, 어떻게 등의 정보가 있어서 읽는 사람이 장면을 직접 상상할 수 있기 때문이다."

---

## ③ TRY — 연습 문제

### [객관식] 문제 1~3: 추가된 정보 찾기

**기본 문장과 확장 문장을 비교하고, 추가된 육하원칙 요소를 모두 고르세요.**

---

**1번**

| 기본 | 고양이가 잤다. |
|------|--------------|
| 확장 | 우리 집 고양이가 오후 내내 소파 위에서 동그랗게 웅크리고 잤다. |

추가된 것을 모두 고르세요: ① 언제  ② 어디서  ③ 어떻게  ④ 왜

> **정답**: ①②③ | **해설**: 오후 내내(언제), 소파 위에서(어디서), 동그랗게 웅크리고(어떻게)

---

**2번**

| 기본 | 비가 왔다. |
|------|----------|
| 확장 | 어젯밤에 갑자기 창문을 두드릴 만큼 세차게 비가 왔다. |

> **정답**: ① 언제(어젯밤에), ③ 어떻게(갑자기, 세차게)

---

**3번**

| 기본 | 나는 공부했다. |
|------|--------------|
| 확장 | 시험이 내일이라서 나는 밤 11시까지 책상 앞에서 열심히 공부했다. |

> **정답**: ①②③④ 모두 | **해설**: 왜(시험이 내일이라서), 언제(밤 11시까지), 어디서(책상 앞에서), 어떻게(열심히)

---

### [주관식] 문제 4~6: 직접 확장하기

> 기본 문장에 육하원칙 요소를 2개 이상 추가해 더 풍부한 문장으로 만들어보세요.

---

**4번 기본 문장**: "친구가 웃었다."

**채점 루브릭**

| 점수 | 기준 | 예시 |
|------|------|------|
| 4점 | 원래 주어+서술어 유지, 육하원칙 2개 이상 자연스럽게 추가 | "점심시간에 우리 반 친구 민호가 선생님의 재미있는 이야기를 듣고 배를 잡으며 크게 웃었다." |
| 3점 | 육하원칙 1개만 추가, 자연스러움 | "수업 시간에 친구가 웃었다." |
| 2점 | 추가했지만 원래 문장이 바뀌거나 어색한 조합 | "친구가 웃었다 재미있어서 교실에서" (어순 이상) |
| 1점 | 단어만 나열, 문장 형태 아님 | "친구가 재미있게 웃었다 어디서" |
| 0점 | 원래 문장 그대로 반복 또는 무응답 | |

> **자주 나오는 오답**: 정보를 너무 많이 넣어 문장이 어색해지는 경우 → 2점  
> **피드백**: "정보를 많이 넣으려고 노력했어요! 그런데 조금 길고 복잡해졌어요. 가장 중요한 정보 2~3개만 골라볼까요?"

---

**5번 기본 문장**: "꽃이 피었다."

**채점 루브릭**

| 점수 | 기준 | 예시 |
|------|------|------|
| 4점 | 육하원칙 2개 이상, 자연스러운 확장 | "4월 초 따뜻한 봄날에 학교 화단에 분홍색 벚꽃이 활짝 피었다." |
| 3점 | 육하원칙 1개, 자연스러움 | "봄에 꽃이 피었다." |
| 2점 | 추가했으나 어색함 | "꽃이 예쁘게 피었다 어디서 학교에서" |
| 1점 | 문장 형태 불완전 | |
| 0점 | 원래 문장 그대로 | |

---

**6번 기본 문장**: "나는 넘어졌다."

**채점 루브릭**

| 점수 | 기준 | 예시 |
|------|------|------|
| 4점 | 육하원칙 2개 이상, 자연스러움 | "어제 체육 시간에 운동장에서 달리기를 하다가 돌부리에 걸려 나는 앞으로 넘어졌다." |
| 3점 | 육하원칙 1개, 자연스러움 | "어제 나는 넘어졌다." |
| 2점 | 추가했으나 어색함 | |
| 1점 | 문장 형태 불완전 | |
| 0점 | 원래 문장 그대로 | |

## ④ WRITE — 오늘의 글쓰기

**과제**: 오늘 먹은 음식에 대해 육하원칙이 담긴 문장 **3개**를 써보세요.

---

### 그래픽 오거나이저

**[포맷 A: 텍스트 표]**

| | 무슨 음식? | 언제? | 어디서? | 누구와? | 어떻게? |
|--|----------|-------|---------|---------|--------|
| 음식 1 | | | | | |
| 음식 2 | | | | | |
| 음식 3 | | | | | |

---

**[포맷 B: 인터랙티브 카드]**  
*(웹에서는 각 칸 클릭 입력)*

```
┌──────────────────────────────────────────────────────┐
│  음식 1                                              │
│                                                      │
│  무슨 음식?   [                ]                     │
│  언제?        [                ]    어디서? [       ] │
│  누구와?      [                ]    어떻게? [       ] │
│                                                      │
│  완성 문장 →                                         │
│  [                                                 ] │
└──────────────────────────────────────────────────────┘
```
*(음식 2, 3도 동일한 카드 형태)*

---

**✏️ 완성 문장 쓰는 곳**

```
문장 1:


문장 2:


문장 3:

```

---

**📌 제출 전 자기 체크리스트**

- □ 각 문장에 주어와 서술어가 있다
- □ 각 문장에 육하원칙 요소가 2개 이상 들어있다
- □ 문장이 자연스럽게 읽힌다 (소리 내서 읽어보세요)
- □ 모든 문장 끝에 마침표(.)를 찍었다

## ⑤ AI 피드백 기준

### 자동 채점 항목

| 항목 | 감지 방법 | 피드백 메시지 예시 |
|------|----------|------------------|
| 육하원칙 수 | 시간·장소·방식 부사어 및 이유절 감지 | "1번 문장에 '언제'나 '어디서'에 해당하는 표현이 없어요. '점심에', '식탁에서' 같은 표현을 추가해볼까요?" |
| 어색한 나열 | 부사어가 3개 이상이고 호응이 부자연스러운 경우 | "문장에 정보가 많이 들어갔는데 조금 복잡해졌어요. 가장 중요한 정보 2~3개만 남겨볼까요?" |
| 주어·서술어 유지 | 확장 중 주어나 서술어 탈락 감지 | — |
| 칭찬 | 가장 구체적이고 자연스러운 문장 | "'어제 저녁에 할머니 댁 부엌에서 갓 지은 쌀밥과 된장찌개를 온 가족이 함께 먹었다'는 문장이 정말 생생해요! 장면이 눈에 그려지네요." |

### 합격 기준
3문장 모두 육하원칙 요소 2개 이상, 자연스러운 문장 → **차시 03 잠금 해제**
