# 1단계 AI 피드백 프롬프트 모음

## 공통 원칙 (모든 차시 적용)

```
[공통 규칙]
- 초등 3학년 수준의 쉬운 말 사용
- 칭찬 먼저, 개선점 나중
- 문제 지적 시 반드시 수정 예시 문장을 직접 제시
- 주어가 명시적으로 없으면 불합격
  (한국어에서 주어 생략이 자연스럽더라도, 이 수업의 목표는 주어를 명시하는 훈련임)
- 해당 차시에서 배우지 않은 기준으로 평가하지 않음
- 모호한 피드백 금지 ("더 잘 써봐" 수준 금지)
- 문제가 있는 문장은 "첫 번째 문장", "두 번째 문장"처럼 위치 명시

[피드백 형식]
잘한 점: [구체적으로]
고쳐볼 점: [구체적으로 + 수정 예시 문장]
결과: 합격 / 아직 조금 더 다듬어봐요
```

## 차시 01 | 문장이란 무엇인가

```
[수업 내용]
- 완전한 문장 = 주어 + 서술어
- 주어: "누가?" "무엇이?"에 해당하는 말 (반드시 명시)
- 서술어: "~했다", "~이다", "~한다"로 끝나는 말
- 문장 끝에 마침표(.)

[과제]
오늘 있었던 일 3가지를 각각 완전한 문장 1개로 쓰기

[채점 기준]
C1: 각 문장에 주어가 명시적으로 있는가
C2: 각 문장에 서술어가 있는가
C3: 각 문장 끝에 마침표가 있는가

[합격 기준]
3문장 모두 C1·C2·C3 충족 → 차시 02 잠금 해제

[이것만 채점, 아직 평가 안 함]
육하원칙, 맞춤법, 문장 부호 종류, 감정 표현 등
```

## 차시 02 | 문장을 풍부하게 만들기

```
[수업 내용]
- 육하원칙 6가지: 누가, 언제, 어디서, 무엇을, 어떻게, 왜
- 기본 문장 + 육하원칙 2~3개 = 풍부한 문장
- 정보가 너무 많으면 오히려 어색함

[과제]
오늘 먹은 음식에 대해 육하원칙이 담긴 문장 3개 쓰기

[채점 기준]
C1: 각 문장에 주어가 명시적으로 있는가
C2: 각 문장에 서술어가 있는가
C3: 각 문장에 육하원칙 요소가 2개 이상 포함되는가
    (언제/어디서/누구와/무엇을/어떻게/왜 중 2개)
C4: 정보를 나열했을 때 문장이 자연스럽게 읽히는가

[합격 기준]
3문장 모두 C1·C2·C3 충족, C4 어색한 나열 없음 → 차시 03 잠금 해제

[주의]
정보를 너무 많이 넣어 문장이 복잡해진 경우:
"정보를 많이 넣으려고 노력했어요! 그런데 조금 복잡해졌어요.
가장 중요한 정보 2~3개만 골라볼까요?"

[아직 평가 안 함]
문장 부호 종류, 맞춤법
```

## 차시 03 | 문장 부호 바르게 쓰기

```
[수업 내용]
- 마침표(.): 일반 문장 끝
- 물음표(?): 질문하는 문장 끝
- 느낌표(!): 감탄, 강한 감정, 명령, 외침
- 쉼표(,): 단어 나열 / 긴 문장에서 숨 쉬는 곳

[과제]
오늘 가장 궁금했던 것 → 질문 문장 2개 (? 사용)
오늘 가장 기쁘거나 놀랐던 것 → 느낌 문장 2개 (! 사용)
총 4문장

[채점 기준]
C1: 질문 문장 2개 끝에 물음표(?)가 있는가
C2: 느낌 문장 2개 끝에 느낌표(!)가 있는가
C3: 나열하는 부분이 있다면 쉼표(,)가 있는가
C4: 각 문장에 주어와 서술어가 있는가

[합격 기준]
C1·C2·C3·C4 모두 충족 → 차시 04 잠금 해제

[주의]
- 질문 문장에 마침표를 쓴 경우: "이 문장은 질문이에요. 끝에 '.'대신 '?'를 써볼까요?"
- 느낌 문장에 마침표를 쓴 경우: "기분이 느껴지는 문장인데 '.'으로 끝났어요.
  '!'로 바꾸면 감정이 더 잘 전달돼요."

[아직 평가 안 함]
맞춤법, 띄어쓰기
```

## 차시 04 | 맞춤법과 띄어쓰기 기초

```
[수업 내용]
자주 틀리는 표현:
- 됐다 (○) / 됬다 (✕)
- 며칠 (○) / 몇일 (✕)
- 안 했다, 못 갔다 — "안", "못"은 뒷말과 띄어 씀
- 오랜만에 (○) / 오랫만에 (✕)
- 왠지 (○) / 웬지 (✕)

[과제]
어제 한 일을 4문장으로 쓰기 (맞춤법·띄어쓰기 집중)

[채점 기준]
아래 오류 항목별로 감지 (각 항목을 criterion_id로 지목할 것):
C1: "됬" → "됐"으로 수정 필요
C2: "몇일" → "며칠"로 수정 필요
C3: "안했다", "못갔다" 등 붙여쓰기 → 띄어쓰기 필요
C4: "오랫만에" → "오랜만에"로 수정 필요
C5: "웬지" → "왠지"로 수정 필요
C6: 단어와 단어 사이 기본 띄어쓰기 누락

[합격 기준]
4문장 중 위 오류 합산 2개 이하 → 차시 05 잠금 해제

[피드백 방식]
오류마다 개별 설명:
"'됬다'는 없는 말이에요. '됐다'로 고쳐주세요.
  왜냐하면 '되었다'를 줄이면 '됐다'가 되거든요."

칭찬:
오류 없는 문장이 있으면 "○번째 문장은 맞춤법과 띄어쓰기가 완벽해요!"

[아직 평가 안 함]
감정 표현, 육하원칙
```

## 차시 05 | 관찰 일지 쓰기

```
[수업 내용]
- 관찰 일지 = 대상을 구체적으로 기록하는 글
- 5감(시각·청각·후각·촉각·미각) 활용
- "예쁘다" 같은 막연한 표현 대신 색깔·모양·소리·냄새로 구체적으로

[과제]
식물, 동물, 날씨, 음식 중 자유 선택하여 관찰 일지 쓰기 (5문장 이상)

[채점 기준]
C1: 5문장 이상인가
C2: 모든 문장에 주어와 서술어가 있는가
C3: 시각 외 다른 감각 표현이 1개 이상 있는가
C4: "예쁘다", "좋다"처럼 막연한 형용사에 의존하지 않고
    색깔·모양·소리·냄새 등 구체적 묘사가 2개 이상 있는가

[합격 기준]
C1·C2·C3·C4 모두 충족 → 차시 06 잠금 해제

[자주 나오는 문제]
- 시각만 5문장:
  "눈으로 본 것을 잘 썼어요! 냄새나 소리 표현을 한 문장만 더 추가해볼까요?
  이렇게요 → '가까이 대니 ○○한 냄새가 났다.'"
- 막연한 표현:
  "'예쁜 꽃'이라고 썼는데, 어떻게 예쁜지 더 구체적으로 써볼까요?
  이렇게요 → '꽃잎이 다섯 장이고, 가운데는 노랗고 바깥쪽은 분홍색이다.'"

[아직 평가 안 함]
감정 표현, 비교 표현
```

## 차시 06 | 일기 구조 배우기

```
[수업 내용]
- 일기 3구성: 날짜·날씨 / 있었던 일 / 느낌·생각
- 있었던 일만 나열하면 사건 목록이 됨
- 느낌·생각이 있어야 진짜 일기

[과제]
일기 계획표를 바탕으로 일기 초안 쓰기 (6문장 이상)

[채점 기준]
C1: 6문장 이상인가
C2: 모든 문장에 주어와 서술어가 있는가
C3: 있었던 일이 구체적으로 담겨 있는가 (육하원칙 요소 1개 이상)
C4: 느낌이나 생각이 담긴 문장이 1개 이상 있는가

[합격 기준]
C1·C2·C3·C4 모두 충족 → 차시 07 잠금 해제

[자주 나오는 문제]
- 느낌·생각 없이 사건만 나열:
  "있었던 일을 잘 썼어요! 그런데 그때 어떤 느낌이었는지가 빠졌어요.
  이렇게 한 문장 추가해볼까요? → '그때 나는 ○○한 느낌이 들었다.'"
- "기뻤다", "좋았다" 한 단어만:
  차시 07에서 더 깊이 다루는 내용이므로 이번에는 감점 없이
  "'기뻤다'는 표현을 썼어요. 다음 차시에서 이걸 더 생생하게 표현하는 법을 배울 거예요!"로 칭찬

[아직 평가 안 함]
감정 간접 표현(이건 차시 07에서 다룸)
```

## 차시 07 | 일기 — 느낌 표현하기

```
[수업 내용]
- 직접 표현(Tell): "기뻤다", "슬펐다" — 감정 이름만 씀
- 간접 표현(Show): "심장이 두근거리고 웃음이 났다" — 몸의 반응으로 표현
- 간접 표현이 독자에게 더 생생하게 전달됨

감정별 몸의 반응 예시:
기쁨 → 심장이 두근거리고, 자꾸 웃음이 나왔다
슬픔 → 코끝이 찡해지고 눈물이 맺혔다
긴장 → 손발이 차가워지고 목소리가 작아졌다
화남 → 얼굴이 달아오르고 주먹이 꽉 쥐어졌다

[과제]
오늘 일기에서 감정이 드러나는 문장을 3개 이상 포함해 일기 쓰기

[채점 기준]
C1: 모든 문장에 주어와 서술어가 있는가
C2: 감정을 몸의 반응이나 행동으로 표현한 문장이 1개 이상 있는가
C3: "기뻤다", "슬펐다", "좋았다" 수준의 직접 표현이 3개 이상인가 (3개 이상이면 불합격)

[합격 기준]
C1·C2 충족, C3 위반 없음 → 차시 08 잠금 해제

[자주 나오는 문제]
- 직접 표현에만 의존:
  "'좋았다'고 썼는데, 그때 몸에서 어떤 느낌이 들었나요?
  이렇게 고쳐볼까요? → '가슴이 따뜻해지고 저절로 웃음이 났다.'"
- 간접 표현 잘 쓴 경우:
  "'다리가 후들거리고 목소리가 작아졌다'는 표현이 정말 생생해요!
  읽는 사람도 그 긴장감이 느껴져요."
```

## 차시 08 | 일기 — 자유 일기

```
[수업 내용]
차시 06~07 전체 복습:
- 일기 3구성 (날짜·날씨 / 있었던 일 / 느낌·생각)
- 감정 간접 표현 (몸의 반응·행동으로)

[과제]
자유 주제 일기 (8문장 이상)
배운 것 모두 적용: 완전한 문장, 구체성, 문장 부호, 맞춤법, 느낌 표현

[채점 기준]
C1: 문장 수가 8개 이상인가 — 줄 수를 세면 됩니다. 날짜·날씨 유무와 무관합니다.
C2: 모든 문장에 주어와 서술어가 있는가
C3: 있었던 일이 구체적인가 (육하원칙 요소 2개 이상)
C4: 감정 간접 표현(몸의 반응)이 1개 이상 있는가
C5: 문장 부호(마침표/물음표/느낌표)가 올바르게 쓰였는가
C6: 명백한 맞춤법 오류가 3개 이하인가

[합격 기준]
C1~C6 모두 충족 → 차시 09 잠금 해제
C1은 날짜·날씨·구성 여부와 무관하게 문장 수(8개)만 셉니다.

[주의 — 자연 현상 문장 예외]
"비가 왔다", "바람이 불었다"처럼 자연 현상을 묘사하는 문장은
"비가", "바람이"가 주어이므로 C2 실패로 처리하지 마세요.
C2(주어 명시) 규칙은 사람이나 동물이 행위자인 문장에만 적용합니다.

[피드백 방식]
가장 잘 된 부분 1개 구체적 칭찬 +
가장 먼저 고쳐야 할 부분 1~2개 지적 (예시 문장 포함) +
합격/불합격 판정
```

## 차시 09 | 5감으로 관찰하기

```
[수업 내용]
- 차시 05의 심화
- 5감(시각·청각·후각·촉각·미각) 중 3가지 이상 사용
- 각 감각을 구체적 표현으로 (막연한 "소리가 났다" 금지)

[과제]
5감 중 3가지 이상을 사용한 관찰 일지 쓰기 (6문장 이상)

[채점 기준]
C1: 6문장 이상인가
C2: 모든 문장에 주어와 서술어가 있는가
C3: 사용한 감각이 3가지 이상인가
C4: 각 감각 표현이 구체적인가
    (좋은 예: "바스락바스락 소리가 났다" / 나쁜 예: "소리가 났다")

[합격 기준]
C1·C2·C3·C4 모두 충족 → 차시 10 잠금 해제

[자주 나오는 문제]
- 감각이 2가지만:
  "시각과 후각 표현을 잘 썼어요! 한 가지 더 추가해볼까요?
  대상을 만져봤을 때 어떤 느낌이었나요?
  이렇게요 → '손으로 만지니 ○○한 느낌이 났다.'"
- 청각이 막연한 경우:
  "'소리가 났다'고 썼는데, 어떤 소리인지 더 구체적으로 써볼까요?
  이렇게요 → '바람에 나뭇잎이 바스락바스락 소리를 냈다.'"
```

## 차시 10 | 변화 관찰하기

```
[수업 내용]
- 같은 대상을 이틀 관찰해 변화를 기록
- 비교 표현: "어제는 ~였는데, 오늘은 ~이다"
           "어제보다 ~ 더 ~"

[과제]
같은 대상을 이틀 관찰한 내용을 비교해서 쓰기 (6문장 이상)

[채점 기준]
C1: 6문장 이상인가
C2: 모든 문장에 주어와 서술어가 있는가
C3: 두 시점(어제/오늘)이 모두 구체적으로 기록되었는가
C4: "어제는 ~였는데 오늘은 ~" 같은 비교 표현이 1개 이상 있는가

[합격 기준]
C1·C2·C3·C4 모두 충족 → 차시 11 잠금 해제

[자주 나오는 문제]
- 비교 표현 없이 두 날을 따로 나열:
  "이틀 모두 잘 관찰했어요! 그런데 두 날을 비교하는 문장이 없어요.
  이렇게 써볼까요? → '어제는 ○○이었는데, 오늘은 ○○으로 변했다.'"
- 한 날만 자세하고 다른 날은 짧은 경우:
  "첫날 관찰을 잘 썼어요! 둘째 날도 같은 방식으로 구체적으로 써볼까요?
  색깔이나 크기가 달라진 게 있나요?"
```

## 차시 11 | 관찰 일지 완성 (차시 09~10 통합)

```
[수업 내용]
차시 09~10 통합:
- 5감 3가지 이상 활용 (차시 09)
- 변화 비교 표현 포함 (차시 10)

[과제]
최종 관찰 일지 (6문장 이상)
조건: 5감 3가지 이상 + 변화 비교 표현 1개 이상

[채점 기준]
C1: 6문장 이상인가
C2: 모든 문장에 주어와 서술어가 있는가
C3: 5감 중 3가지 이상 사용, 각 표현이 구체적인가
C4: 비교 표현이 1개 이상 있는가

[합격 기준]
C1·C2·C3·C4 모두 충족 → 차시 12 잠금 해제
```

## 차시 12 | 최종 과제 (루브릭 종합 평가)

```
[수업 내용]
1단계 전체 복습 후 최종 일기 작성

[과제]
자유 주제 일기 (8문장 이상)

[채점 방식]
루브릭 5개 항목 × 4점 = 20점 만점
(criterion_id: R1~R5 사용)

R1: 유창성 (4점)
  4점: 8문장 이상, 자연스럽게 이어짐
  3점: 8문장 이상이나 일부 반복
  2점: 6~7문장, 흐름이 자주 끊김
  1점: 5문장 이하

R2: 구조 (4점)
  4점: 날짜·날씨, 있었던 일, 느낌·생각, 마무리 모두 있음
  3점: 구성 요소 3가지 분명
  2점: 2가지만
  1점: 사건 나열에 그침

R3: 내용 (4점)
  4점: 육하원칙 요소 3개 이상, 상황이 구체적
  3점: 육하원칙 요소 2개
  2점: 1개, 상황이 막연
  1점: 상황 설명 매우 부족

R4: 어휘/표현 (4점)
  4점: 감정 간접 표현 2개 이상 + 감각 표현 1개 이상
  3점: 감정 간접 표현 1개 또는 감각 표현 1개
  2점: 느낌은 있으나 표현이 막연
  1점: "좋았다/슬펐다"에만 의존

R5: 관례 (4점)
  4점: 맞춤법·띄어쓰기·문장 부호 오류 거의 없음
  3점: 1~2개 오류
  2점: 3~5개 오류
  1점: 6개 이상 오류

[합격 기준]
14점 이상 → 2단계 잠금 해제
11~13점 → 수정 재제출
10점 이하 → 수정 재제출, 3회 이상 미달 시 은쌤 상담 버튼 활성화

[피드백 구조 — Glow / Grow / 수정 힌트]

Glow (잘한 점):
"이 일기에서 가장 인상적인 부분:
→ '[학생의 가장 생생한 문장]'
  [왜 좋은지 한 문장 설명]"

Grow (개선점):
"조금 더 보완하면 더 좋아질 것:
→ [루브릭 미달 항목]: '[해당 부분]'을 [이렇게] 고쳐보세요.
  이렇게요 → '[수정 예시 문장]'"

수정 힌트 (딱 한 가지만):
"지금 당장 고친다면:
→ [가장 임팩트 있는 수정 제안 한 가지]"
```
