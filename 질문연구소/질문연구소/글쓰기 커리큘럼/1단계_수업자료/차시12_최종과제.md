# 차시 12 | 최종 과제 & 포트폴리오

**단계**: 1단계 문장 쓰기  
**소요 시간**: 약 45분  
**학습 목표**: 1단계에서 배운 모든 것을 적용해 완성도 있는 일기를 혼자서 쓴다

---

## 최종 과제 안내

> 드디어 1단계 마지막 과제예요!  
> 지금까지 배운 것을 모두 담아 일기를 써보세요.  
> 틀려도 괜찮아요. AI 피드백을 받고 수정하면 됩니다.

## ① 개념 읽기

### 1단계 전체 총복습

최종 과제를 쓰기 전에, 지금까지 배운 것을 한눈에 정리해봅니다.

| 배운 것 | 핵심 포인트 | 체크 |
|---------|-----------|------|
| **완전한 문장** | 주어 + 서술어 필수 | "아이가 달렸다." ✅ / "달렸다." ❌ |
| **문장 확장** | 육하원칙으로 구체적으로 | "오늘 아침 공원에서 강아지가 신나게 달렸다." |
| **문장 부호** | . ? ! , 역할 구별 | 질문엔 ?, 감탄엔 !, 나열엔 , |
| **맞춤법** | 자주 틀리는 표현 | 며칠 ✅ / 몇일 ❌, 됐다 ✅ / 됬다 ❌ |
| **관찰 일지** | 5감 활용, 구체적 묘사 | 시각 외 다른 감각 1개 이상 |
| **일기** | 있었던 일 + 느낌·생각 | 감정은 몸의 반응으로 표현 |

---

**최종 과제에서 특히 신경 쓸 것**

좋은 일기 = 구체적 상황 + 감정 간접 표현 + 다감각 묘사

```
❌ 나쁜 예:
오늘 발표를 했다. 긴장됐다. 잘 했다. 기뻤다.

✅ 좋은 예:
오늘 국어 시간에 내가 직접 쓴 시를 발표했다.
이름이 불리자 다리가 후들거리고 목소리가 떨렸다.
그래도 끝까지 읽었더니 선생님이 박수를 쳐주셨다.
나도 모르게 입가에 웃음이 번졌다.
```

## ② READ — 예시 분석

**예시 A** (기준이 부족한 글)

```
4월 15일 화요일 맑음

오늘 발표를 했다.
긴장됐다.
잘 했다.
기뻤다.
집에 왔다.
```

**예시 B** (기준을 잘 반영한 글)

```
4월 15일 화요일 맑음

오늘 국어 시간에 내가 직접 쓴 시를 발표했다.
이름이 불리자 다리가 후들거리고 목소리가 떨렸다.
교실이 조용해져서 내 숨소리까지 크게 들리는 것 같았다.
그래도 끝까지 읽었더니 선생님이 박수를 쳐주셨다.
그 순간 얼굴이 뜨거워지고 입가에 웃음이 자꾸 번졌다.
집에 오는 길에는 봄바람이 가볍게 느껴졌다.
다음에도 용기 내서 내 글을 읽어 보고 싶다.
```

**분석 질문**

> 예시 B가 예시 A보다 더 좋은 이유를 3가지 써보세요.

## ③ TRY — 제출 전 미니 점검

아래 질문에 답하며 내 글에 들어갈 요소를 확인해보세요.

1. 내가 쓸 사건은 하나로 모여 있나요?
2. 언제, 어디서, 누구와 있었는지 넣을 수 있나요?
3. 느낌이나 생각 문장을 2개 이상 넣을 수 있나요?
4. 몸의 반응으로 감정을 보여줄 수 있나요?
5. 시각 외 감각 표현을 1개 이상 넣을 수 있나요?

## ④ WRITE — 최종 과제

**과제**: 자유 주제 일기 (8문장 이상)

**주제 힌트** (마음에 드는 것 하나 선택, 완전 자유 선택도 가능)

- 최근에 가장 자랑스러웠던 순간
- 최근에 가장 긴장됐던 순간
- 요즘 가장 궁금한 것
- 내가 가장 좋아하는 장소에서 있었던 일
- 완전 자유 주제 (직접 선택!)

**📌 제출 전 자기 평가 (모두 체크해야 제출 가능)**

```
[ 문장 ]
□ 모든 문장에 주어와 서술어가 있다
□ 육하원칙 요소가 문장에 포함되어 있다 (최소 2개 이상)

[ 문장 부호 · 맞춤법 ]
□ 마침표(.), 물음표(?), 느낌표(!) 를 올바르게 사용했다
□ 나열하는 부분에 쉼표(,)를 썼다
□ "안 했다", "못 했다" (띄어쓰기)를 확인했다
□ "며칠" / "됐다" 등 맞춤법을 확인했다

[ 표현 ]
□ 느낌이나 생각이 담긴 문장이 2개 이상 있다
□ 감정을 몸의 반응이나 행동으로 표현한 부분이 있다
□ 시각 외 다른 감각 표현이 1개 이상 있다

[ 분량 ]
□ 8문장 이상 썼다

[ 나의 평가 ]
□ 이전에 쓴 글보다 나아진 점이 있다
  → 어떤 점이 나아졌나요? (직접 입력)
```

## ⑤ AI 피드백 기준 (루브릭 최종 평가)

### 최종 평가 항목 (각 4점, 총 20점)

| 항목 | 4점 | 3점 | 2점 | 1점 |
|------|-----|-----|-----|-----|
| **유창성** | 8문장 이상을 자연스럽게 이어 썼다 | 8문장 이상이나 일부 반복이 있다 | 6~7문장이고 흐름이 자주 끊긴다 | 5문장 이하이거나 흐름이 매우 약하다 |
| **구조** | 날짜·날씨, 있었던 일, 느낌·생각, 마무리가 모두 드러난다 | 요소 3가지는 분명하다 | 요소 2가지만 드러난다 | 사건 나열에 그친다 |
| **내용** | 육하원칙 요소 3개 이상, 상황이 구체적이다 | 육하원칙 요소 2개, 상황이 대체로 보인다 | 육하원칙 요소 1개, 상황이 막연하다 | 상황 설명이 매우 부족하다 |
| **어휘/표현** | 감정 간접 표현 2개 이상 + 감각 표현 1개 이상 | 감정 간접 표현 1개 또는 감각 표현 1개가 있다 | 느낌은 있으나 표현이 막연하다 | 직접 감정 표현에만 의존한다 |
| **관례** | 맞춤법·띄어쓰기·문장 부호 오류가 거의 없다 | 1~2개 오류 | 3~5개 오류 | 6개 이상 오류 |

### AI 피드백 3-파트 구조

**1. 칭찬 (Glow)**
```
이 일기에서 가장 인상적인 부분:
→ "[학생의 가장 생생한 문장]" — 이 표현이 특히 좋아요.
   [왜 좋은지 한 문장 설명]
```

**2. 개선점 (Grow)**
```
조금 더 보완하면 더 좋아질 것:
→ [루브릭 미달 항목명]: "[해당 부분]"을 [이렇게] 바꿔보세요.
```

**3. 수정 힌트**
```
딱 한 가지만 고친다면:
→ [가장 임팩트 있는 수정 제안 한 가지]
```

## ⑥ REVISE — 수정 후 재제출

- AI 피드백에서 가장 먼저 고칠 1가지를 고른다
- 수정 전 문장과 수정 후 문장을 비교해본다
- 다시 제출해 14점 이상이면 2단계 잠금 해제

---

### 진급 처리

| 점수 | 결과 |
|------|------|
| **14점 이상** | ✅ **2단계 잠금 해제** + 수료 메시지 표시 |
| 11~13점 | AI 피드백 확인 후 수정 재제출 요청 |
| 10점 이하 | AI 피드백 확인 후 수정, 3회 이상 미달 시 은쌤 상담 버튼 활성화 |
