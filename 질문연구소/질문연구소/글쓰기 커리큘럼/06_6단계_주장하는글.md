# 6단계: 주장하는 글

## 기본 정보

| 항목 | 내용 |
|------|------|
| **대상** | 중학교 1~2학년 (또는 5단계 이수자) |
| **총 차시** | 16차시 |
| **차시당 학습 시간** | 약 50분 |
| **핵심 질문** | "내 주장을 읽는 사람이 납득하게 만들려면 어떤 근거가 필요할까?" |
| **진급 조건** | 최종 과제 루브릭 합산 16점 이상 (20점 만점) |

---

## 단계 목표

이 단계를 마치면 학습자는:
- 논증의 구조(주장-근거-반박 고려)를 이해하고 적용할 수 있다
- 타당한 근거와 타당하지 않은 근거를 구별할 수 있다
- 반박 의견을 예상하고 이를 미리 재반론할 수 있다
- 감정적 호소가 아닌 논리적·사실적 근거로 주장을 뒷받침할 수 있다
- 1000~1200자 분량의 논설문을 완성할 수 있다

---

## 핵심 개념

### 논증의 기본 구조 (ARE 모델)
```
A — Assertion (주장):  내가 옳다고 믿는 것
R — Reason (이유):     왜 그렇게 생각하는가
E — Evidence (증거):   그것을 뒷받침하는 사실/통계/사례
```

### 5단 논증 구조

| 단계 | 역할 |
|------|------|
| **1. 도입** | 이슈 제시 + 내 입장 선언 |
| **2. 근거 1** | 가장 강한 근거 + 증거 |
| **3. 근거 2** | 두 번째 근거 + 증거 |
| **4. 반박 고려** | 반대 의견 인정 + 재반박 |
| **5. 결론** | 주장 재확인 + 행동 촉구 |

### 좋은 근거 vs 나쁜 근거

| 좋은 근거 | 나쁜 근거 |
|-----------|-----------|
| 통계, 연구 결과 | "내 친구도 그렇게 생각한다" |
| 전문가 의견 | "다들 그렇게 한다" |
| 실제 사례·사건 | "당연히 맞는 것 아닌가?" |
| 역사적 사실 | 감정에만 호소하는 주장 |

### 피해야 할 논리적 오류 4가지

| 오류 | 예 |
|------|----|
| **일반화 오류** | "내 주변 친구들이 그러니 모든 학생이 그렇다" |
| **거짓 선택지** | "이 법을 지지하지 않으면 환경을 파괴하는 것이다" |
| **감정 호소** | 논리 없이 "얼마나 슬픈 일인지 생각해보세요" |
| **권위에 호소** | "유명인이 말했으니 맞다" |

## 차시별 학습 내용

### 차시 1: 주장과 의견의 차이

**① 개념 읽기**
- 핵심 내용: 의견과 주장은 무엇이 다른가 — 주장은 근거를 요구한다

**② READ — 예시 분석**
- 의견만 있는 글 vs 주장+근거가 있는 글 비교
- 분석 질문: "근거가 있는 글이 더 설득력 있는 이유는 무엇인가요?"

**③ TRY — 연습 문제**
- 10개 문장이 "의견"인지 "주장(근거 필요)"인지 분류하기

**④ WRITE**
- 과제: 내가 강하게 믿는 것 3가지를 주장 형태로 쓰기
- 자기 체크리스트:
  - □ 반박 가능한 형태인가? (모든 사람이 동의하면 주장이 아님)
  - □ 구체적인 하나의 주장인가?

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 2 잠금 해제

---

### 차시 2: 근거 찾기와 평가하기

**① 개념 읽기**
- 핵심 내용: 좋은 근거의 3가지 조건 — 타당성(주장과 연결), 신뢰성(믿을 수 있는 출처), 관련성(맥락에 맞음)

**② READ — 예시 분석**
- 동일한 주장에 대해 강한 근거 3개 + 약한 근거 3개 제시
- 분석 질문: "강한 근거와 약한 근거의 차이는 무엇인가요?"

**③ TRY — 연습 문제**
- 5개의 근거를 받아 강도 순으로 배열하기 (2세트)

**④ WRITE**
- 과제: 차시 1에서 쓴 주장 중 하나를 골라 근거 3개 찾기 + 각 근거의 강도 평가 메모
- 자기 체크리스트:
  - □ 각 근거가 주장과 직접 연결된다
  - □ 통계 또는 사례가 1개 이상 포함된다

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 3 잠금 해제

---

### 차시 3~4: ARE 모델로 단락 쓰기 (2차시)

#### 차시 3: ARE 단락 구조

**① 개념 읽기**
- 핵심 내용: 주장-이유-증거 단락 쓰는 법

**② READ — 예시 분석**
- ARE 구조로 쓴 근거 단락 2개 분석
- 분석 질문: "주장/이유/증거를 각각 찾아 표시해보세요."

**③ TRY — 연습 문제**
- 주장만 있는 단락에 이유 + 증거 추가하기 (2세트)

**④ WRITE**
- 과제: ARE 구조로 근거 단락 1개 작성

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 4 잠금 해제

---

#### 차시 4: ARE 단락 심화

**④ WRITE**
- 과제: ARE 구조로 근거 단락 2개 작성 (다른 방법의 증거 사용)
  - 단락 1: 통계 또는 연구 결과 사용
  - 단락 2: 실제 사례 또는 경험 사용

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 5 잠금 해제

---

### 차시 5: 반박 고려하기

**① 개념 읽기**
- 핵심 내용: 왜 반대 의견을 먼저 인정하는 것이 더 설득력 있는가 — "비록 ~라는 주장도 있다. 그러나 ~때문에 ~이다" 구조

**② READ — 예시 분석**
- 반박 고려 없는 논설문 vs 반박 고려 있는 논설문 비교
- 분석 질문: "반박을 먼저 인정하면 독자에게 어떤 인상을 주나요?"

**③ TRY — 연습 문제**
- 주어진 주장에 대해 반대 측 가장 강한 근거 찾기 + 재반박 쓰기 (2세트)

**④ WRITE**
- 과제: 반박 고려 단락 작성
  - "비록 ~ 측 주장처럼 ~ 이지만, ~ 때문에 ~이다"
- 자기 체크리스트:
  - □ 반대 의견을 공정하게 제시했다 (일부러 약하게 만들지 않았다)
  - □ 재반박이 구체적인 이유를 포함한다

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 6 잠금 해제

### 차시 6: 논리적 오류 피하기

**① 개념 읽기**
- 핵심 내용: 논설문을 약하게 만드는 4가지 오류

**② READ — 예시 분석**
- 논리적 오류가 있는 논설문 분석
- 분석 질문: "이 글에서 발견한 오류의 이름과 이유를 설명해보세요."

**③ TRY — 연습 문제**
- 오류가 있는 문장 5개에서 오류 유형 찾기

**④ WRITE**
- 과제: 지금까지 쓴 글에서 논리적 오류가 있는지 스스로 점검하고 수정하기
  - 수정 전/후 나란히 제출

**⑤ FEEDBACK** — AI 확인: 오류 감지

**⑥ REVISE** → 합격 시 차시 7 잠금 해제

## 그래픽 오거나이저

### 5단 논증 계획표
```
┌─────────────────────────────────────────────┐
│  주장 (한 문장):                            │
│  대상 독자:         내 입장:               │
├─────────────────────────────────────────────┤
│  1. 도입                                    │
│  이슈: 왜 중요한 문제인가?                  │
│  내 입장 선언:                              │
├─────────────────────────────────────────────┤
│  2. 근거 1 (가장 강한 것)                   │
│  주장:                                      │
│  이유:                                      │
│  증거 (통계/사례/전문가):                   │
├─────────────────────────────────────────────┤
│  3. 근거 2                                  │
│  주장:                                      │
│  이유:                                      │
│  증거:                                      │
├─────────────────────────────────────────────┤
│  4. 반박 고려                               │
│  반대 측 주장:                              │
│  인정할 점:                                 │
│  그러나 내 주장이 더 타당한 이유:           │
├─────────────────────────────────────────────┤
│  5. 결론                                    │
│  핵심 주장 재확인:                          │
│  독자에게 요청하는 것:                      │
└─────────────────────────────────────────────┘
```

## 평가 루브릭 (최종 과제)

| 평가 기준 | 4점 | 3점 | 2점 | 1점 |
|-----------|-----|-----|-----|-----|
| **주장 명확성** | 처음부터 끝까지 일관된 명확한 주장 | 주장 있으나 다소 모호 | 주장이 흔들림 | 주장 불명확 |
| **근거의 질** | 2개, 각각 통계·사례·전문가 포함 | 2개, 1개만 증거 있음 | 근거 1개 | 근거 없음 |
| **반박 고려** | 공정한 반대 의견 + 효과적 재반박 | 반박 있으나 재반박 약함 | 반박만 나열 | 없음 |
| **논리 오류** | 오류 없음 | 1개 | 2개 | 3개 이상 |
| **분량/구조** | 1100자 이상, 5단 구조 완성 | 1000자 이상 | 900자 이상 | 900자 미만 |

**진급 기준**: 합산 16점 이상
