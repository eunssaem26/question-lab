# 3단계: 짧은 글 완성

## 기본 정보

| 항목 | 내용 |
|------|------|
| **대상** | 초등 4~5학년 (또는 2단계 이수자) |
| **총 차시** | 14차시 |
| **차시당 학습 시간** | 약 45분 |
| **핵심 질문** | "내 글에 처음, 중간, 끝이 있는가? 읽는 사람이 자연스럽게 따라올 수 있는가?" |
| **진급 조건** | 최종 과제 루브릭 합산 15점 이상 (20점 만점) |

---

## 단계 목표

이 단계를 마치면 학습자는:
- 서론-본론-결론의 3부 구조를 이해하고 글에 적용할 수 있다
- 서론에서 독자의 관심을 끄는 도입 문장을 쓸 수 있다
- 본론에서 2개 이상의 단락으로 내용을 전개할 수 있다
- 결론에서 글 전체를 자연스럽게 마무리할 수 있다
- 독후감과 편지 형식의 글을 완성할 수 있다

---

## 핵심 개념

### 3부 구조

| 부분 | 역할 | 분량 | 핵심 질문 |
|------|------|------|----------|
| **서론** | 독자의 관심을 끌고 주제를 제시 | 전체의 20% | "왜 이 글을 써야 하는가?" |
| **본론** | 주제를 구체적으로 전개 | 전체의 60% | "무슨 내용을 어떤 순서로 담을까?" |
| **결론** | 핵심 내용을 정리하고 여운을 남김 | 전체의 20% | "독자에게 마지막으로 무엇을 전달할까?" |

### 서론 도입 방법 4가지

| 방법 | 예시 |
|------|------|
| **질문으로 시작** | "여러분은 책을 읽고 나서 어떤 느낌이 드나요?" |
| **장면 묘사** | "창밖에 첫눈이 내리던 그날, 나는 ..." |
| **놀라운 사실** | "우리나라 초등학생이 하루 평균 책을 읽는 시간은 단 14분이라고 한다." |
| **직접 주제 제시** | "내가 가장 좋아하는 책은 '왕자와 거지'이다." |

## 차시별 학습 내용

### 차시 1: 3부 구조 이해하기

**① 개념 읽기**
- 핵심 내용: 서론, 본론, 결론이 필요한 이유

**② READ — 예시 분석**
- 3부 구조가 없는 글 vs 있는 글 비교 (300자 내외)
- 분석 질문:
  - "구조가 없는 글을 읽었을 때 어떤 느낌이 드나요?"
  - "잘 쓴 글에서 서론/본론/결론은 각각 어디에서 어디까지인가요?"

**③ TRY — 연습 문제**
- 글을 읽고 서론·본론·결론을 색깔로 구분하기 (3개 글)

**④ WRITE**
- 과제: "내가 좋아하는 음식"에 대해 서론 2가지 버전 써보기
  - 버전 1: 질문으로 시작
  - 버전 2: 직접 주제 제시로 시작

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 2 잠금 해제

---

### 차시 2: 서론 — 독자의 관심 끌기

**① 개념 읽기**
- 핵심 내용: 4가지 도입 방법 심화 — 좋은 서론 vs 나쁜 서론 비교

**② READ — 예시 분석**
- 동일한 주제로 4가지 도입 방법을 사용한 서론 4개
- 분석 질문:
  - "가장 읽고 싶어지는 서론은 어느 것인가요? 왜인가요?"

**③ TRY — 연습 문제**
- 서론을 읽고 어떤 도입 방법을 사용했는지 맞추기 (4문항)

**④ WRITE**
- 과제: "내가 좋아하는 계절" 서론 쓰기
  - 4가지 방법 중 하나를 선택 + 선택 이유 한 줄 메모
- 자기 체크리스트:
  - □ 독자가 계속 읽고 싶어지는 도입인가?
  - □ 이 글의 주제가 서론에서 암시된다

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 3 잠금 해제

---

### 차시 3~4: 본론 — 내용 전개 (2차시)

#### 차시 3: 단락 순서 정하기

**① 개념 읽기**
- 핵심 내용: 본론 단락의 순서를 어떻게 정하는가 — 시간 순서, 중요도 순서, 원인→결과 순서

**② READ — 예시 분석**
- 단락 순서가 바뀐 본론 vs 논리적 본론
- 분석 질문: "순서를 바꾸면 어떤 문제가 생기나요?"

**③ TRY — 연습 문제**
- 흩어진 단락 3개를 논리적 순서로 배열하기 (2세트)

**④ WRITE**
- 과제: "내가 좋아하는 계절" 본론 계획 작성
  - 본론 단락 2개의 중심 문장 + 순서 근거 메모

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 4 잠금 해제

---

#### 차시 4: 본론 2단락 쓰기

**④ WRITE**
- 과제: 차시 3의 계획을 바탕으로 본론 2단락 작성
  - 각 단락: 중심 문장 + 뒷받침 3개 + 마무리 문장
- 자기 체크리스트:
  - □ 두 단락이 각각 다른 내용을 다룬다
  - □ 단락 간 접속어로 연결된다
  - □ 각 단락에 중심 문장이 있다

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 5 잠금 해제

---

### 차시 5: 결론 — 여운 남기기

**① 개념 읽기**
- 핵심 내용: 결론은 단순히 요약이 아니다 — 나쁜 결론 패턴("이상으로 마치겠습니다") vs 좋은 결론

**② READ — 예시 분석**
- 나쁜 결론 3가지 예시 분석
- 좋은 결론 3가지 예시 분석
- 분석 질문: "좋은 결론의 공통점은 무엇인가요?"

**③ TRY — 연습 문제**
- 서론+본론이 주어진 글에 결론 쓰기 (2세트)

**④ WRITE**
- 과제: 지금까지 쓴 서론+본론에 결론 추가하기
- 자기 체크리스트:
  - □ 결론이 본론의 내용을 자연스럽게 마무리한다
  - □ 새로운 내용을 추가하지 않는다
  - □ 단순히 반복하거나 "이상으로 ..."로 끝내지 않는다

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 6 잠금 해제

---

### 차시 6: 서론-본론-결론 한 번에 완성

**③ TRY — 계획표 작성**
- 3부 구조 계획표 완성 (주제: 자유 선택)

**④ WRITE**
- 과제: 처음부터 끝까지 3부 구조 글 완성 (400자 이상)
- 자기 체크리스트 (전체):
  - □ 서론에 도입 방법을 사용했다
  - □ 본론이 2개 단락 이상이다
  - □ 결론이 자연스럽게 마무리된다
  - □ 맞춤법·문장 부호가 올바르다

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 7 잠금 해제

### 차시 7~9: 독후감 쓰기 (3차시)

#### 차시 7: 독후감 구조 배우기

**① 개념 읽기**
- 핵심 내용: 독후감은 줄거리 요약이 아니다
- 독후감 4부 구조:
  1. 책 소개 (어떤 책인가?)
  2. 인상 깊은 장면 (왜 기억에 남는가?)
  3. 느낀 점 (어떤 생각이 들었나?)
  4. 내 삶과의 연결 (이 책이 내게 주는 의미)

**② READ — 예시 분석**
- 줄거리만 쓴 독후감 vs 잘 쓴 독후감 비교
- 분석 질문: "잘 쓴 독후감의 어떤 부분이 더 읽고 싶어지게 만드나요?"

**③ TRY — 독후감 계획표 작성**
- 최근 읽은 책 선택 → 독후감 계획표 채우기

**④ WRITE**
- 과제: 독후감 초안 (400자 이상)

**⑤ FEEDBACK** — AI 확인: 줄거리 요약 위주인지, 느낀 점과 연결이 있는지

**⑥ REVISE** → 합격 시 차시 8 잠금 해제

---

#### 차시 8: 독후감 — 느낀 점 깊이 쓰기

**① 개념 읽기**
- 핵심 내용: 느낀 점을 구체적으로 표현하는 3가지 방법

**④ WRITE**
- 과제: 차시 7 독후감에서 느낀 점 단락을 더 풍부하게 다시 쓰기
  - 수정 전/후 비교 메모 필수

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 9 잠금 해제

---

#### 차시 9: 독후감 최종 완성

**④ WRITE**
- 과제: 독후감 최종본 완성 (500자 이상)
- 자기 체크리스트:
  - □ 줄거리 요약이 전체의 30% 이하이다
  - □ 인상 깊은 장면의 이유가 구체적이다
  - □ 내 삶과의 연결이 포함되어 있다

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 10 잠금 해제

### 차시 10~11: 편지 쓰기 (2차시)

#### 차시 10: 편지 형식과 표현

**① 개념 읽기**
- 핵심 내용: 편지의 형식과 진심을 담는 법
- 편지 5구성: 받는 사람 + 첫인사 + 본문 + 끝인사 + 보내는 사람

**② READ — 예시 분석**
- 형식은 있지만 내용이 없는 편지 vs 진심이 담긴 편지
- 분석 질문: "잘 쓴 편지에서 어떤 부분이 가장 마음에 닿았나요?"

**④ WRITE**
- 과제: 가족 또는 친구에게 감사 편지 쓰기
  - 실제로 전달하고 싶은 내용으로

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 11 잠금 해제

## 그래픽 오거나이저

### 3부 구조 계획표
```
┌─────────────────────────────────────────────┐
│  제목 (가제):                               │
│  독자:                  목적:               │
├─────────────────────────────────────────────┤
│  서론                                       │
│  도입 방법: 질문 / 장면 / 사실 / 직접 제시  │
│  핵심 내용:                                 │
│                                             │
├─────────────────────────────────────────────┤
│  본론 단락 1          본론 단락 2           │
│  중심 문장:           중심 문장:            │
│  뒷받침 1:            뒷받침 1:            │
│  뒷받침 2:            뒷받침 2:            │
│  뒷받침 3:            뒷받침 3:            │
├─────────────────────────────────────────────┤
│  결론                                       │
│  핵심 요약:                                 │
│  독자에게 전하고 싶은 것:                   │
│                                             │
└─────────────────────────────────────────────┘
```

### 독후감 계획표
```
┌─────────────────────────────────────────────┐
│  책 제목:              작가:               │
├──────────────┬──────────────────────────────┤
│  책 소개     │ 어떤 내용의 책인가? (2~3문장) │
├──────────────┼──────────────────────────────┤
│  인상 깊은   │ 어떤 장면/인물?              │
│  장면        │ 왜 기억에 남나?              │
├──────────────┼──────────────────────────────┤
│  느낀 점     │ 어떤 생각/감정이 들었나?     │
├──────────────┼──────────────────────────────┤
│  내 삶 연결  │ 이 책이 내게 주는 의미는?    │
└──────────────┴──────────────────────────────┘
```

## 평가 루브릭 (최종 과제)

| 평가 기준 | 4점 | 3점 | 2점 | 1점 |
|-----------|-----|-----|-----|-----|
| **서론** | 흥미로운 도입 + 주제 명확히 제시 | 주제 제시는 있으나 도입 평범 | 주제 불분명 | 서론 역할 못 함 |
| **본론 구성** | 2개 이상 단락, 각 중심 문장 명확 | 2개 단락, 중심 문장 있음 | 1개 단락 | 구분 불명확 |
| **결론** | 핵심 정리 + 새로운 여운 | 핵심 정리 | 단순 반복 | 없거나 갑작스럽게 끝남 |
| **단락 연결** | 접속어로 자연스럽게 연결 | 대부분 연결 | 일부 연결 | 연결 없음 |
| **분량** | 500자 이상 | 400~499자 | 300~399자 | 300자 미만 |

**진급 기준**: 합산 15점 이상
