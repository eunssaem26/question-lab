# 2단계: 단락 쓰기

## 기본 정보

| 항목 | 내용 |
|------|------|
| **대상** | 초등 3~4학년 (또는 1단계 이수자) |
| **총 차시** | 12차시 |
| **차시당 학습 시간** | 약 40분 |
| **핵심 질문** | "한 가지 생각을 여러 문장으로 풀어내려면 어떻게 해야 할까?" |
| **진급 조건** | 최종 과제 루브릭 합산 14점 이상 (20점 만점) |

---

## 단계 목표

이 단계를 마치면 학습자는:
- 중심 문장과 뒷받침 문장의 차이를 설명할 수 있다
- 중심 문장 1개 + 뒷받침 문장 3~4개 + 마무리 문장으로 완전한 단락을 쓸 수 있다
- 뒷받침 방법(예시, 이유, 경험, 비교) 4가지를 구별하고 활용할 수 있다
- 2~3개의 단락을 연결해 짧은 소개글을 완성할 수 있다

---

## 핵심 개념

### 단락의 구조 (햄버거 모델)
```
        ┌──────────────────┐
        │   중심 문장       │  ← 이 단락에서 말하고 싶은 핵심
        ├──────────────────┤
        │  뒷받침 문장 1    │  ← 이유, 예시, 경험, 비교 중 선택
        │  뒷받침 문장 2    │
        │  뒷받침 문장 3    │
        ├──────────────────┤
        │   마무리 문장     │  ← 중심 내용 다시 강조
        └──────────────────┘
```

### 뒷받침 방법 4가지

| 방법 | 신호어 | 예시 |
|------|--------|------|
| **예시** | "예를 들어", "예컨대" | "예를 들어 나는 매일 아침 달리기를 한다." |
| **이유** | "왜냐하면", "그 이유는" | "왜냐하면 운동이 건강에 좋기 때문이다." |
| **경험** | "내가 직접", "한 번은" | "한 번은 비 오는 날 달리기를 했는데..." |
| **비교** | "~와 달리", "~에 비해" | "달리기는 수영과 달리 장비가 필요 없다." |

## 차시별 학습 내용

### 차시 1: 중심 문장이란 무엇인가

**① 개념 읽기**
- 핵심 내용: 중심 문장과 뒷받침 문장의 차이

**② READ — 예시 분석**
- 단락 예시 2개 (중심 문장이 명확한 것 / 모호한 것)
- 분석 질문:
  - "이 단락에서 중심 문장은 어느 것인가요? 밑줄을 그어보세요."
  - "나머지 문장들은 중심 문장과 어떻게 연결되나요?"

**③ TRY — 연습 문제**
- 3개의 단락에서 중심 문장 찾기 (각 단락 5~6문장)
- 5개의 문장 묶음에서 중심 문장이 될 수 없는 것 고르기

**④ WRITE**
- 과제: 내가 좋아하는 것에 대한 중심 문장 3개 만들기
  - 예: 음식, 취미, 계절, 동물 등
- 자기 체크리스트:
  - □ 각 문장이 하나의 명확한 생각을 담고 있다
  - □ 너무 넓거나 좁지 않다 ("나는 좋은 것들이 많다" → 너무 넓음)

**⑤ FEEDBACK** — AI 확인 항목
- 문장이 하나의 명확한 주제에 집중하는지 확인
- 너무 광범위하거나 애매한 중심 문장 지적

**⑥ REVISE** → 합격 시 차시 2 잠금 해제

### 차시 2: 뒷받침 문장 — 예시와 이유

**① 개념 읽기**
- 핵심 내용: '예를 들어'와 '왜냐하면'으로 뒷받침하기

**② READ — 예시 분석**
- 중심 문장 + 예시 뒷받침 단락
- 중심 문장 + 이유 뒷받침 단락
- 분석 질문:
  - "예시 방법과 이유 방법의 차이는 무엇인가요?"
  - "이 두 단락 중 더 설득력 있는 것은 어느 쪽인가요? 왜 그렇게 생각하나요?"

**③ TRY — 연습 문제**
- 중심 문장이 주어졌을 때 예시 문장 1개 + 이유 문장 1개 추가하기 (2세트)

**④ WRITE**
- 과제: "내 친구(또는 가족)를 소개하는 단락" 쓰기
  - 중심 문장 1개 + 예시 또는 이유 뒷받침 3개
- 자기 체크리스트:
  - □ 중심 문장이 첫 번째 문장이다
  - □ 뒷받침 문장이 3개 있다
  - □ 각 뒷받침 문장이 중심 문장과 연결된다

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 3 잠금 해제

### 차시 3: 뒷받침 문장 — 경험과 비교

**① 개념 읽기**
- 핵심 내용: '내 경험'과 '비교'로 뒷받침하기

**② READ — 예시 분석**
- 경험 뒷받침 단락 예시
- 비교 뒷받침 단락 예시
- 분석 질문:
  - "경험 뒷받침이 효과적인 이유는 무엇인가요?"
  - "비교 방법을 쓸 때 주의할 점은 무엇인가요?"

**③ TRY — 연습 문제**
- 동일한 중심 문장에 대해 경험 뒷받침 1개, 비교 뒷받침 1개 써보기

**④ WRITE**
- 과제: "내가 좋아하는 계절" 단락 쓰기
  - 경험 뒷받침 1개 이상 반드시 포함
- 자기 체크리스트:
  - □ 경험 뒷받침 문장에 "한 번은", "내가 직접" 등 신호어가 있다
  - □ 뒷받침이 중심 문장에서 벗어나지 않는다

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 4 잠금 해제

### 차시 4: 뒷받침 방법 혼합하기

**① 개념 읽기**
- 핵심 내용: 4가지 방법을 섞으면 단락이 더 풍부해진다

**② READ — 예시 분석**
- 3가지 방법을 혼합한 단락 예시
- 분석 질문:
  - "이 단락에서 사용된 뒷받침 방법을 모두 찾고 이름을 적어보세요."

**③ TRY — 연습 문제**
- 단락을 읽고 각 뒷받침 문장에 방법 이름 붙이기 (예시/이유/경험/비교)

**④ WRITE**
- 과제: 자유 주제 단락 (뒷받침 방법 2가지 이상 혼합)
- 자기 체크리스트:
  - □ 2가지 이상의 뒷받침 방법을 사용했다
  - □ 각 방법의 신호어를 사용했다

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 5 잠금 해제

### 차시 5: 마무리 문장 쓰기

**① 개념 읽기**
- 핵심 내용: 마무리 문장은 중심 문장의 복사본이 아니다 — 단순 반복 금지, 의미 확장 또는 새로운 시각으로 마무리

**② READ — 예시 분석**
- 나쁜 마무리 문장 vs 좋은 마무리 문장 비교
- 분석 질문:
  - "나쁜 마무리 문장의 문제점은 무엇인가요?"

**③ TRY — 연습 문제**
- 중심 문장 + 뒷받침 3개가 주어졌을 때 마무리 문장 쓰기 (2세트)

**④ WRITE**
- 과제: 지금까지 쓴 단락 중 하나에 마무리 문장 추가하기
- 자기 체크리스트:
  - □ 마무리 문장이 중심 문장을 그대로 반복하지 않는다
  - □ 단락 전체의 내용을 자연스럽게 마무리한다

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 6 잠금 해제

### 차시 6: 완전한 단락 1개 완성

**④ WRITE**

  - 주제: 자유 선택
- 자기 체크리스트 (전체):

  - □ 뒷받침 문장이 3개 있다
  - □ 뒷받침 방법 2가지 이상을 사용했다
  - □ 마무리 문장이 단순 반복이 아니다
  - □ 맞춤법·문장 부호가 올바르다

### 차시 7: 접속어로 단락 연결하기

**① 개념 읽기**
- 핵심 내용: 단락과 단락을 잇는 접속어 — 그리고, 하지만, 그래서, 또한, 반면에, 따라서

**② READ — 예시 분석**
- 접속어 없이 연결된 두 단락 vs 접속어로 연결된 두 단락
- 분석 질문: "접속어가 없을 때 어떤 느낌이 드나요?"

**③ TRY — 연습 문제**
- 두 단락 사이에 알맞은 접속어 고르기 (5문항)

**④ WRITE**
- 과제: 이미 쓴 단락 2개를 접속어로 자연스럽게 연결하기
  - 또는 새 주제로 2단락 글 쓰기 (접속어 연결 포함)

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 8 잠금 해제

**④ WRITE**

**⑤ FEEDBACK** — AI 확인: 각 단락 구조, 접속어 사용
