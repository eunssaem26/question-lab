# 5단계: 이야기 글

## 기본 정보

| 항목 | 내용 |
|------|------|
| **대상** | 초등 6학년 ~ 중학교 1학년 (또는 4단계 이수자) |
| **총 차시** | 16차시 |
| **차시당 학습 시간** | 약 50분 |
| **핵심 질문** | "독자가 내 이야기 속으로 빠져들게 하려면 어떻게 써야 할까?" |
| **진급 조건** | 최종 과제 루브릭 합산 15점 이상 (20점 만점) |

---

## 단계 목표

이 단계를 마치면 학습자는:
- 서사의 3요소(인물, 배경, 사건)를 이해하고 글에 구현할 수 있다
- 이야기의 구성 단계(발단-전개-절정-결말)를 설계하고 쓸 수 있다
- "보여주기(Show)"와 "말하기(Tell)"의 차이를 알고 묘사 글을 쓸 수 있다
- 인물의 감정을 직접 서술하지 않고 행동과 대화로 표현할 수 있다
- 800~1000자 분량의 단편 서사 글을 완성할 수 있다

---

## 핵심 개념

### 서사의 3요소

| 요소 | 핵심 질문 |
|------|----------|
| **인물** | 이 사람은 무엇을 원하는가? 무엇이 방해하는가? |
| **배경** | 언제, 어디서 벌어지는 이야기인가? |
| **사건** | 무슨 일이 벌어졌나? 어떻게 해결됐나? |

### 이야기 4막 구조
```
발단 → 전개 → 절정 → 결말

발단: 인물과 배경 소개, 사건의 씨앗
전개: 문제 발생, 인물의 해결 시도
절정: 가장 긴장된 순간 (위기의 정점)
결말: 문제 해결, 인물의 변화 또는 교훈
```

### Show vs Tell (보여주기 vs 말하기)

| Tell (직접 서술) | Show (보여주기) |
|-----------------|----------------|
| "민준이는 화가 났다." | "민준이는 주먹을 꽉 쥐고 발을 구르며 방문을 쾅 닫았다." |
| "날씨가 무서웠다." | "하늘이 노랗게 물들고 바람이 귀를 때리며 불었다." |
| "그 음식은 맛있었다." | "한 입 베어 무는 순간 달콤한 즙이 입 안 가득 퍼졌다." |

**Show의 도구**: 감각(시각·청각·후각·미각·촉각), 구체적 행동, 대화

## 차시별 학습 내용

### 차시 1: 이야기의 4막 구조

**① 개념 읽기**
- 핵심 내용: 모든 이야기에는 구조가 있다

**② READ — 예시 분석**
- 익숙한 단편 이야기 하나를 4막으로 분석
- 분석 질문:
  - "절정은 어디인가? 왜 그 부분이 가장 긴장되는가?"
  - "결말에서 인물이 어떻게 변화했나?"

**③ TRY — 연습 문제**
- 뒤섞인 이야기 단편 4개를 발단-전개-절정-결말 순서로 배열

**④ WRITE**
- 과제: 내가 겪은 사건 하나를 4막 구조로 한 문장씩 정리하기

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 2 잠금 해제

---

### 차시 2: 인물 만들기

**① 개념 읽기**
- 핵심 내용: 입체적 인물 vs 평면적 인물 — 악인도 나름의 이유가 있다, 완벽한 주인공은 재미없다

**② READ — 예시 분석**
- 평면적 인물 묘사 vs 입체적 인물 묘사 비교
- 분석 질문: "입체적 인물이 더 기억에 남는 이유는 무엇인가요?"

**③ TRY — 연습 문제**
- 인물 카드 작성: 교사가 제시한 인물에 대해 빈칸 채우기 (원하는 것, 두려워하는 것, 비밀)

**④ WRITE**
- 과제: 내 이야기의 주인공 인물 카드 작성 + 인물 소개 단락 (행동과 대화로 성격 드러내기)
- 자기 체크리스트:
  - □ 인물의 성격이 말이나 행동으로 드러난다
  - □ "착하다", "나쁘다"를 직접 쓰지 않았다

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 3 잠금 해제

---

### 차시 3: 배경 묘사하기

**① 개념 읽기**
- 핵심 내용: 배경은 장식이 아니다 — 분위기를 만드는 법

**② READ — 예시 분석**
- 같은 장소를 기쁠 때 / 슬플 때 묘사한 두 단락 비교
- 분석 질문: "감정이 다르면 같은 장소가 어떻게 다르게 묘사되나요?"

**③ TRY — 연습 문제**
- 사진 1장을 보고 5감 관찰 오거나이저 채우기

**④ WRITE**
- 과제: 이야기의 시작 배경 묘사 단락 (5감 중 3가지 이상 사용)
- 자기 체크리스트:
  - □ 시각 외에 2가지 이상 감각을 사용했다
  - □ 배경이 이야기의 분위기를 만든다

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 4 잠금 해제

---

### 차시 4~5: Show vs Tell (2차시)

#### 차시 4: Tell을 Show로 바꾸기

**① 개념 읽기**
- 핵심 내용: 보여주기 vs 말하기 — 왜 Show가 더 좋은가

**② READ — 예시 분석**
- Tell 단락 vs Show 단락 비교
- 분석 질문: "Show 단락에서 감정을 직접 쓴 단어가 있나요? 그런데 왜 감정이 느껴지나요?"

**③ TRY — 연습 문제**
- Tell 문장을 Show 문장으로 바꾸기 (8문항)
  - "화가 났다", "무서웠다", "기뻤다", "슬펐다" 등

**④ WRITE**
- 과제: 절정 장면 하나를 Show 기법으로 쓰기 (150자 이상)
- 자기 체크리스트:
  - □ 감정을 직접 서술하는 단어가 없다
  - □ 행동, 대화, 감각 묘사로 감정이 드러난다

**⑤ FEEDBACK** — AI 확인: Tell 표현 감지

**⑥ REVISE** → 합격 시 차시 5 잠금 해제

---

#### 차시 5: Show 기법 심화

**④ WRITE**
- 과제: 이야기 속 인물이 두려움을 느끼는 장면 + 기쁨을 느끼는 장면을 각각 Show로 쓰기 (각 100자 이상)

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 6 잠금 해제

---

### 차시 6: 대화 쓰기

**① 개념 읽기**
- 핵심 내용: 대화로 인물을 살아있게 만드는 법 — 대화 문장 부호 규칙, 대화를 통해 인물 성격 드러내기

**② READ — 예시 분석**
- 대화 없는 장면 vs 대화 있는 장면 비교
- 분석 질문: "대화가 있을 때 어떤 점이 달라지나요?"

**③ TRY — 연습 문제**
- 대화 문장 부호 바르게 고치기 (5문항)
- 인물의 성격(겁쟁이 / 용감한 / 소심한)을 보여주는 대화 한 줄씩 쓰기

**④ WRITE**
- 과제: 이야기 중 갈등 장면을 대화 포함해 작성 (200자 이상)
- 자기 체크리스트:
  - □ 대화 문장 부호가 올바르다
  - □ 대화가 인물의 성격을 보여준다

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 7 잠금 해제

**④ WRITE**

## 그래픽 오거나이저

### 인물 카드
```
┌─────────────────────────────────────────────┐
│  이름:                나이:                 │
├──────────────┬──────────────────────────────┤
│  외모 (3가지) │                             │
├──────────────┼──────────────────────────────┤
│  성격         │ 장점:                       │
│               │ 단점:                       │
├──────────────┼──────────────────────────────┤
│  원하는 것    │ 가장 바라는 것:             │
│               │ 두려워하는 것:              │
├──────────────┼──────────────────────────────┤
│  비밀         │                             │
└──────────────┴──────────────────────────────┘
```

### 4막 구성표
```
┌─────────────┬──────────────────────────────────┐
│  발단        │ 어떤 인물이 어디에 있나?         │
│              │ 이야기 시작의 계기는?            │
├─────────────┼──────────────────────────────────┤
│  전개        │ 어떤 문제/갈등이 생겼나?         │
│              │ 인물이 어떻게 해결하려 했나?     │
├─────────────┼──────────────────────────────────┤
│  절정        │ 가장 긴장된 순간은?              │
│              │ 어떤 선택의 기로에 섰나?         │
├─────────────┼──────────────────────────────────┤
│  결말        │ 어떻게 해결됐나?                 │
│              │ 인물은 어떻게 변했나?            │
└─────────────┴──────────────────────────────────┘
```

## 평가 루브릭 (최종 과제)

| 평가 기준 | 4점 | 3점 | 2점 | 1점 |
|-----------|-----|-----|-----|-----|
| **구조** | 4막 구조 명확, 자연스러운 흐름 | 4막 있으나 일부 어색 | 일부 막 불분명 | 구조 없음 |
| **인물** | 행동·대화로 입체적 인물 표현 | 성격 일부 드러남 | 평면적 서술 | 인물 묘사 없음 |
| **Show 기법** | 전반적으로 Show 사용, Tell 최소화 | Show와 Tell 균형 | 주로 Tell | 완전히 Tell |
| **배경 묘사** | 2가지 이상 감각 사용한 생생한 묘사 | 시각 중심 묘사 | 간단한 언급 | 묘사 없음 |
| **분량** | 900자 이상 | 800~899자 | 700~799자 | 700자 미만 |

**진급 기준**: 합산 15점 이상
