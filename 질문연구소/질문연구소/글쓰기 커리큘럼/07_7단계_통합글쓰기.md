# 7단계: 통합 글쓰기

## 기본 정보

| 항목 | 내용 |
|------|------|
| **대상** | 중학교 2~3학년 (또는 6단계 이수자) |
| **총 차시** | 18차시 |
| **차시당 학습 시간** | 약 50분 |
| **핵심 질문** | "이 글의 목적은 무엇이고, 독자는 누구이며, 가장 적합한 형식은 무엇인가?" |
| **이수 조건** | 최종 과제 루브릭 합산 17점 이상 (20점 만점) + 포트폴리오 완성 |

---

## 단계 목표

이 단계를 마치면 학습자는:
- 글의 목적·독자·형식을 스스로 결정하고 그에 맞는 글을 쓸 수 있다
- 체계적인 4단계 퇴고(내용→구조→문장→어휘)를 독립적으로 수행할 수 있다
- 에세이, 칼럼, 제안서 등 다양한 형식의 글을 완성할 수 있다
- 자신의 포트폴리오를 분석해 성장 지점을 스스로 파악할 수 있다
- 1200자 이상의 완성도 있는 글을 기획·작성·퇴고할 수 있다

---

## 핵심 개념

### 글쓰기 3축 (목적 × 독자 × 형식)
```
       목적
        │
        │   이 세 가지가 일치할 때
        │   좋은 글이 완성된다
독자 ───┼─── 형식
```

| 목적 | 독자 예 | 적합한 형식 |
|------|---------|------------|
| **정보 전달** | 모르는 사람 | 설명문, 칼럼 |
| **설득** | 반대 의견을 가진 사람 | 논설문, 제안서 |
| **표현·공유** | 공감을 원하는 독자 | 에세이, 수필 |
| **요청·제안** | 결정권자 | 제안서, 건의문 |

### 4단계 퇴고

| 순서 | 초점 | 핵심 질문 |
|------|------|----------|
| **1단계: 내용** | 무엇을 말했는가 | 빠진 내용은? 불필요한 내용은? |
| **2단계: 구조** | 어떻게 배치했는가 | 논리 순서가 맞는가? 단락 구분이 적절한가? |
| **3단계: 문장** | 어떻게 표현했는가 | 어색한 문장은? 문장 길이가 다양한가? |
| **4단계: 어휘** | 어떤 단어를 썼는가 | 더 정확한 단어는? 반복 표현은? |

### 7단계에서 다루는 글 형식

| 형식 | 특징 | 목표 분량 |
|------|------|----------|
| **개인 에세이** | 경험·생각·감정을 솔직하게 탐구 | 1200~1500자 |
| **칼럼** | 시사 이슈에 대한 개인 관점 | 1000~1200자 |
| **제안서/건의문** | 문제 제시 + 구체적 해결책 | 1200~1500자 |

## 차시별 학습 내용

### 차시 1: 목적·독자·형식 결정하기

**① 개념 읽기**
- 핵심 내용: 같은 주제, 다른 형식 — 에세이/논설문/제안서 비교 (동일한 주제를 형식별로 어떻게 다르게 쓰는가)

**② READ — 예시 분석**
- 3가지 형식의 글 비교 분석
- 분석 질문:
  - "세 글의 어조가 어떻게 다른가요?"
  - "독자가 다르면 어떤 부분이 달라지나요?"

**③ TRY — 연습 문제**
- 10가지 시나리오에서 목적·독자·형식 결정하기

**④ WRITE**
- 과제: 내가 관심 있는 주제로 목적·독자·형식 결정 + 이유 설명하기
- 자기 체크리스트:
  - □ 3가지 축이 서로 일관성 있게 맞는가?
  - □ 다른 형식이 더 적합하지는 않은가?

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 2 잠금 해제

### 차시 2~4: 개인 에세이 (3차시)

#### 차시 2: 에세이 구조 배우기

**① 개념 읽기**
- 핵심 내용: 에세이란 무엇인가 — 나를 탐구하는 글
- 에세이 4부 구조:
  1. 도입 훅 (독자의 관심을 끄는 시작)
  2. 경험/관찰 (구체적 장면 묘사)
  3. 의미 탐구 (이 경험이 왜 의미 있는가?)
  4. 깨달음 (내가 얻은 통찰)

**② READ — 예시 분석**
- 잘 쓴 에세이 1편 분석 (1200자 내외)
- 분석 질문:
  - "도입에서 어떤 방법으로 독자의 관심을 끌었나요?"
  - "경험이 구체적으로 묘사된 부분을 찾아보세요."

**④ WRITE**
- 과제: 에세이 주제 선정 + 4부 구조 계획표 작성
  - 주제 예: 나를 변화시킨 경험 / 내가 가진 편견 / 내가 확신하는 것

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 3 잠금 해제

#### 차시 3: 에세이 초안 쓰기

**④ WRITE**

  - □ 내 진짜 생각이 담겨 있다
  - □ 경험이 구체적인 장면으로 묘사된다
  - □ 단순한 사건 나열이 아니라 의미 탐구가 있다

**⑤ FEEDBACK** → **⑥ REVISE** → 합격 시 차시 4 잠금 해제

## 그래픽 오거나이저

### 글쓰기 3축 결정표
```
┌─────────────────────────────────────────────┐
│  주제:                                      │
├──────────────────┬──────────────────────────┤
│  목적            │ □정보 전달 □설득         │
│                  │ □표현·공유 □요청·제안    │
│                  │ 구체적 목적:             │
├──────────────────┼──────────────────────────┤
│  독자            │ 누구인가?                │
│                  │ 독자가 이미 아는 것:     │
│                  │ 독자가 원하는 것:        │
├──────────────────┼──────────────────────────┤
│  형식            │ □에세이 □칼럼 □제안서   │
│                  │ 이 형식을 선택한 이유:   │
└──────────────────┴──────────────────────────┘
```

### 4단계 퇴고 체크리스트
```
[ 1단계: 내용 ]
□ 주제에서 벗어난 내용이 없다
□ 꼭 필요한 내용이 모두 포함되어 있다
□ 사실과 의견이 명확히 구분된다

[ 2단계: 구조 ]
□ 서론-본론-결론이 균형 잡혀 있다
□ 단락마다 중심 생각이 있다
□ 단락 순서가 논리적이다

[ 3단계: 문장 ]
□ 어색하게 읽히는 문장이 없다
□ 너무 길거나 짧은 문장이 반복되지 않는다
□ 주어-서술어 호응이 맞다

[ 4단계: 어휘 ]
□ 같은 단어가 한 단락에 3번 이상 반복되지 않는다
□ 더 정확한 단어로 바꿀 수 있는 곳이 없다
□ 맞춤법·띄어쓰기 오류가 없다
```

### 제안서 구조표
```
┌────────────────────────────────────────────┐
│  문제 제시                                 │
│  어떤 문제가 있는가? 왜 중요한가?          │
├────────────────────────────────────────────┤
│  현황 분석                                 │
│  현재 상황은? (수치, 사례 포함)            │
├────────────────────────────────────────────┤
│  해결책 1:  무엇을, 어떻게?               │
│  해결책 2:  무엇을, 어떻게?               │
├────────────────────────────────────────────┤
│  기대 효과                                 │
│  이 제안이 실현되면 어떤 변화가 생기는가?  │
└────────────────────────────────────────────┘
```

## 평가 루브릭 (최종 과제)

| 평가 기준 | 4점 | 3점 | 2점 | 1점 |
|-----------|-----|-----|-----|-----|
| **3축 일치** | 목적·독자·형식 완전히 일관 | 2가지 일관 | 1가지만 명확 | 고려 없음 |
| **퇴고** | 4단계 모두 적용, 의미 있는 수정 | 3단계 적용 | 2단계 | 수정 흔적 없음 |
| **완성도** | 처음 읽는 독자가 이해하고 설득됨 | 대체로 이해됨 | 일부 혼란 | 전반적으로 혼란 |
| **기법 통합** | 이전 단계 기법 3가지 이상 명확히 적용 | 2가지 | 1가지 | 없음 |
| **분량** | 1400자 이상 | 1200자 이상 | 1000자 이상 | 1000자 미만 |

**이수 기준**: 합산 17점 이상 + 포트폴리오 완성
