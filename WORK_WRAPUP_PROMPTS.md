# 작업 마감용 프롬프트 모음

## 1. 짧은 요약

지금까지 작업한 내용을 10줄 이내로 요약해줘.

## 2. 파일 중심 요약

오늘 수정한 파일별로 무엇이 바뀌었는지 짧게 정리해줘.

## 3. 다음 작업만 추리기

남은 작업 중 다음 세션에서 바로 해야 할 일만 우선순위 순으로 정리해줘.

## 4. 체크포인트 만들기

지금까지의 상태를 다음 세션에서 바로 이어갈 수 있도록 체크포인트 형태로 정리해줘.

## 5. SESSION_NOTE용 정리

지금까지 작업한 내용을 `SESSION_NOTE_TEMPLATE.md` 형식에 맞춰 채워줘.

## 6. 새 스레드 시작용 프롬프트

다음 세션에서 새 대화를 열었을 때 붙여넣을 시작 프롬프트를 만들어줘.

## 7. 협업 인수인계용 요약

클로드 또는 다른 작업자에게 넘길 수 있게 현재 상태, 완료 사항, 남은 일, 주의사항을 정리해줘.

## 8. 검수 대기 목록

아직 검수되지 않았거나 다시 봐야 하는 항목만 따로 목록으로 정리해줘.

## 9. 규칙만 따로 정리

지금 프로젝트에서 꼭 지켜야 하는 규칙만 짧게 다시 정리해줘.

## 10. 오늘 마감용 추천 프롬프트

아래 프롬프트를 그대로 쓰면 된다.

`오늘 작업한 내용을 SESSION_NOTE_TEMPLATE.md 형식으로 정리하고, 다음 세션 시작점 3가지만 뽑아줘.`
