## 🔄 사용 안내

### 학생 입장에서
1. 위 링크 중 원하는 단계 선택
2. 각 영역별 문제를 읽고 답변
3. 마지막에 "진단 완료하기" 버튼 클릭
4. 임시 결과 화면 표시 (향후 학습 통계 추가 예정)

### 현재 제한사항
- ⚠️ 학생 로그인 없음 (향후 추가)
- ⚠️ 결과 저장 없음 (시뮬레이션 단계)
- ⚠️ 즉시 채점 및 상세 분석 미지원

## 💬 피드백 요청

다음 항목에 대한 의견을 부탁드립니다:

### 1. 진단 난도
- [ ] 1단계가 너무 쉽다 / 적절하다 / 어렵다
- [ ] 2단계가 너무 쉽다 / 적절하다 / 어렵다

### 2. 문항 내용
- [ ] 교육과정 성취기준 반영도
- [ ] 단계별 위계 명확성
- [ ] 선택지의 구분 용이성

### 3. 사용자 경험
- [ ] 페이지 로딩 속도
- [ ] 문제 가독성
- [ ] 선택지 선택 용이성
- [ ] 버튼·네비게이션 명확성

### 4. 기능 개선 사항
- [ ] 시간 제한
- [ ] 진행률 표시
- [ ] 즉시 채점 및 해설 제공
- [ ] 결과 비교/분석
