- **완료된 작업**:

사용자 친화적인 디자인을 목표로 했어요.

---

## 어려움과 해결 과정: 프레임워크 선택

Next.js와 Astro 두 가지를 검토했어요. Next.js는 React 기반으로 동적 앱에 강하지만, 서버 사이드 렌더링이 필요해서 인프라가 복잡해져요. 질문연구소 사이트는 진단 문항과 결과를 보여주는 정적 콘텐츠가 중심이라 서버가 없어도 되고, 빠른 로딩과 SEO가 더 중요했어요.

Astro는 기본적으로 JavaScript를 최소화해서 페이지가 가볍고, 콘텐츠 중심 사이트에 최적화되어 있어요. 서버 없이 정적 파일만으로 배포할 수 있다는 것도 장점이었습니다. 호기가 두 프레임워크로 간단한 프로토타입을 만들어 비교했고, 성능 차이가 명확했어요.
